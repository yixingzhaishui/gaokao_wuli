- 播放过程：先看纸带不是“画好”的，而是按周期 `T` 一个点一个点留下。
- 拖蓝色速度把手：看所有点距是否整体变大。
- 拖红色加速度把手：看后面点距是否越来越大，并在 `v-t` 图像中看到斜率变大。
- 拖绿色测点：看 $x_{i-1}$、$x_{i+1}$ 和 $vi\approx (x_{i+1}-x_{i-1})/(2\ \mathrm{T})$ 如何一一对应到纸带、刻度尺和图像游标。



#### 探究问题与目标
- 会用打点计时器在纸带上记录物体运动位置。
- 会根据相邻点或相邻计数点间距求平均速度。
- 会用中间时刻速度近似公式求某点瞬时速度。
- 会从纸带点距变化判断物体运动快慢变化。

**装置或现象预览。**
打点计时器每隔相同时间在纸带上打一个点。小车拉动纸带运动，点与点之间的时间间隔固定；点距越大，说明这段时间内纸带通过的距离越大，速度越大。

#### 原理、变量、单位与适用条件





打点周期为 `T`，相邻两点时间间隔相同。若从点 `i` 到点 `j` 的距离为 $\Delta x$，经历时间为：

```text
Δt = (j-i)T
v平均 = Δx/Δt
```

若要求某点附近的瞬时速度，常用该点前后相邻点间的平均速度近似：

```text
vi ≈ (x_{i+1}-x_{i-1})/(2T)
```

#### 装置、器材与安全
- 电磁打点计时器使用配套低压交流电源；电火花计时器按设备铭牌和说明书接入规定交流电源。两类设备的供电端、纸带安装和耗材不同，不得凭“都是打点计时器”混接。
- 纸带；电磁打点计时器还需复写纸，电火花计时器按设备配置使用墨粉纸盘等耗材。
- 小车、长木板或气垫导轨。
- 刻度尺。
- 数据记录表。

#### 操作步骤
1. 先辨认计时器类型，核对铭牌、配套电源、接线端和开关；断电状态下完成接线。
2. 固定打点计时器，使纸带能平直穿过限位孔，并按设备类型正确安装复写纸或墨粉纸盘。
3. 将纸带一端连在小车上，另一端穿过打点计时器。
4. 先接通电源，让打点计时器稳定打点。
5. 再释放小车，使纸带随小车运动。
6. 实验结束立即关闭电源，再取下纸带，选择点迹清晰的一段。
7. 用刻度尺测各点到起点的距离，填入数据表。
8. 用相邻点或计数点间距计算平均速度或瞬时速度估计值。

#### 数据处理与证据
| 物理量 | 记录方法 | 计算关系 |
|---|---|---|
| 打点周期 | 由电源频率确定，如 $T=0.02\ \mathrm{s}$ | 相邻点时间间隔 |
| 点位置 $x_i$ | 用刻度尺测到统一起点的距离 | 注意统一单位 |
| 平均速度 | 选一段纸带测 $\Delta x$ | $v=\Delta x/\Delta t$ |
| 某点瞬时速度 | 取该点前后相邻点 | $vi\approx (x_{i+1}-x_{i-1})/(2\ \mathrm{T})$ |

#### 不确定度、失效情形与改进
- 纸带与打点计时器摩擦会影响小车运动。
- 点迹过密或模糊会导致读数误差。
- 刻度尺读数要估读到最小分度后一位。
- 起点选择应避开开始阶段点迹重叠、运动不稳定的部分。
- 计算瞬时速度时，所选前后点应尽量靠近目标点。

**适用边界。**
| 能用 | 不能用或需重做 |
|---|---|
| 纸带平直穿过计时器，点迹清晰 | 纸带歪斜、卡滞，点迹成片或明显断点 |
| 先接通电源，打点稳定后再释放小车 | 先释放小车后通电，开头数据漏记或混乱 |
| 相邻点时间间隔由电源频率稳定决定 | 电源频率不稳，`T` 不能视为常量 |
| 选取点距适中、便于读数的一段 | 点太密无法区分，或点太疏导致样本太少 |
| 用同一把刻度尺从同一起点测 $x_i$ | 每段分别量且起点不统一，容易累积读数错误 |

- 对应 [B1-05 平均速度与瞬时速度](bx1?id=velocity)：用纸带位移和时间理解平均速度与瞬时速度估计。
- 对应 [B1-08 v-t 图像](bx1?id=vt-graph)：由不同点速度作图，观察速度随时间变化。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “打点间隔是距离间隔” | 打点计时器保证时间间隔相等 | 距离由运动决定 |
| “先放小车再开电源” | 会漏掉开始阶段点迹 | 先通电后释放 |
| “点距越小速度越大” | 相同时间内位移越大速度越大 | 点距大速度大 |
| “瞬时速度直接用某一点距离除时间” | 单点没有位移段 | 用前后点平均速度近似 |
| “所有点都必须使用” | 开始或末尾点迹可能不清晰 | 选清晰稳定段 |

#### 原理、变量、单位与适用条件
匀变速直线运动的速度随时间线性变化：






```text
v = v0 + at
```

所以 `v-t` 图像应为直线，斜率为加速度：

```text
a = Δv/Δt
```

若连续相等时间 `T` 内位移分别为 $s_{1}、s_{2}、s_{3}...$，匀变速运动还满足：

```text
s2-s1 = s3-s2 = s4-s3 = aT²
```

实际实验有误差，因此看“近似相等”。

#### 装置、器材与安全
- 打点计时器、电源、纸带、复写纸。
- 小车、长木板、滑轮、细线、钩码。
- 刻度尺、坐标纸或数据处理工具。

#### 操作步骤
1. 安装打点计时器，让纸带平直穿过。
2. 连接小车、细线和钩码，使小车能近似匀加速运动。
3. 先接通打点计时器电源，再释放小车。
4. 取下纸带，选取点迹清晰、间距逐渐变化的一段。
5. 每隔若干个点取一个计数点，记录各计数点到起点的距离。
6. 用前后计数点距离估计各计数点速度。
7. 作 `v-t` 图像，判断是否近似直线，并由斜率求加速度。

#### 数据处理与证据
| 数据 | 处理方法 | 物理意义 |
|---|---|---|
| 计数点位置 $x_i$ | 统一从同一原点测量 | 得到各时刻位置 |
| 相邻段位移 $s_i$ | $s_i=x_i-x_{i-1}$ | 看相等时间内位移 |
| 相邻位移差 | $s_{i+1}-s_i$ | 判断加速度是否恒定 |
| 点速度 | $v_i\approx (x_{i+1}-x_{i-1})/(2\Delta t)$ | 作 `v-t` 图 |
| 图像斜率 | $a=\Delta v/\Delta t$ | 求加速度 |

#### 不确定度、失效情形与改进
- 纸带摩擦、滑轮摩擦会使加速度偏小。
- 计数点选得太近，距离读数相对误差会变大。
- 计数点选得太远，瞬时速度近似会变差。
- 纸带起始段可能点迹重叠，末段可能受碰撞或拉线影响，应剔除。
- 作图时不要强行让直线过所有点，应看整体趋势并合理拟合。

**适用边界。**
| 能用 | 不能用或需重做 |
|---|---|
| 纸带点迹清晰，计数点间隔合适 | 点太密、太模糊或起始段重叠 |
| 小车受力近似恒定，轨道摩擦较小 | 拉线松弛、滑轮卡滞或末端碰撞影响纸带 |
| `v-t` 点总体接近一条直线 | 速度点弯曲明显，说明加速度不恒定 |
| 相邻位移差近似相等 | $s_{i+1}-s_i$ 波动过大，不能支持匀变速 |
| 统一原点测量 $x_i$ | 每段分开量且读数基准不统一 |

#### 结论与迁移
- 对应 [B1-10 匀变速直线运动基本规律](bx1?id=uniform-acceleration)：实验支撑 $v=v_{0}+at$ 与 `s` 的二次变化。
- 关联 [B1-08 v-t 图像](bx1?id=vt-graph)：斜率表示加速度，面积表示位移。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “点距变大就一定严格匀加速” | 点距变大只说明速度增大 | 还要看位移差或 v-t 直线 |
| “v-t 图必须每个点都在线上” | 实验有误差 | 看近似直线和拟合 |
| “相邻位移差等于速度” | 位移差与加速度有关 | $s_{i+1}-s_i=aT^{2}$ |
| “计数点越密越好” | 太密读数误差大 | 适当隔点取计数点 |
| “斜率单位是 m/s” | $\Delta v/\Delta t$ 单位是 m/s² | 斜率是加速度 |

#### 原理、变量、单位与适用条件
弹簧静止时，钩码受到重力和弹簧弹力平衡：






```text
F弹 = G = mg
```

弹簧形变量为：

```text
x = l - l0
```

弹性限度内：

```text
F弹 = kx
```

所以作 `F-x` 图像，若图像近似过原点直线，则验证胡克定律；直线斜率就是劲度系数 `k`。

#### 装置、器材与安全
- 弹簧、铁架台、刻度尺。
- 钩码若干。
- 指针或标记线。

#### 操作步骤
1. 将弹簧竖直悬挂，待其静止。
2. 记录弹簧原长 $l_{0}$。
3. 逐个增加钩码，待静止后记录弹簧长度 `l`。
4. 计算形变量 $x=l-l_{0}$。
5. 计算弹力 $F=mg$，填写数据表。
6. 以 `x` 为横轴、`F` 为纵轴作图。
7. 判断图像是否近似过原点直线，并由斜率求 `k`。

#### 数据处理与证据
| 数据 | 记录方法 | 计算关系 |
|---|---|---|
| 原长 $l_{0}$ | 不挂钩码时测量 | 基准长度 |
| 现长 `l` | 每次挂钩码后测量 | 注意视线平视 |
| 形变量 `x` | $x=l-l_{0}$ | 用米作单位 |
| 弹力 `F` | $F=mg$ | 静止时等于钩码重力 |
| 劲度系数 `k` | `F-x` 图像斜率 | $k=\Delta F/\Delta x$ |

#### 不确定度、失效情形与改进
- 弹簧和钩码没有完全静止时读数会偏差。
- 刻度尺未竖直或视线不平会产生读数误差。
- 钩码质量标称值与实际值可能有误差。
- 超过弹性限度后，`F-x` 不再保持线性。
- 原长测量误差会影响所有形变量。

#### 结论与迁移
- 对应 [B1-17 胡克定律](bx1?id=hooke-law)：用实验图像验证 $F=kx$，并理解 `k` 是图像斜率。
- 关联 [B1-16 弹力](bx1?id=elastic-force)：弹力方向总是指向恢复原状。

| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “x 是弹簧总长度” | x 是形变量 | $x=l-l_{0}$ |
| “F-x 图像横轴放 F 也一样” | 斜率物理意义会改变 | 常取 x 横轴、F 纵轴 |

#### 原理、变量、单位与适用条件
牛顿第二定律：

```text
F合 = ma
```

当质量 `m` 不变时：

```text
a ∝ F
```

当合力 `F` 不变时：

```text
a ∝ 1/m
```

实验中要先平衡摩擦力，使小车受到的合力尽量等于细线拉力。

#### 装置、器材与安全
- 小车、长木板、滑轮、细线、钩码。
- 打点计时器和纸带，或位移/速度传感器。
- 天平、刻度尺。
- 垫片或可调倾角装置，用于平衡摩擦力。

#### 操作步骤
1. 调整木板倾角或补偿方式，使小车在不挂钩码时能近似匀速运动，以平衡摩擦力。
2. 保持小车质量不变，改变钩码质量，测多组加速度，作 `a-F` 图像。
3. 保持拉力近似不变，改变小车质量，测多组加速度，作 `a-1/m` 图像。
4. 每组数据中，用纸带或传感器求加速度。
5. 判断图像是否近似过原点直线。
6. 由图像斜率和实验条件检查 $F=ma$。

#### 数据处理与证据
| 模式 | 控制量 | 改变量 | 图像 | 结论 |
|---|---|---|---|---|
| 探究 `a-F` | 质量 `m` 不变 | 合力 `F` | `a-F` | 直线斜率约 `1/m` |
| 探究 `a-m` | 合力 `F` 不变 | 质量 `m` | `a-1/m` | 直线斜率约 `F` |
| 摩擦补偿 | 补偿充分 | 看截距 | 是否过原点 | 判断系统误差 |

#### 不确定度、失效情形与改进
- 未平衡摩擦力会使实际合力小于细线拉力。
- 钩码质量不够小，细线拉力不严格等于钩码重力。
- 滑轮摩擦和细线质量会影响拉力。
- 小车质量改变时，若没有重新检查摩擦补偿，会引入系统误差。
- 纸带求加速度本身有读数和作图误差。

#### 结论与迁移
- 对应 [B1-24 牛顿第二定律](bx1?id=newton-second)：实验验证 $F_{\text{合}}=ma$。
- 关联 [B1-10 匀变速直线运动](bx1?id=uniform-acceleration)：用纸带或图像求加速度。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “F 是任意外力” | 牛顿第二定律中 F 是合力 | 先平衡摩擦、求合力 |
| “研究 a-m 直接画 a-m 图” | a 与 m 成反比，不是正比 | 画 `a-1/m` |
| “不需要平衡摩擦力” | 摩擦会造成截距和系统误差 | 先补偿摩擦 |
| “钩码越重越好” | 拉力近似会变差 | 钩码质量应较小 |
| “图像不过原点一定是偶然误差” | 可能是摩擦未补偿 | 检查系统误差 |

#### 原理、变量、单位与适用条件
若重物下落过程中空气阻力、纸带摩擦等阻力很小，只有重力做功，则机械能守恒。取释放点为零速度处，测点下落高度为 `y`：

```text
ΔEp = mgy
ΔEₖ = 1/2 mv²
```

纸带上相邻点时间间隔为 `T`。第 `n` 点的瞬时速度不能用“第 n 点的位置除以时间”，应取前后两点的平均速度近似：

```text
v_n ≈ (y_{n+1}-y_{n-1})/(2T)
```

若 $\Delta E_{k}$ 与 $\Delta E_{p}$ 在误差范围内接近，就说明实验支持机械能守恒。

#### 装置、器材与安全
- 打点计时器、电源、纸带、复写纸。
- 重物、铁架台、夹子。
- 刻度尺。
- 天平或已知质量的重物。

#### 操作步骤
1. 将打点计时器固定在铁架台上，使纸带竖直穿过限位孔。
2. 纸带下端连接重物，手扶重物使纸带绷直。
3. 先接通电源，使打点计时器稳定工作。
4. 再释放重物，让重物带动纸带下落。
5. 取点迹清晰的一段纸带，选择起点和测量点。
6. 测出测点到起点的距离 $y_n$，以及前后两点位置 $y_{n-1}$、$y_{n+1}$。
7. 计算 $v_n\approx (y_{n+1}-y_{n-1})/(2\ \mathrm{T})$。
8. 分别计算 $\Delta E_{p}=mgy_n$、$\Delta E_{k}=1/2mv_n^{2}$，比较两者并求相对误差。

#### 数据处理与证据
| 记录量 | 从哪里读 | 用来计算什么 |
|---|---|---|
| 打点周期 `T` | 电源频率或实验设定 | 相邻点时间间隔 |
| 测点下落高度 $y_n$ | 纸带上起点到第 n 点距离 | $\Delta E_{p}=mgy_n$ |
| 前后点位置 $y_{n-1}$、$y_{n+1}$ | 测点前后一格 | $v_n\approx (y_{n+1}-y_{n-1})/(2\ \mathrm{T})$ |
| 重物质量 `m` | 天平或标称值 | $\Delta E_{p}$、$\Delta E_{k}$ |
| 相对误差 | 计算得到 | $\vert \Delta E_{p}-\Delta E_{k}\vert /\Delta E_{p}\times 100%$ |

判断时不要只看两个数“差不多”，要看相对误差是否在实验允许范围内；若始终 $\Delta E_{k}<\Delta E_{p}$，通常说明阻力和纸带摩擦带走了一部分机械能。

#### 不确定度、失效情形与改进
- 纸带与打点计时器之间有摩擦，使动能增加量偏小。
- 空气阻力会使一部分机械能转化为内能。
- 起点选择不准，会直接影响下落高度 `y`。
- 测量点过低时，点迹可能模糊或超出有效纸带范围。
- 用前后两点平均速度近似瞬时速度，本身存在近似误差。
- 刻度尺读数、点迹粗细和纸带倾斜都会带来读数误差。

- 对应 [B2-21 机械能守恒](bx2?id=mechanical-energy)：实验检验 `Eₖ+Ep` 是否守恒。
- 关联 [B1-05 平均速度与瞬时速度](bx1?id=velocity)：用纸带前后点平均速度近似瞬时速度。
- 关联 [B2-19 重力势能](bx2?id=gravitational-potential-energy)：用 `mgy` 表示重力势能减少量。

| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “直接用 v=gt 验证” | 这样等于借用了自由落体结论 | 应用纸带位置算速度 |
| “某点速度等于 y_n/(nT)” | 这是从起点到该点的平均速度 | 用前后两点求瞬时速度 |
| “ΔEp 一定等于 ΔEₖ” | 实验有阻力和读数误差 | 比较是否在误差范围内接近 |
| “质量越大越容易守恒” | 两边都有 m，质量主要影响阻力相对作用 | 关注能量比值和误差 |
| “先放手再通电” | 会漏掉开始下落的点迹 | 先通电后释放 |

| 竖直坐标 `y` | 坐标纸读数 | $2y/t^{2}$ 是否接近 `g` |
| 桌面高度 `h` | 由发射点到地面测量 | 判断测点是否落地前有效 |
| 发射口水平情况 | 调平和轨迹检查 | 排除竖直初速度 |

处理数据时，若 `x/t` 基本不变，说明水平速度基本恒定；若 $2y/t^{2}$ 接近 $9.8\ \mathrm{m/s^{2}}$，说明竖直方向符合自由落体。

#### 不确定度、失效情形与改进
- 发射口未调平会使小球带有竖直初速度。
- 坐标原点或竖直方向确定不准，会影响所有坐标。
- 轨迹点较粗或重复释放不一致，会造成读数偏差。
- 空气阻力会使水平速度略有减小。
- 桌面高度不够时，后面的测点已经落地，不能再用于验证。

**适用边界。**
| 能用 | 不能用或需重做 |
|---|---|
| 发射口水平，离口后小球只受重力 | 发射口上仰或下俯，竖直方向带初速度 |
| 坐标原点在离开发射口的位置，竖直方向由铅垂线确定 | 原点偏移或坐标纸歪斜 |
| 选取落地前的清晰频闪点 | 测点已低于桌面高度或点迹重叠模糊 |
| 相邻闪光时间 $\Delta t$ 稳定 | 频闪时间不等，不能用 $t=n\Delta t$ |
| 空气阻力可忽略，速度不太大 | 小球太轻或速度太大，水平速度明显衰减 |
- 斜槽末端若不光滑或有碰撞，会改变初速度方向。

#### 结论与迁移
- 对应 [B2-03 平抛运动](bx2?id=projectile-motion)：实验说明平抛可分解为水平匀速和竖直自由落体。
- 关联 [B2-02 运动合成与分解](bx2?id=motion-composition)：把一个曲线运动拆成两个互相垂直的分运动。
- 关联 [B1-12 自由落体](bx1?id=free-fall)：竖直方向符合 $y=1/2gt^{2}$。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “平抛速度不变” | 合速度方向和大小都在变 | 只有水平分速度不变 |
| “轨迹像抛物线就算验证了” | 还要用坐标数据验证分运动 | 看 `x/t` 和 $2y/t^{2}$ |
| “落地时间由 v0 决定” | 竖直方向决定落地时间 | 用高度求时间 |
| “发射角小误差不重要” | 竖直初速度会破坏自由落体关系 | 先调平发射口 |
| “后面点越多越好” | 落地后或模糊点无效 | 只取有效清晰点 |

#### 原理、变量、单位与适用条件
匀速圆周运动的加速度指向圆心：

```text
a向 = ω²r = v²/r
```

由牛顿第二定律，提供这个加速度的合力大小为：

```text
F向 = ma向 = mω²r = mv²/r
```

实验中通常固定两个量，改变第三个量：固定 $r、\omega$ 改变 `m`，看 `F-m` 图像；固定 $m、\omega$ 改变 `r`，看 `F-r` 图像；固定 `m、r` 改变 $\omega$，看 $F-\omega ^{2}$ 图像。

#### 装置、器材与安全
- 向心力演示器或转台。
- 小球、质量片、转臂或细线。
- 测力计、弹簧片或力传感器。
- 计时器、转速计。
- 刻度尺。

#### 操作步骤
1. 安装小球和转臂，确定转动半径 `r`。
2. 调整转台，使小球做稳定匀速圆周运动。
3. 固定 `r` 和 $\omega$，改变小球质量，记录向心力读数。
4. 固定 `m` 和 $\omega$，改变半径，记录读数。
5. 固定 `m` 和 `r`，改变转速，由周期或转速求 $\omega$，记录读数。
6. 分别作 `F-m`、`F-r`、$F-\omega ^{2}$ 图像。
7. 判断图像是否近似过原点直线，并整理出 $F=m\omega ^{2}r$。

#### 数据处理与证据
| 探究关系 | 控制量 | 改变量 | 作图 | 结论 |
|---|---|---|---|---|
| `F` 与 `m` | $r、\omega$ 不变 | `m` | `F-m` | 直线，$F\propto m$ |
| `F` 与 `r` | $m、\omega$ 不变 | `r` | `F-r` | 直线，$F\propto r$ |
| `F` 与 $\omega$ | `m、r` 不变 | $\omega$ | $F-\omega ^{2}$ | 直线，$F\propto \omega ^{2}$ |
| 综合 | 全部读数 | $m\omega ^{2}r$ | $F-(m\omega ^{2}r)$ | 斜率接近 1 |

若图像明显不过原点，要检查零点、摩擦、转速是否稳定，以及是否把 $\omega$ 与 $\omega ^{2}$ 混作一类。

#### 不确定度、失效情形与改进
- 转速不稳定会使 $\omega$ 读数不准，且误差会被平方放大。
- 转动半径读错会直接影响理论值。
- 测力计零点未校准会导致图像有截距。
- 小球不在水平面内运动时，测得的力不完全是向心方向合力。
- 摩擦和空气阻力会带来附加力。
- 用周期求角速度时，少数圈计时误差较大，应多转几圈取平均。

- 关联 [B2-05 圆周运动基本量](bx2?id=circular-motion)：用 $v=\omega r$、$\omega =2\pi /T$ 转换物理量。

#### 原理、变量、单位与适用条件
设每个小球受到大球引力：

```text
F = GMm/r²
```

两端小球对扭丝产生同向转动力矩，若小球到悬点距离为 `L`，总引力矩近似为：

```text
τ引 = 2FL
```

扭丝扭转角为 $\theta$ 时，恢复力矩为：

```text
τ扭 = κθ
```

平衡时：

```text
κθ = 2FL
G = κθr²/(2MmL)
```

#### 装置、器材与安全
- 卡文迪许扭秤、扭丝和轻杆。
- 两个小球、两个大球。
- 光源、反射镜、标尺或光电读数装置。
- 测量距离的刻度装置。
- 防风罩、减振装置。

#### 操作步骤
1. 调整扭秤，使轻杆能稳定静止，记录零点。
2. 将大球放到规定位置，等待系统缓慢达到新平衡。
3. 读出光斑位移或扭转角 $\theta$。
4. 测量球心距离 `r`、大小球质量 `M、m`，记录扭臂长度 `L`。
5. 用已知方法标定扭丝扭转系数 `κ`。
6. 代入 $G=\kappa \theta r^{2}/(2MmL)$ 计算万有引力常数。
7. 改变大球位置重复测量，取平均并分析误差。

#### 数据处理与证据
| 记录量 | 物理意义 | 用在公式中 |
|---|---|---|
| `M、m` | 大球、小球质量 | $G=\kappa \theta r^{2}/(2MmL)$ |
| `r` | 大小球球心距离 | 平方项，误差影响很大 |
| `L` | 小球到悬点的力臂 | 引力矩 `2FL` |
| `κ` | 扭丝扭转系数 | 恢复力矩 $\kappa \theta$ |
| $\theta$ | 平衡扭转角 | 由光斑位移换算 |

由于 `r` 出现在平方项中，球心距离测量是关键；由于 $\theta$ 很小，必须防止气流、振动和读数漂移。

#### 不确定度、失效情形与改进
- 球心距离 `r` 测量不准会造成较大相对误差。
- 扭丝系数 `κ` 标定不准会整体改变 `G`。
- 扭转角很小，光斑读数和零点漂移影响明显。
- 外界振动、空气流动、温度变化会使扭秤难以平衡。
- 大小球不是理想质点，球心位置和附近物体引力都需控制。
- 等待时间不足时，读到的是振动过程中的瞬时位置，不是平衡位置。

#### 结论与迁移
- 对应 [B2-09 万有引力定律](bx2?id=gravitation)：实验测定公式中的常量 `G`。
- 关联 [B1-24 牛顿第二定律](bx1?id=newton-second)：由力和运动状态变化理解极小引力的测量。
- 关联 [B2-06 向心力](bx2?id=centripetal)：万有引力在天体问题中常提供向心力。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “G 是计算出来的定义值” | G 来自实验测量 | 理解扭秤测量过程 |
| “大球越远越安全、误差更小” | 引力随 $1/r^{2}$ 减小，信号会更弱 | 在可控距离内测量 |
| “只要读到角度就能算 G” | 还需要 `κ、M、m、r、L` | 数据必须配套 |
| “θ 越大 G 越大” | θ 还受 M、m、r、κ 影响 | 用完整公式反算 |
| “扭秤马上停下就读数” | 系统会缓慢振荡 | 等平衡或用振动法处理 |

#### 原理、变量、单位与适用条件
真空中两个静止点电荷之间的库仑力大小为：

```text
F = k|q1q2|/r²
```

方向沿两电荷连线：

```text
同号电荷相斥，异号电荷相吸
```

实验验证时常用两组图像：

```text
固定 r：F ∝ |q1q2|
固定 q1、q2：F ∝ 1/r²
```

也可由测量值反查：

```text
k ≈ Fr²/|q1q2|
```

#### 装置、器材与安全
- 库仑扭秤或静电力测量装置。
- 两个导体小球、绝缘支架。
- 起电机或电荷转移装置。
- 距离调节与读数装置。
- 防潮、防风、防漏电装置。

#### 操作步骤
1. 调整扭秤零点，保证小球能稳定静止。
2. 给两个小球带已知或可比较的电荷量。
3. 固定球心距离 `r`，改变电荷量组合，记录静电力读数。
4. 固定电荷量，改变球心距离，记录读数。
5. 分别作 $F-\vert q_{1}q_{2}\vert$ 图像和 $F-1/r^{2}$ 图像。
6. 用 $k\approx Fr^{2}/\vert q_{1}q_{2}\vert$ 反查静电力常量。
7. 分析漏电、距离和球体大小对实验的影响。

#### 数据处理与证据
| 探究关系 | 控制量 | 改变量 | 作图 | 结论 |
|---|---|---|---|---|
| `F` 与电荷量 | `r` 不变 | $\vert q_{1}q_{2}\vert$ | $F-\vert q_{1}q_{2}\vert$ | 近似正比 |
| `F` 与距离 | $q_{1}、q_{2}$ 不变 | `r` | $F-1/r^{2}$ | 近似正比 |
| 静电力常量 | 全部读数 | $F、r、q_{1}、q_{2}$ | 计算 $Fr^{2}/\vert q_{1}q_{2}\vert$ | 接近 `k` |

若图像不过原点或反查 `k` 明显偏离，要检查漏电、距离读数、球是否可看作点电荷。

#### 不确定度、失效情形与改进
- 空气潮湿或绝缘不好会导致电荷漏失。
- 球心距离 `r` 测量不准会因平方项造成明显误差。
- 两球大小不能忽略时，不能简单看作点电荷。
- 附近带电体或人体接近会改变电场分布。
- 扭秤零点漂移会造成系统误差。
- 电荷量分配不均或接触带电过程不稳定会影响 $q_{1}、q_{2}$。

#### 结论与迁移
- 对应 [B3-02 库仑定律](bx3?id=coulomb-law)：实验检验 $F=k\vert q_{1}q_{2}\vert /r^{2}$。
- 关联 [B3-03 电场强度](bx3?id=electric-field)：库仑力除以试探电荷得到电场强度。
- 关联 [B1-24 牛顿第二定律](bx1?id=newton-second)：带电小球受力后运动状态改变。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “异号电荷力大小为负” | 负号只表示方向，大小取正值 | 大小用 $\vert q_{1}q_{2}\vert$ |
| “F 与 r 成反比” | 库仑力是平方反比 | 作 $F-1/r^{2}$ 图像 |
| “带电球都能当点电荷” | 距离不够大时球大小影响明显 | 要满足点电荷近似 |
| “图像偏小一定是 k 变了” | 可能是漏电或距离误差 | 先查实验条件 |
| “方向只看哪个电荷大” | 方向由同号/异号决定 | 同号斥、异号吸 |

> **公式首次使用卡**：适用边界——等势线是电势相等的点组成的线。若两点在同一等势线上：<br>UAB = φA - φB = 0。符号含义在紧邻公式的正文、图注或表格中定义；除题目另有单位外，代入前统一为 SI，结果保留单位并做量纲检查。

- 改变目标电势：看目标等势线如何平移，并确认不同等势线不能相交。
- 增大电源电压或改变电极间距：看 $\varphi -x$ 图线斜率和 $E\approx U/d$ 如何变化。
- 沿等势线移动探针：看 $UAB=0$，所以 $W=qUAB=0$。

#### 原理、变量、单位与适用条件
等势线是电势相等的点组成的线。若两点在同一等势线上：






```text
UAB = φA - φB = 0
```

电场力做功：

```text
WAB = qUAB = 0
```

在平行电极之间，若忽略边缘效应，可近似看成匀强电场：

```text
E ≈ U/d
```

等势线与电场线垂直；若不垂直，沿等势线移动电荷时电场力就会有沿线方向分量并做功，这与 $UAB=0$ 矛盾。

#### 装置、器材与安全
- 导电纸、白纸或坐标纸。
- 两个电极、稳压电源。
- 电压表或灵敏电流计。
- 探针、导线、开关。
- 铅笔、直尺。

#### 操作步骤
1. 将导电纸固定在坐标纸上，安装电极并接入电源。
2. 选择一个参考电极为零电势点。
3. 用探针在导电纸上移动，读取该点电势。
4. 找到若干个电势相同的点，并在纸上标记。
5. 用平滑曲线连接这些同电势点，得到一条等势线。
6. 改变目标电势，重复描出多条等势线。
7. 根据等势线疏密和方向，判断电场强弱和电场线方向。

#### 数据处理与证据
| 记录量 | 从哪里读 | 用来判断什么 |
|---|---|---|
| 目标电势 $\varphi 0$ | 电压表或设定值 | 要寻找的等势线 |
| 探针坐标 `x、y` | 坐标纸 | 标出同电势点 |
| 探针电势 $\varphi$ | 电压表 | 是否满足 $\varphi \approx \varphi 0$ |
| 电源电压 `U` | 电源或电压表 | 场强估计 $E\approx U/d$ |
| 电极间距 `d` | 尺测 | 判断等势线间距 |

处理数据时，不要把一个点当成一条线。应找多个同电势点，再用平滑线连接。

#### 不确定度、失效情形与改进
- 探针接触不良会导致电压表读数跳动。
- 导电纸不均匀或潮湿程度不同会改变电势分布。
- 电极形状和边缘效应会使等势线弯曲。
- 读数时手接近导电纸可能影响电场。
- 标点过少会使连线不准确。
- 电源电压不稳定会导致同一位置读数变化。

- 对应 [B3-05 等势面](bx3?id=equipotential)：实验把“等电势点”描成可见等势线。
- 关联 [B3-03 电场强度](bx3?id=electric-field)：等势线密处场强较大，电场线垂直等势线。
- 关联 [B3-06 电势差](bx3?id=potential-difference)：用电压表读出两点电势差。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “等势线就是电场线” | 二者处处垂直，不是同一条线 | 分清方向线和等值线 |
| “一个读数点就是等势线” | 等势线由多个同电势点连成 | 多点描线 |
| “沿等势线移动不受力” | 可能受电场力，但力垂直位移 | 说不做功，不说不受力 |
| “等势线可以相交” | 交点会有两个电势值 | 等势线不能相交 |
| “边缘处仍是匀强电场” | 边缘效应使场线弯曲 | 中央区域近似匀强 |

#### 原理、变量、单位与适用条件
伏安法测电阻：

```text
R = U/I
```

圆形金属丝横截面积：

```text
S = πd²/4
```

电阻定律：

```text
R = ρL/S  →  ρ = RS/L
```

直径 `d` 出现在平方项中，所以直径读数误差会被放大；金属丝发热会使电阻变大，因此应控制电流，快速读数。

#### 装置、器材与安全
- 待测金属丝、刻度尺、螺旋测微器。
- 电源、开关、滑动变阻器。
- 电流表、电压表。
- 导线、接线柱。

#### 操作步骤
1. 用刻度尺测出金属丝接入电路的有效长度 `L`。
2. 用螺旋测微器在金属丝不同位置、不同方向多次测直径 `d`，取平均值。
3. 按伏安法连接电路，电压表并联在待测金属丝两端，电流表串联。
4. 合上开关前，将滑动变阻器调到保护位置。
5. 改变滑动变阻器，记录多组 `U、I`。
6. 由 $R=U/I$ 或 I-U 图像斜率求电阻。
7. 计算 $S=\pi d^{2}/4$，再算 $\rho =RS/L$。
8. 比较多次测量结果，分析误差。

#### 数据处理与证据
| 记录量 | 测量工具 | 计算用途 |
|---|---|---|
| 电压 `U` | 电压表 | $R=U/I$ |
| 电流 `I` | 电流表 | $R=U/I$ 或 I-U 图像 |
| 长度 `L` | 刻度尺 | $\rho =RS/L$ |
| 直径 `d` | 螺旋测微器 | $S=\pi d^{2}/4$ |
| 多组 `U-I` | 伏安法 | 减小偶然误差，检查发热 |

若 I-U 图像弯曲，说明电阻可能随温度变化，应减小电流或缩短通电时间。

#### 不确定度、失效情形与改进
- 金属丝发热会使电阻增大，导致电阻率偏大。
- 直径测量误差会因 $S=\pi d^{2}/4$ 被放大。
- 只测一个位置的直径不能反映金属丝粗细不均。
- 电压表、电流表内阻会影响伏安法测量。
- 长度应取接入电路的有效长度，不是整根金属丝长度。
- 螺旋测微器零点误差若不修正，会造成系统误差。

**适用边界。**
| 能用 | 不能用或需重做 |
|---|---|
| 电流适中，金属丝温升不明显 | 电流过大，I-U 图线弯曲，电阻随温度变 |
| 电压表并联在待测金属丝两端，电流表串联 | 电表接法错误或量程不合适 |
| 有效长度 `L` 是两接线夹之间的金属丝长度 | 把整根金属丝长度当作接入长度 |
| 直径在不同位置、不同方向多次测量取平均 | 只测一次直径，且未修正零点误差 |
| I-U 图像近似直线，可用斜率求 `R` | 图线明显弯曲，说明发热或接触不稳定 |

#### 结论与迁移
- 对应 [B3-12 电阻与电阻定律](bx3?id=resistance)：用实验测 $\rho$ 并验证 $R=\rho L/S$。
- 关联 [B3-11 电流](bx3?id=current)：通过 $I=U/R$ 理解伏安关系。
- 关联 [B3-13 欧姆定律](bx3?id=ohm-law)：用 I-U 图像求电阻。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “直接用整根线长度” | 有效长度是接入电路的长度 | 测接线柱之间的长度 |
| “直径测一次就够” | 金属丝可能不均匀，且 d 误差被平方放大 | 多点多方向测平均 |
| “电流越大越好读” | 发热会使 R 增大 | 控制电流，快速读数 |
| “S=πd²” | 圆面积公式是 $\pi (d/2)^{2}$ | 写 $S=\pi d^{2}/4$ |
| “I-U 图像弯了也照样取一点算 R” | 弯曲说明温度影响明显 | 取小电流段或减小电压 |

| 路端电压 `U` | 电压表 | 纵轴坐标 |
| 多组 `U、I` | 改变外阻得到 | 拟合 U-I 直线 |
| 纵截距 | 直线延长到 U 轴 | 电动势 `E` |
| 斜率 | $\Delta U/\Delta I$ | `-r` |

如果只取两组数据，偶然误差会明显影响斜率；应尽量取多组分布较均匀的数据点。

#### 不确定度、失效情形与改进
- 电压表和电流表内阻会带来系统误差。
- 电源长时间大电流工作会发热，使内阻变化。
- 读数点过少会使拟合直线不稳定。
- 电流过大时，路端电压下降明显且电源状态可能改变。
- 图像纵截距需要外推，作图误差会影响 `E`。
- 滑动变阻器接触不良会造成读数跳动。

#### 结论与迁移
- 对应 [B3-16 闭合电路欧姆定律](bx3?id=closed-circuit)：实验验证 $U=E-Ir$。
- 关联 [B3-17 电源电动势](bx3?id=emf)：U-I 图像纵截距对应电源电动势。
- 关联 [B3-13 欧姆定律](bx3?id=ohm-law)：外电路满足 $U=IR$。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “路端电压就是电动势” | 接外电路时 $U=E-Ir$ | 断路时 U 才接近 E |
| “U-I 图像斜率就是 r” | 斜率为负，$r=\vert \text{斜率}\vert$ | 注意符号 |
| “一组 U、I 就能精确求 E、r” | 两个未知量需多组数据减小误差 | 作图拟合 |
| “电流越大数据越明显” | 大电流会使电源发热和状态改变 | 控制电流 |
| “横纵轴随便画” | 常规图像是横轴 I、纵轴 U | 统一图像含义 |

#### 原理、变量、单位与适用条件
若两小车组成系统，在水平方向合外力近似为零，则碰撞过程中系统总动量守恒：

```text
p前 = m1v1 + m2v2
p后 = m1v1′ + m2v2′
p前 ≈ p后
```

速度是矢量，一维实验中必须规定正方向，反向速度要取负值。弹性碰撞中动能也近似守恒；完全非弹性碰撞中动能减少，但动量仍可守恒。

#### 装置、器材与安全
- 气垫导轨或低摩擦轨道。
- 两辆小车、碰撞弹簧或粘扣。
- 光电门、挡光片、计时器。
- 天平、刻度尺。

#### 操作步骤
1. 调平导轨，使小车能近似匀速运动。
2. 用天平测两辆小车质量 $m_{1}、m_{2}$。
3. 安装光电门和挡光片，测出碰前速度 $v_{1}、v_{2}$。
4. 让两车发生碰撞，测出碰后速度 $v_{1}′、v_{2}′$。
5. 规定正方向，给所有速度带正负号。
6. 分别计算 $p_{\text{前}}$ 和 $p_{\text{后}}$。
7. 求相对差，判断是否在允许误差范围内。
8. 可同时计算动能，判断碰撞类型。

#### 数据处理与证据
| 记录量 | 测量方法 | 用途 |
|---|---|---|
| $m_{1}、m_{2}$ | 天平 | 计算各车动量 |
| $v_{1}、v_{2}$ | 光电门或纸带 | 碰前动量 |
| $v_{1}′、v_{2}′$ | 光电门或纸带 | 碰后动量 |
| 正方向 | 实验前规定 | 决定速度正负 |
| 相对差 | $\vert p_{\text{后}}-p_{\text{前}}\vert /\vert p_{\text{前}}\vert$ | 判断实验是否通过 |

处理时不能只比较速度，要比较 `mv` 的代数和。

#### 不确定度、失效情形与改进
- 导轨未调平，重力分力会形成外力。
- 摩擦和空气阻力使系统水平合外力不为零。
- 光电门位置或挡光片宽度误差会影响速度。
- 小车碰撞不正，会出现转动或侧向运动。
- 质量测量不准会直接影响动量。
- 速度正负号写错会导致总动量计算错误。

**适用边界。**
| 能用 | 不能用或需重做 |
|---|---|
| 导轨调平，水平方向合外力近似为零 | 导轨倾斜或摩擦明显，小车速度持续衰减 |
| 两车沿同一直线正碰 | 斜碰、侧向摆动或碰后转动明显 |
| 光电门能分别读出碰前、碰后速度 | 光电门触发混乱，无法区分碰前后速度 |
| 速度统一规定正方向并带正负号 | 只用速度大小相加，忽略方向 |
| 相对差在允许误差范围内 | $\vert p_{\text{后}}-p_{\text{前}}\vert /\vert p_{\text{前}}\vert$ 过大，不能支持守恒结论 |

#### 结论与迁移
- 对应 [X1-04 动量守恒定律](xb1?id=momentum-conservation)：实验检验 $p_{\text{前}}\approx p_{\text{后}}$。
- 关联 [X1-01 动量](xb1?id=momentum)：动量是矢量，$p=mv$。
- 关联 [X1-05 弹性碰撞](xb1?id=elastic-collision)：弹性碰撞还要检查动能。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “速度相等才守恒” | 守恒的是总动量，不是速度 | 比较 $m_{1}v_{1}+m_{2}v_{2}$ |
| “碰撞后动能减少就不守恒” | 非弹性碰撞动量仍可守恒 | 分开看动量和动能 |
| “速度不用带方向” | 动量是矢量 | 规定正方向，反向取负 |
| “导轨有点斜没关系” | 斜面分力是外力 | 先调平导轨 |
| “只测一次就可靠” | 偶然误差较大 | 多次实验取平均 |

#### 原理、变量、单位与适用条件
小角度摆动时，单摆周期近似为：

```text
T = 2π√(L/g)
```

两边平方：

```text
T² = (4π²/g)L
```

所以作 $T^{2}-L$ 图像应为过原点直线，斜率为 $4\pi ^{2}/g$，可反求：

```text
g = 4π²L/T²
```

#### 装置、器材与安全
- 铁架台、细线、小球。
- 米尺、游标卡尺或直尺。
- 秒表。
- 量角器。

#### 操作步骤
1. 固定细线和小球，测出悬点到球心的摆长 `L`。
2. 让小球偏离平衡位置一个小角度后释放，不要推。
3. 经过平衡位置时开始计时，减少起点判断误差。
4. 记录 `N` 次全振动总时间 `t`。
5. 计算周期 $T=t/N$。
6. 改变摆长，重复测量多组数据。
7. 作 $T^{2}-L$ 图像，判断是否为直线。
8. 用斜率或 $g=4\pi ^{2}L/T^{2}$ 反算重力加速度。

#### 数据处理与证据
| 记录量 | 测量方法 | 用途 |
|---|---|---|
| 摆长 `L` | 悬点到球心 | 周期公式自变量 |
| 总时间 `t` | 秒表测 N 次全振动 | $T=t/N$ |
| 次数 `N` | 人工计数 | 减小计时相对误差 |
| 周期 `T` | 计算得到 | 作 $T^{2}-L$ 图像 |
| 摆角 $\theta$ | 量角器或估计 | 检查小角度条件 |
| 斜率 `k` | $T^{2}-L$ 图像求得 | $g=4\pi ^{2}/k$ |
| 重力加速度 `g` | 由周期或斜率反算 | 与当地标准值比较 |

数据处理时应作 $T^{2}-L$ 图像，而不是直接作 `T-L` 图像来判断正比。若使用单组数据，可用 $g=4\pi ^{2}L/T^{2}$；若使用多组数据，应优先用图像斜率：

```text
k = ΔT² / ΔL
g = 4π² / k
```

#### 不确定度、失效情形与改进
- 摆长若只量线长，会漏掉小球半径。
- 摆角过大，小角度近似不成立。
- 人反应时间会影响单次计时，测多次全振动可减小相对误差。
- 释放时若给了初速度，会影响摆动状态。
- 空气阻力和悬点摩擦会使振幅逐渐变小。
- 计数全振动次数时漏数或多数会造成明显错误。

#### 原理、变量、单位与适用条件
光从空气射入玻璃时，满足折射定律：

```text
sin i / sin r = n
```

其中 `i` 是空气中的入射角，`r` 是玻璃中的折射角，二者都相对法线测量。若角度太小，读数误差会使 `sin i/sin r` 的相对误差变大。

#### 装置、器材与安全
- 玻璃砖、白纸、铅笔、直尺。
- 大头针若干。
- 量角器。

#### 操作步骤
1. 将玻璃砖放在白纸上，描出两条平行边界。
2. 在玻璃砖一侧插两枚针，确定入射线。
3. 透过玻璃砖观察前两针的像，在另一侧插两枚针，使四针看起来在同一直线上。
4. 取下玻璃砖和大头针，连接入射线、玻璃内折射线和出射线。
5. 在入射点作法线。
6. 用量角器测入射角 `i` 和折射角 `r`。
7. 代入 $n=\sin i/\sin r$，多次测量取平均。

#### 数据处理与证据
| 记录量 | 获取方式 | 用途 |
|---|---|---|
| 入射角 `i` | 量角器，相对法线 | 计算 `sin i` |
| 折射角 `r` | 量角器，相对法线 | 计算 `sin r` |
| 插针间距 | 尺测或实验设定 | 判断光线方向精度 |
| 折射率 `n` | `sin i/sin r` | 实验结果 |
| 多组角度 | 改变入射角 | 取平均减小误差 |

也可作纵轴 `sin i`、横轴 `sin r` 的图像，斜率约为玻璃对空气的折射率；若交换横纵轴，斜率将变为其倒数 `1/n`。

#### 不确定度、失效情形与改进
- 玻璃砖边界描线不准会影响入射点和法线。
- 大头针间距太近，确定光线方向误差大。
- 量角器读数有视差。
- 角度太小，计算折射率相对误差大。
- 玻璃砖移动后未重新描边会导致光路错位。
- 针没有竖直插入会影响对准。

| 能用 | 不能用或需重做 |
|---|---|
| 玻璃砖边界清楚，入射点和法线画准 | 玻璃砖移动后仍用旧边界，法线位置错误 |
| 入射角适中，通常不小于约 $25^{\circ}$ | 入射角太小，角度读数误差被 `sin` 比值放大 |
| 两枚针间距足够大，方向线稳定 | 插针太近，连线方向不可靠 |
| 针竖直插入，观察四针共线 | 针歪斜或视线没有对准，出射线偏差大 |

#### 原理、变量、单位与适用条件
相邻亮纹对应光程差相差一个波长。小角度条件下，屏上相邻亮纹间距近似为：

```text
Δx = Lλ/d
λ = dΔx/L
```

其中 `L` 是双缝到屏的距离，`d` 是双缝间距，$\Delta x$ 是相邻亮纹间距。因为单个 $\Delta x$ 很小，实验中常测 `N` 个条纹间距的总宽度 $N\Delta x$，再用 $\Delta x=(N\Delta x)/N$。

#### 装置、器材与安全
- 单色光源、滤光片或激光。
- 双缝、遮光筒、接收屏。
- 测量头或刻度尺。
- 光具座。

#### 操作步骤
1. 调整光源、单缝、双缝和屏，使条纹清晰且中央亮纹位于视野中央。
2. 记录双缝间距 `d` 和双缝到屏的距离 `L`。
3. 选择若干相邻亮纹，测出从第 1 条到第 `N+1` 条亮纹的距离 $N\Delta x$。
4. 计算 $\Delta x=(N\Delta x)/N$。
5. 代入 $\lambda =d\Delta x/L$ 求波长。
6. 改变测量范围，多次测量取平均。

#### 数据处理与证据
| 记录量 | 获取方式 | 用途 |
|---|---|---|
| `d` | 双缝规格 | 代入波长公式 |
| `L` | 测双缝到屏距离 | 决定条纹宽度 |
| `N` | 选取相邻条纹个数 | 平均条纹间距 |
| $N\Delta x$ | 测量多条条纹总宽度 | 求 $\Delta x$ |
| $\lambda$ | $d\Delta x/L$ | 实验结果 |

单位必须统一。常用做法是把 `d`、$\Delta x$ 换成米，最后把 $\lambda$ 换成纳米。

#### 不确定度、失效情形与改进
- 条纹太密时，亮纹中心位置难以判断。
- `L` 测量起点应从双缝位置到屏，不能从光源开始。
- 只测一个条纹间距，读数误差占比大。
- 亮纹较宽时，应取亮纹中心位置。
- 双缝、屏和光源未调正会使条纹倾斜或模糊。
- 使用白光时不同波长条纹重叠，不适合直接用单色光公式处理。

#### 结论与迁移
- 对应 [X1-21 光的干涉](xb1?id=interference)：双缝干涉是波动性的直接证据。
- 关联 [X1-18 光的传播](xb1?id=light-propagation)：光路和屏距决定几何位置。
- 关联 [X1-22 光的衍射](xb1?id=diffraction)：狭缝越窄衍射越明显，才能让两缝光束重叠。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “直接量一条亮纹宽度” | 公式用的是相邻亮纹中心距 | 找亮纹中心到亮纹中心 |
| “L 是光源到屏距离” | 公式中的 L 是双缝到屏距离 | 从双缝位置量到屏 |
| “d 越大条纹越宽” | $\Delta x=L\lambda /d$，d 越大 Δx 越小 | 看反比例 |
| “测一格就够” | 单次读数相对误差大 | 测多条条纹总宽 |
| “白光条纹都等价” | 白光含多种波长 | 用单色光测量 |

#### 原理、变量、单位与适用条件
油酸溶液被稀释后，一滴溶液中只有很少的纯油酸。若一滴溶液体积为 $V_{\text{滴}}$，稀释倍数为 `k`，滴数为 `n`，则纯油酸体积为：

```text
V = nV滴/k
```

若油酸在水面上形成单分子层，油膜面积为 `S`，则：

```text
d = V/S
```

这里 `d` 是油膜厚度，也是油酸分子直径的数量级估计。这个实验追求的是 $10^-10\ \mathrm{m}$ 量级，而不是精确测出某个分子的真实形状。

#### 装置、器材与安全
- 油酸酒精溶液、滴管。
- 浅盘、水、痱子粉或细粉。
- 坐标纸或透明方格纸。
- 量筒或移液管。

#### 操作步骤
1. 配制已知稀释倍数的油酸酒精溶液。
2. 用量筒测出若干滴溶液的总体积，求每滴体积 $V_{\text{滴}}$。
3. 在浅盘水面均匀撒少量细粉。
4. 向水面滴入一滴油酸溶液，等待油膜充分铺展。
5. 描出油膜轮廓，放到方格纸上估算面积 `S`。
6. 由 $V=nV_{\text{滴}}/k$ 求纯油酸体积。
7. 代入 $d=V/S$，得到分子直径数量级。

#### 数据处理与证据
| 记录量 | 获取方式 | 用途 |
|---|---|---|
| 稀释倍数 `k` | 配液记录 | 求纯油酸体积 |
| 每滴体积 $V_{\text{滴}}$ | 多滴总体积除以滴数 | 减小滴液体积误差 |
| 滴数 `n` | 实际滴入水面的滴数 | 求总纯油酸体积 |
| 油膜面积 `S` | 方格法估算 | 代入 $d=V/S$ |
| 分子直径 `d` | 计算得到 | 判断数量级 |

数格时，完整方格直接计入，不完整方格可按超过半格计一格、不足半格舍去，或多次描边取平均。

#### 不确定度、失效情形与改进
- 油酸没有形成单分子层，会使估算偏离真实分子尺度。
- 稀释倍数记错或溶液不均匀，会直接影响 `V`。
- 每滴体积不能只量一滴，应测多滴总体积再平均。
- 油膜边界不规则，数格会带来面积误差。
- 油膜面积偏大时，由 $d=V/S$ 得到的 `d` 偏小。
- 水面粉末过多或水面不洁，会阻碍油膜铺展。

**适用边界。**
| 能用 | 不能用或需重做 |
|---|---|
| 油酸充分稀释，滴入后能铺成近似单分子层 | 油滴太大或未稀释充分，形成多分子层 |
| 水面干净，粉末少量且分布均匀 | 粉末太厚、水面有油污，油膜无法自由铺展 |
| 油膜边界清楚，可用方格法估算面积 | 边界模糊、断裂或被容器边缘截断 |
| 每滴体积由多滴总体积平均得到 | 只估一滴体积，滴管残留导致体积误差大 |
| 结果在 $10^-10\ \mathrm{m}$ 数量级 | 算出 $10^-8\ \mathrm{m}$ 或 $10^-12\ \mathrm{m}$，通常是单位或面积错误 |

#### 结论与迁移
- 对应 [X3-01 分子动理论基本观点](xb3?id=molecular-kinetic-theory)：物质由大量分子组成，分子尺度极小。
- 关联 [X3-02 阿伏伽德罗常数](xb3?id=avogadro)：微观数量级估算需要从宏观量推微观量。
- 关联 [X3-03 分子热运动](xb3?id=brownian-motion)：实验体现微观模型不能直接看见，需要用宏观可测量量间接推断。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “一滴溶液体积就是纯油酸体积” | 溶液被稀释，只有一部分是油酸 | 先除以稀释倍数 |
| “油膜厚度是水深” | 厚度指水面上油酸单分子层厚度 | 用 $d=V/S$ |
| “面积越大分子越大” | $d=V/S$，V 一定时 S 越大 d 越小 | 看反比例 |
| “只量一滴体积最直接” | 一滴体积太小，误差大 | 测多滴总体积取平均 |
| “结果要非常精确” | 该实验是数量级估算 | 重点判断 $10^-10\ \mathrm{m}$ 量级 |

```

判断顺序固定为：先判断原磁场方向，再判断穿过线圈的磁通量增大还是减小，最后让感应电流产生的磁场去“反着变化趋势”。

5. 改变速度和线圈匝数，比较偏转幅度。
6. 把每一次的磁极、运动、磁通量增减、电流计偏转和结论写入表格。

| 1 | N 极 | 插入 | 向右磁通量增大 | 右偏或左偏，按接法记录 | N 极 | 排斥靠近的 N 极 |

```

- 电流计接线方向不同，左偏和右偏的约定会互换，所以实验结论不能只背偏转方向。
- 磁铁运动速度不稳定，会使偏转幅度忽大忽小。
- 线圈与磁铁没有共轴，穿过线圈的磁通量变化会变小。
- 磁铁离线圈太远时磁通量变化很弱，指针偏转不明显。
- 指针未调零会影响方向判断和幅度比较。

| 易错说法 | 为什么错 | 改法 |

| “匝数变多方向会变” | 匝数只放大 `e` 的大小 | 方向仍由楞次定律决定 |

#### 原理、变量、单位与适用条件
原线圈接交流电，铁芯中磁通量不断变化，副线圈中产生感应电动势。理想变压器中：

```text
U1 / U2 = n1 / n2
U2 = U1 × n2 / n1
I1 / I2 = n2 / n1
P1 ≈ P2
```

本实验主要验证电压与匝数的比例关系，所以更适合轻载或空载测量，减少线圈电阻和铁芯损耗影响。

#### 操作步骤
1. 选择低压交流电源，连接原线圈，电压表并联在原线圈两端。
2. 在铁芯另一侧套上副线圈，电压表并联在副线圈两端。
3. 固定 $U_{1}$ 和 $n_{1}$，先记录一次 $n_{2}$ 与 $U_{2}$。
4. 逐次更换或调节副线圈匝数，读出 $U_{2}$。
5. 计算 $U_{2}/U_{1}$ 和 $n_{2}/n_{1}$，比较两者。
6. 以 $n_{2}$ 为横轴、$U_{2}$ 为纵轴作图，求斜率并与 $U_{1}/n_{1}$ 比较。

#### 数据处理与证据
| 次数 | n1 | U1/V | n2 | U2/V | n2/n1 | U2/U1 |
|---:|---:|---:|---:|---:|---:|---:|
| 1 | 800 | 12.0 | 200 | 3.0 | 0.250 | 0.250 |
| 2 | 800 | 12.0 | 400 | 6.0 | 0.500 | 0.500 |
| 3 | 800 | 12.0 | 600 | 9.0 | 0.750 | 0.750 |

作图处理：

```text
横轴：n2
纵轴：U2
斜率 k = ΔU2 / Δn2
理论斜率 k0 = U1 / n1
若 k 接近 k0，说明 U2 与 n2 成正比
```

#### 不确定度、失效情形与改进
- 线圈电阻导致端电压与理想感应电动势略有差别。
- 铁芯漏磁使副线圈获得的磁通量小于理想值。
- 铁芯涡流和磁滞会造成能量损耗。
- 电压表内阻有限、负载较重时，会拉低副线圈端电压。
- 使用直流电源时稳定后磁通量不变，不能得到持续变压效果。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “变压器能把直流电压也变高” | 稳定直流不产生变化磁通 | 变压器只适用于交流或变化电流 |
| “升压就是功率变大” | 理想变压器近似功率守恒 | 升压必降流 |
| “电压比和匝数比反着写” | 电压与匝数成正比 | $U_{1}/U_{2}=n_{1}/n_{2}$ |
| “用瞬时值随便代入” | 电压表读数通常是有效值 | 变压器计算用有效值 |

4. 缓慢改变光照或温度，记录传感器电压和执行器状态。
5. 调整阈值和滞回宽度，比较控制灵敏度和稳定性。

#### 原理、变量、单位与适用条件
一定质量气体在温度不变时满足玻意耳定律：






```text
PV = C
P = C · 1/V
P-V 图像：反比例曲线
P-1/V 图像：过原点附近的直线，斜率约为 C
```

因此实验既可以比较多组 `PV` 是否接近常量，也可以作 `P-1/V` 图像并用斜率验证。

#### 操作步骤
1. 封闭一定质量气体，检查装置是否漏气。
2. 把装置放入恒温环境，等待温度稳定。
3. 缓慢移动活塞到某一体积刻度，等待压强表示数稳定。
4. 记录体积 `V` 和压强 `P`。
5. 改变体积，重复记录至少 5 组数据。
6. 计算 `1/V` 和 `PV`，作 `P-V` 与 `P-1/V` 图像。

#### 数据处理与证据
| 次数 | V/mL | P/kPa | 1/V/mL^-1 | PV/(kPa·mL) |
|---:|---:|---:|---:|---:|
| 1 | 80.0 | 50.0 | 0.0125 | 4000 |
| 2 | 50.0 | 80.0 | 0.0200 | 4000 |
| 3 | 40.0 | 100.0 | 0.0250 | 4000 |
| 4 | 25.0 | 160.0 | 0.0400 | 4000 |

处理方法：

```text
逐行计算 PV
若 PV 基本不变，说明 P 与 V 成反比
作 P-1/V 图像，斜率 k = ΔP / Δ(1/V) ≈ PV
```

#### 不确定度、失效情形与改进
- 快速压缩气体会使温度暂时升高，压强偏大。
- 活塞与筒壁摩擦会使读数滞后。
- 装置漏气会使一定质量条件被破坏。
- 压强表零点不准会带来系统误差。
- 体积读数要扣除连接管等死体积，否则小体积时误差明显。

#### 故障诊断与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “P 与 V 成正比” | 等温时 `PV` 恒定，P 与 V 反比 | 看 `P-1/V` 直线 |
| “压缩越快越好” | 快速压缩会升温 | 缓慢移动并等待稳定 |
| “只画 P-V 就够了” | 反比例曲线不如直线便于判断 | 同时作 `P-1/V` 图 |
| “漏气只影响一次数据” | 漏气改变气体质量，整组规律被破坏 | 先检查密封 |

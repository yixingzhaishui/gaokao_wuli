## 目标

把网站从“按钮可点击、自动测试通过”提升到“学生能看见真实物理过程、能操作变量、能读出变化、能理解公式”的可用学习站点。

重点修复下列历史问题：

1. 动画播放后只有时钟或按钮文字变化，物体、场、轨迹、图像或物理读数没有变化。
2. 参数改变后公式、仪器读数和画面不一致，或物理规律错误。
3. 每个按钮、模式、选择项没有真实效果或重置错误。
4. 手机端目录无法跳转、画布/控件溢出、触控不能操作。
5. 综合专题出现无意义动画、错误状态或不能完成完整作答流程。

## 单页通过条件

一页只有同时满足以下条件，才能在逐页账本记为 `PASS`：

| 检查项 | 验收证据 |
| --- | --- |
| 物理过程 | 播放后有真实物体/场/轨迹/图像/物理读数变化，不能只变化计时或按钮文字。 |
| 所有按钮 | 每个可见按钮或模式都执行一次，并改变预期状态、画面或结果。 |
| 操作联动 | 至少一个参数或直接操作能使画面、读数和公式同步变化。 |
| 物理正确性 | 量纲、方向、临界条件、守恒关系和过程逻辑无明显错误。 |
| 重置与复入 | 重置恢复初始状态；离开后重新进入仍可工作。 |
| 手机端 | 390px 宽度下无横向溢出，画布、滑块和按钮可见且可操作。 |
| 教学闭环 | 有观察任务；公式可对应画面量；动画不是装饰。 |

任何一项失败均为 `FIX_REQUIRED`，必须修复、复测后才可通过。

## 执行顺序

1. 目录与移动端跳转。
2. 必修三电路、电场、电磁波；选择性必修二磁场与电磁感应。
3. 必修一运动、受力、连接体、弹簧瞬时问题。
4. 必修二曲线运动、万有引力与能量。
5. 动量、振动、波动、光学、热学、原子核。
6. 实验、模型、综合专题、图谱视图。

发现高风险页面（错误物理、无效按钮、手机端不可用）时，可以插队修复，但必须在逐页账本中说明原因。

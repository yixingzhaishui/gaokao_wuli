# 四川高考物理

> 让高三学生**看见现象、感受到变化、理解规律由来、掌握适用条件、形成解题思路**。

本站依据《四川高考物理网站开发规格 V1.2.2》开发，覆盖四川新高考物理全部必修与选择性必修内容。

## 快速开始

- 先看 [高考依据与使用说明](syllabus) 了解本站范围和学习方法；
- 按官方模块顺序学习：[必修1](bx1) → [必修2](bx2) → [必修3](bx3) → [选择性必修1](xb1) → [选择性必修2](xb2) → [选择性必修3](xb3)；
- 复习强化：[实验专题](experiments) · [模型专题](models) · [易错点](errors) · [真题训练](papers) · [知识图谱](graph)。

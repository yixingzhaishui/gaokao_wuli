# 四川高考物理

> 让高三学生**看见现象、感受到变化、理解规律由来、掌握适用条件、形成解题思路**。

> **非四川省教育考试院官方项目。当前为内容审校测试版；交互通过不代表物理内容、答案与来源均已审定。**

本站依据《四川高考物理网站开发规格 V1.2.2》建设。当前自定义目录已覆盖必修与选择性必修模块，课标条目映射、内容和题目来源正在逐项复核，暂不宣称“全部覆盖”或“全部核验”。

## 快速开始

- 先看 [高考依据与使用说明](syllabus) 了解本站范围和学习方法；
- 公式、下标与单位统一遵循 [数学符号与单位表](MATH_SYMBOLS)；内容结构按 [分类编辑模板](CONTENT_TEMPLATES) 选择；
- 按官方模块顺序学习：[必修1](bx1) → [必修2](bx2) → [必修3](bx3) → [选择性必修1](xb1) → [选择性必修2](xb2) → [选择性必修3](xb3)；
- 复习强化：[实验专题](experiments) · [模型专题](models) · [易错点](errors) · [来源复核中的题库](papers) · [知识图谱](graph)。

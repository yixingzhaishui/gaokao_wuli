#### 情境与现象
把小球水平抛出，小球刚离手时速度水平向前，但重力竖直向下，轨迹会向下弯。汽车转弯时，速度沿车头方向，但合力要指向转弯内侧，汽车才会改变方向。

曲线运动的关键不是“有没有力”，而是看**合力方向与速度方向是否共线**：

```text
合力与速度共线：只改变速度大小，轨迹仍为直线。
合力与速度不共线：速度方向改变，轨迹成为曲线。
```

图中蓝色箭头是瞬时速度 `v`，沿轨迹切线方向；红色箭头是合力 $F_{\text{合}}$，指向轨迹弯曲的内侧；紫色箭头是合力垂直速度方向的分量 `F⊥`。

**动态表征。**
实验台把速度切线、合力箭头、`F∥` 和 `F⊥` 放在轨迹同一点上。抓住红色合力箭头端点拖动时，轨迹会立刻从直线变成曲线，学生能看到：`F∥` 改变速度大小，`F⊥` 改变速度方向。只要 $F⊥\ne 0$，轨迹就会弯曲。

可直接拖动：红色合力箭头端点、底部 `F`、`v`、$\alpha$ 三个手柄。实时变化：轨迹形状、`F∥`、`F⊥`、夹角 $\alpha$ 和“曲线/直线”判断。

**证据任务。**
- 把红色合力箭头拖到与蓝色速度箭头同向：看 $F⊥=0$，轨迹仍为直线，只会加速。
- 把红色合力箭头拖到与蓝色速度箭头反向：看 $F⊥=0$，轨迹仍为直线，只会减速。
- 把红色合力箭头拖到垂直速度方向：看 $F∥=0$，合力只改变速度方向。
- 拖大红色合力箭头：看 `F⊥` 变大时，轨迹弯曲是否更明显。

#### 规律、证据与核心概念
速度是矢量，既有大小也有方向。由牛顿第二定律：

```text
F合 = ma
```

加速度方向与合力方向相同。把合力按速度方向分解：

```text
F∥ = Fcosα
F⊥ = Fsinα
```

`F∥` 沿速度方向，只改变速度大小；`F⊥` 垂直速度方向，会改变速度方向。曲线运动的条件就是：

```text
F合 与 v 不共线，或 F⊥ ≠ 0
```

**概念辨析。**
曲线运动中，速度方向一定沿轨迹切线方向；合力方向一般不沿轨迹切线，而指向轨迹弯曲的内侧。若合力恰好沿切线方向或反切线方向，物体只会沿直线加速或减速，不会“拐弯”。

判断轨迹时先画速度切线，再画合力方向。看合力有没有垂直速度的分量，比死记“凹侧”更可靠。

#### 公式、变量、单位与条件
| 公式 | 图中对应 | 读数含义 |
|---|---|---|
| $F_{\text{合}}=ma$ | 红色合力箭头 | 加速度与合力同向 |
| $F∥=Fcos\alpha$ | 绿色沿速度方向分量 | 改变速度大小 |
| $F⊥=Fsin\alpha$ | 紫色垂直速度方向分量 | 改变速度方向 |
| $F⊥\ne 0$ | 动画中的“曲线运动”状态 | 曲线运动条件 |

**适用边界。**
- 适用于质点模型，研究某一瞬时速度和合力方向。
- 若合力为零，物体保持匀速直线运动。
- 若合力与速度共线，物体做直线变速运动。
- 若合力与速度不共线，物体做曲线运动。
- 曲线运动不一定变速率；只有 $F∥=0$ 时速率才不变。

#### 应用与迁移
- 选择题：根据速度和合力方向判断轨迹弯向哪边。
- 平抛题：初速度水平、重力竖直向下，所以轨迹向下弯。
- 圆周题：速度沿切线，合力或向心力指向圆心。
- 轨迹判断题：根据受力方向排除不可能轨迹。

**解题路径。**
1. 在轨迹某点画速度方向：沿切线。
2. 画合力方向：由受力分析或题给条件确定。
3. 比较 $F_{\text{合}}$ 与 `v` 是否共线。
4. 若不共线，轨迹向合力垂直速度分量一侧弯曲。
5. 若题目问速度大小变化，再看 `F∥` 与速度同向还是反向。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “曲线运动中合力沿速度方向” | 沿速度方向只改变速率，不改变方向 | 合力应有垂直速度分量 |
| “曲线运动一定变速率” | 速度方向变了就已经是变速；速率可不变 | 区分速度和速率 |
| “合力指向运动方向” | 合力指向加速度方向，不等于速度方向 | 画切线速度和凹侧合力 |
| “曲线运动合力可以为零” | 合力为零时速度不变，轨迹直线 | 曲线运动必有非零合力且不共线 |

这就是运动合成与分解：

**动态表征。**
实验台用小船过河展示“两个分运动同时发生”。抓住绿色船相对水速度箭头改变船头方向时，紫色水流箭头仍同时作用，红色合速度和黑色实际航迹会立即改变。垂直河岸分量决定船多久到达对岸；沿河方向分量决定船被冲到下游多少。

可直接拖动：绿色船相对水速度箭头端点、紫色水流速度箭头端点、底部 $v_{\text{船}}$、$v_{\text{水}}$、$\theta$、`d` 四个手柄。实时变化：分速度、合速度、实际轨迹、过河时间和下游偏移。

**证据任务。**
- 让船头垂直河岸 $\theta =90^{\circ}$，拖大紫色水流箭头：看过河时间是否不变，下游偏移是否变大。
- 拖大绿色船速箭头：看垂直河岸分量变大时，过河时间是否变短。
- 把船头向上游偏：看绿色箭头的上游分量能否抵消水流，使红色合速度垂直河岸。
- 拖动河宽 `d`：看过河时间和下游偏移是否按比例变化。

#### 规律、证据与核心概念
位移、速度、加速度都是矢量，遵守平行四边形定则。小船过河中：

```text
v船/岸 = v船/水 + v水/岸
```

若取 `x` 轴沿水流方向，`y` 轴垂直河岸方向，则：

```text
vx = v水 + v船 cosθ
vy = v船 sinθ
```

两个分运动同时发生、互不影响：

```text
过河时间 t = d / vy
下游偏移 x = vx t
```

**概念辨析。**
合运动是实际看到的运动，分运动是为了分析方便分解出来的运动。分运动不是“先发生一个，再发生另一个”，而是同时发生。

小船过河最常见的误区是把“船头指向”当成“实际运动方向”。船头方向只决定船相对水的速度；岸上看到的实际运动方向还要叠加水流速度。

#### 公式、变量、单位与条件
| 公式 | 图中对应 | 读数含义 |
|---|---|---|
| $v_{\text{合}}=v_{\text{船/水}}+v_{\text{水/岸}}$ | 红色合速度箭头 | 岸上看到的实际速度 |
| $vx=v_{\text{水}}+v_{\text{船}}\cos \theta$ | 沿河分量读数 | 决定下游偏移 |
| $vy=v_{\text{船}}\sin \theta$ | 垂直河岸分量读数 | 决定过河时间 |
| $t=d/vy$ | 河宽与垂直分量 | 船到达对岸所需时间 |
| $x=vx t$ | 轨迹终点横向偏移 | 被水冲到下游的距离 |

**适用边界。**
- 合成与分解适用于矢量：位移、速度、加速度。
- 分运动必须相对同一参考系表达，不能把不同参考系的量乱加。
- 分运动具有等时性：同一段合运动和各分运动用时相同。
- 小船模型中常默认水流速度恒定、船相对水速度恒定。
- 若船速小于水速，不一定能正对到达对岸，但仍能过河。

#### 应用与迁移
- 小船过河：求最短时间、最小偏移、实际速度方向。
- 平抛运动：水平匀速与竖直自由落体合成。
- 斜抛运动：水平和竖直两个方向分解。
- 相对运动：判断不同参考系下速度关系。

**解题路径。**
1. 先选参考系，常选地面或河岸。
2. 画出已知速度矢量，标清“谁相对谁”。
3. 沿方便方向分解，通常取两个互相垂直方向。
4. 用分运动独立性分别列式。
5. 用等时性把两个方向联系起来。
6. 最后合成得到实际位移或速度。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “船头方向就是实际运动方向” | 水流速度还要叠加 | 画出合速度 |
| “先过河再被水冲” | 两个分运动同时发生 | 用等时性分析 |
| “水速影响过河时间” | 若船头垂直河岸，水速只影响偏移 | 时间看垂直河岸分量 |
| “不同参考系速度可直接乱加” | 必须写清相对关系 | 用 $v_{\text{船/岸}}=v_{\text{船/水}}+v_{\text{水/岸}}$ |
| “合速度大小直接相加” | 速度是矢量 | 按方向合成或分解 |

**证据任务。**
- 拖大蓝色 $v_{0}$ 箭头：看水平射程怎么变，但落地时间是否不变。
- 拖高桌面高度 `h`：看落地时间怎么变，水平射程是否跟着变。
- 拖动观察时刻 `t`：看 `vx` 是否始终不变，`vy` 是否按 `gt` 增大。
- 切到电场类平抛：改变 `q` 的正负，看偏转方向为什么改变。

#### 规律、证据与核心概念
理想平抛可分解为：水平方向不受力 → 匀速 $x=v_{0}t$；竖直方向只受重力 → 自由落体 $y=\tfrac12gt^{2}$。在同一 `h`、同一 `g` 且忽略空气阻力时，落地时间由竖直方向决定 $t=\sqrt{2h/g}$，与 $v_{0}$ 无关。水平射程 $x=v_{0}\sqrt{2h/g}$。

**概念辨析。**
平抛的"平"指初速度水平。两个分运动独立互不影响。落地速度是水平速度和竖直速度的矢量和，不是简单相加。

#### 公式、变量、单位与条件
- 水平：x = v0·t → 图中水平射程
- 竖直：y = ½g·t² → 图中下落高度 h
- 落地时间：t = √(2h/g) → 只由 h 决定
- 落地速度：v = √(v0² + 2gh) → 水平和竖直合成

**适用边界。**
- 只受重力（忽略空气阻力）。
- 初速度水平。
- 抛出点为参考点。

#### 应用与迁移
- 选择题：比较落地时间、射程、速度方向。
- 实验题：探究平抛运动特点。
- 计算题：平抛 + 斜面、平抛 + 障碍物。
- 综合题：类平抛（电场偏转）。

**解题路径。**
1. 分解为水平和竖直两个分运动；
2. 竖直方向先求时间 t=√(2h/g)；
3. 水平方向求射程 x=v0·t；
4. 求速度时分别求 vx=v0、vy=g·t，再合成；
5. 检查速度方向用 tanθ=vy/vx。

#### 边界检查与易错点
- 认为初速度越大落地越快（落地时间与 v0 无关）。
- 把落地速度当水平速度（要合成两个分速度）。
- 忽略竖直方向初速度为 0。

#### 规律、证据与核心概念
斜抛运动只受重力，重力竖直向下，所以：

```text
水平方向 ax=0
竖直方向 ay=-g
```

把初速度分解：

```text
v0x = v0cosθ
v0y = v0sinθ
```

于是两个方向分别列运动学公式：

```text
x = v0x t
y = h + v0y t - 1/2 gt²
vy = v0y - gt
```

最高点满足 $vy=0$，所以 $t_{\text{顶}}=v_{0}y/g$。

**概念辨析。**
斜抛不是一个新运动公式包，而是运动分解的直接应用。水平方向和竖直方向互不干扰，但共用同一个时间 `t`。

最高点最容易误解：竖直速度为零，不代表合速度为零。只要 $v_{0}x$ 不为零，物体在最高点仍然向前运动。

- 忽略空气阻力，只受重力。
- 重力加速度 `g` 近似恒定。

- 同一段运动中，两个分运动具有等时性。
- 类平抛本质相同：一个方向匀速，另一个方向匀变速。

#### 应用与迁移
- 选择题：判断最高点速度、加速度、速度方向。
- 计算题：求飞行时间、射程、最大高度。
- 类平抛题：带电粒子在匀强电场中的偏转。
- 综合题：斜面、平台、落点约束下的分运动。

**解题路径。**
1. 画初速度并分解为 $v_{0}x、v_{0}y$。
2. 水平方向列 $x=v_{0}x t$。
3. 竖直方向列 $y=h+v_{0}y t-1/2gt^{2}$。
4. 用同一个时间 `t` 联系两个方向。
5. 最高点用 $vy=0$，落地用 $y=0$。
6. 检查 sin/cos：靠近水平的分量用 $\cos \theta$。

| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| $v_{0}x=v_{0}sin\theta$ | θ 通常是与水平夹角 | 水平邻边用 cos |
| “最高点速度为 0” | 最高点只有竖直速度为 0 | 速度仍有 `vx` |
| “水平和竖直时间不同” | 分运动同时发生 | 两方向共用同一个 `t` |
| “斜抛加速度沿轨迹切线” | 只受重力时加速度竖直向下 | 加速度恒为 `g` |
| “45° 总是射程最大” | 仅同高落地且无阻力时成立 | 有初始高度时要重新计算 |

**证据任务。**
- 保持 `T` 不变，把蓝点向外拖：看 $\omega$ 不变，但 $v=\omega r$ 随半径变大而变大。
- 保持 `r` 不变，减小周期 `T`：看 `f`、$\omega$、`v` 同时变大。
- 沿圆周拖蓝点或拖 $\theta$：看黄色弧长 `s` 如何随角度增大而增大。
- 点击播放：看速度箭头始终沿切线，方向不断改变。

#### 规律、证据与核心概念
圆周运动的基本量不是凭空规定的，而是从“重复转圈”和“沿圆弧走路程”来的。

```text
周期 T：转一圈所用时间
频率 f：每秒转的圈数，所以 f = 1/T
角速度 ω：单位时间转过的角度
```

匀速圆周运动一圈转过 $2\pi rad$，用时 `T`，所以：

```text
ω = 2π/T = 2πf
```

圆弧长度满足 $s=r\theta$。两边同时除以时间 `t`：

```text
s/t = r·θ/t
v = rω
```

这就是 $v=\omega r$ 的来源：同样的角速度 $\omega$，半径 `r` 越大，单位时间扫过的弧长越长。

**概念辨析。**
角速度 $\omega$ 描述“转盘转得多快”，线速度 `v` 描述“圆周上的点跑得多快”。同一个刚性转盘上，各点同一时间转过的角度相同，所以 $\omega$ 相同；但不同点半径不同，弧长不同，所以 `v` 不同。

匀速圆周运动中的“匀速”，指速率大小不变，不是速度矢量不变。因为速度方向不断变，速度这个矢量一直在变。

#### 公式、变量、单位与条件
| 公式 | 图中对应 | 动画读数怎么验证 |
|---|---|---|
| $f=1/T$ | 右侧周期 `T` 输入 | 把 `T` 调小，`f` 变大 |
| $\omega =2\pi /T=2\pi f$ | 蓝点单位时间扫过的角度 | 把 `T` 调小，转动更快，$\omega$ 变大 |
| $s=r\theta$ | 黄色圆弧、灰色半径、紫色角 | 拖动 $\theta$ 或 `r`，弧长读数同步变化 |
| $v=s/t=\omega r$ | 绿色切线速度箭头 | 固定 `T` 拖大 `r`，`v` 变大 |

注意：$s=r\theta$ 中的 $\theta$ 必须用弧度。若题目给角度，要先换成弧度，例如 $120^{\circ}=2\pi /3\ \mathrm{rad}$。

**适用边界。**
- 本节公式适用于圆周运动基本量，尤其是匀速圆周运动的量值关系。
- $v=\omega r$ 要求 `v` 是圆周上该点的线速度，`r` 是该点到圆心的半径。
- $\omega =2\pi /T$、$f=1/T$ 适用于周期性圆周运动。
- $s=r\theta$ 中 $\theta$ 必须用弧度，不能直接把角度数代入。
- 速度方向沿切线；半径方向不是线速度方向。

#### 应用与迁移
- 选择题：判断同轴转盘、皮带传动、齿轮传动中 $\omega$、`v` 谁相等。
- 计算题：由 $T、f、\omega 、r$ 中的已知量求其他基本量。
- 图像题：根据转过角、时间、弧长读出角速度或线速度。
- 综合题：为向心加速度 $a=v^{2}/r=\omega ^{2}r$ 做准备。

**解题路径。**
1. 先判断题目问角量还是线量：$T、f、\omega 、\theta$ 属于角量，`s、v、r` 属于线量。
2. 若给周期或频率，先写 $f=1/T$、$\omega =2\pi /T=2\pi f$。
3. 若涉及圆弧路程，用 $s=r\theta$，并先把角度化成弧度。
4. 若涉及圆周上某点快慢，用 $v=\omega r$。
5. 最后检查比较关系：同轴转动通常 $\omega$ 相同；同一皮带接触处通常 `v` 相同。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “同一转盘上各点线速度相同” | 同一转盘是 $\omega$ 相同，不同半径 $v=\omega r$ 不同 | 先判同轴，再用 $v=\omega r$ |

**证据任务。**
- 抓蓝色速度箭头端点把 `v` 调大：看 $F_{\text{向}}$ 是否按 $v^{2}$ 增大。
- 把小球向外拖使 `r` 变大：看同一 `v` 下 $F_{\text{向}}$ 是增大还是减小。
- 拖动 `m`：看 $F_{\text{向}}$ 是否与质量成正比，而 $a_{\text{向}}$ 是否不受质量影响。
- 播放动画：看速度方向一直沿切线，向心力和向心加速度一直指向圆心。

#### 规律、证据与核心概念
从运动学推导：圆周运动速度方向改变，Δv 指向圆心，a=Δv/Δt=v²/r。从动力学：牛顿第二定律 F=ma → F向=mv²/r。向心力不是新性质的力，是合力指向圆心的效果。

**概念辨析。**
向心力不是一种新的力，而是某个力或几个力的**径向合力效果**。绳拉力、重力、万有引力、洛伦兹力都可以提供向心力。只有当合力纯粹指向圆心、切向合力为零时，它才只改变速度方向而不改变速率；一般曲线运动若还存在切向合力，速度大小也会改变。

#### 公式、变量、单位与条件
- 向心加速度：$a_{\text{向}}=v^{2}/r=\omega ^{2}r$ → 图中 $a_{\text{向}}$ 读数和红色指向圆心箭头
- 向心力：$F_{\text{向}}=mv^{2}/r=m\omega ^{2}r$ → 图中红色 $F_{\text{向}}$ 箭头；它表示“指向圆心的合力效果”，不是一种新性质力
- 线速度与角速度：$v=\omega r$ → 图中蓝色切线速度箭头和 $\omega$ 读数
- 周期：$T=2\pi r/v=2\pi /\omega$ → 图中周期读数

**适用边界。**
- 匀速圆周运动（速率不变，方向变）。
- 向心力是效果力，由具体力提供。
- 匀速圆周运动中，若合力只有径向分量，则合力始终与速度垂直、不做功；变速圆周运动还可能有切向合力并做功。

#### 应用与迁移
- 选择题：比较向心加速度、向心力大小。
- 计算题：圆锥摆、竖直圆周、汽车转弯。
- 综合题：天体圆周、磁场圆周。

**解题路径。**
1. 确定圆心和半径；
2. 受力分析，找指向圆心的合力；
3. 列 F向 = mv²/r = mω²r；
4. 求解未知量；
5. 检查临界条件（如绳恰好拉直、恰好脱离）。

#### 边界检查与易错点
- 认为向心力是一种新的力（向心力是效果，不是性质力）。
- 不加条件地说“圆周运动合力不做功”（只有纯径向合力与瞬时速度垂直时才不做功；切向分力可以改变速率并做功）。
- 混淆 a=v²/r 和 a=ω²r（两者等价，但适用场景不同）。

- 把侧视图小球向外拖：看 $r = Lsin\theta$ 变大后，所需速度和拉力怎样变大。
- 拖动 `m`：看 `T` 和 $F_{\text{向}}$ 随质量变大，但同一 $L、\theta$ 下的 `v` 与周期不变。
- 拖动 `L`：看半径变大、周期变长，比较长绳和短绳哪一个转得更慢。

**观察记录表**

| 操作 | 画面中最该看的量 | 应得结论 | 高考问法 |
|---|---|---|---|
| 只增大 `m` | `T`、$F_{\text{向}}$ 变大，`v`、周期几乎不变 | 质量影响受力，不决定同一几何形状下的快慢 | “换成质量更大的小球，周期如何变” |
| 只增大 $\theta$ | 圆半径 `r` 变大，拉力箭头变长 | 摆角越大，需要越大的水平分力和速度 | “夹角增大，绳拉力如何变” |
| 只增大 `L` | 轨道半径变大，周期变长 | 长绳转得慢，周期与 $\sqrt{Lcos\theta }$ 成正比 | “绳长变为 4 倍，周期如何变” |

**一句话结论**：圆锥摆的“快慢”由 $L、\theta 、g$ 决定；“绳有多吃力”才跟质量有关。

#### 规律、证据与核心概念
圆锥摆在竖直方向没有加速度，因此竖直方向合力为零：

```text
Tcosθ = mg
```

小球在水平方向做匀速圆周运动，水平合力提供向心力：

```text
Tsinθ = mv²/r = mω²r
```

又因为几何关系 $r = Lsin\theta$，联立可得：

```text
v = √(rg tanθ)
ω = √(g/(Lcosθ))
周期 T周期 = 2π√(Lcosθ/g)
```

这里有一个很重要的化简：把水平方程除以竖直方程，`T` 和 `m` 会同时约掉：

```text
Tsinθ / Tcosθ = (mv²/r) / mg
tanθ = v²/(rg)
```

所以同一绳长、同一摆角下，换一个更重的小球，运动速度和周期不变。这正是动画里“拖动质量，看速度读数不变”的原因。

**概念辨析。**
圆锥摆的关键不是“多了一个向心力”，而是把绳拉力按竖直和水平方向分解：竖直分力抵消重力，水平分力改变速度方向。摆角越大，绳越“横”，水平分力比例越大，转得必须更快才能维持这个圆周。

从学生视角可以这样记：圆锥摆不是背公式，而是看绳拉力这一根箭头同时做两件事。

```text
竖直方向：别掉下去 → Tcosθ = mg
水平方向：别飞出去 → Tsinθ = mv²/r
```

两个方向缺一不可。只写向心力方程、不写竖直平衡，是圆锥摆题最常见的漏步。

#### 公式、变量、单位与条件
- $r = Lsin\theta$：图中从竖直轴到小球圆轨道的水平半径。
- $Tcos\theta = mg$：图中拉力的竖直分量与重力箭头等大反向。
- $Tsin\theta = mv^{2}/r$：图中拉力的水平分量对应紫色向心箭头。
- $\omega = √(g/(Lcos\theta ))$：动画读数中的角速度由绳长和摆角决定。

**从画面读方程**

1. 先看角 $\theta$ 标在“绳与竖直线”之间，所以靠近竖直方向的是 $Tcos\theta$。
2. 再看紫色向心箭头水平指向圆心，所以它对应 $Tsin\theta$，不是额外受力。
3. 最后看圆轨道半径，图上水平长度是 `r`，不是绳长 `L`。

**适用边界。**
- 小球做稳定的水平匀速圆周运动，摆角 $\theta$ 不变。
- 绳轻且不可伸长，空气阻力忽略。
- 只适用于圆锥摆模型；若速度变化或摆角变化，就不能直接套稳定公式。

#### 应用与迁移
- 选择题：判断拉力、周期、速度随 $m、L、\theta$ 的变化。
- 计算题：已知绳长和摆角求速度、角速度、周期或拉力。
- 综合题：圆锥摆与力的分解、临界张力、圆周运动合并考查。

**解题路径。**
1. 选小球为研究对象，画出 `mg` 和 `T`；
2. 建立竖直方向和水平方向：竖直平衡，水平圆周；
3. 写 $Tcos\theta =mg$、$Tsin\theta =mv^{2}/r$；
4. 用 $r=Lsin\theta$ 把几何量代入；
5. 检查结果是否符合直觉：质量不影响同一 $L、\theta$ 下的周期和速度，但会影响拉力。

#### 边界检查与易错点
- 把向心力当作额外受力，受力图中多画一个“向心力”。正确做法：受力图只画真实力，列方程时把水平合力叫向心力。
- 把 $Tsin\theta$ 和 $Tcos\theta$ 写反。角 $\theta$ 是绳与竖直方向的夹角，所以竖直分量是 $Tcos\theta$，水平分量是 $Tsin\theta$。
- 误以为质量越大转得越快。稳定圆锥摆中 `v` 和周期由 $L、\theta 、g$ 决定，质量只改变拉力和向心力大小。

**错误诊断卡**

| 学生写法 | 错在哪里 | 该怎么改 |
|---|---|---|
| $T = mv^{2}/r$ | 把整根绳拉力都当成向心力 | 改为 $Tsin\theta = mv^{2}/r$ |
| $Tsin\theta = mg$ | 把分量和角度写反 | 因为 $\theta$ 对着竖直方向，竖直分量是 $Tcos\theta$ |
| $r = L$ | 把绳长当半径 | 圆轨道半径是水平投影 $r=Lsin\theta$ |
| 质量加倍，周期加倍 | 没有看到 `m` 会约掉 | 同一 $L、\theta$ 下周期与质量无关 |

图中最高点的向心方向向下。绳只能向下拉小球，因此要求 $mg + T = mv^{2}/r$ 且 $T \ge 0$；杆可以向下拉也可以向上托，约束力可正可负。

**动态表征。**
实验台把完整竖直圆周和最高点受力放在同一画面。抓住最高点蓝色 $v_top$ 箭头端点时，约束力会即时变号；抓橙色半径手柄时，临界速度 $\sqrt{gr}$ 会同步改变。若绳模型速度低于临界，动画会提示绳松弛，说明这不是完整竖直圆周。

**交互探究。**

**证据任务。**
- 在绳模型下抓住蓝色 $v_top$ 箭头慢慢缩短，看什么时候出现“绳松弛”。
- 拖动橙色半径手柄增大 `r`，观察临界速度 $\sqrt{gr}$ 变大，理解大半径圆周更难过顶。
- 拖动 `g`：看 $v_min=\sqrt{gr}$ 和约束力读数如何同步改变。
- 拖动 $position_angle$：看最高点、最低点的径向方程为什么不同。
- 切到杆模型，令 $v_top < \sqrt{gr}$，看约束力方向如何变为“向上托”。

**观察记录表**

| 操作 | 状态提示 | 物理含义 | 题目语言 |
|---|---|---|---|
| 绳模型，$v_top > \sqrt{gr}$ | 能过顶，$T>0$ | 重力和绳拉力共同提供向心力 | “绳仍绷紧” |
| 绳模型，$v_top = \sqrt{gr}$ | 恰好临界，$T=0$ | 最高点只有重力提供向心力 | “恰好通过最高点” |
| 绳模型，$v_top < \sqrt{gr}$ | 绳松弛 | 绳不能提供负拉力，小球脱离圆轨道 | “不能完成竖直圆周” |
| 杆模型，$v_top < \sqrt{gr}$ | 杆向上托 | 杆可提供向上的约束力，所以仍能过顶 | “轻杆连接” |

#### 规律、证据与核心概念
最高点圆心在小球下方，所以向心方向向下。对绳模型，最高点受力为重力 `mg` 和绳拉力 `T`，二者都向下：

```text
mg + T = mv²/r
```

临界时绳刚好不松，$T = 0$：

```text
mg = mv²/r
v_min = √(gr)
```

对杆或圆管模型，最高点约束力可以向下也可以向上，统一写作 $mg + N = mv^{2}/r$。当 $v < \sqrt{gr}$ 时，$N < 0$，表示杆对小球的力实际向上。

如果题目给的是最低点速度，不要直接把最低点速度代入最高点方程。先用机械能守恒换到最高点：

```text
1/2 mv_bottom² = 1/2 mv_top² + mg(2r)
v_bottom² = v_top² + 4gr
```

绳模型“从最低点恰好做完整竖直圆周”时，最高点 $v_top^{2} = gr$，所以：

```text
v_bottom,min² = gr + 4gr = 5gr
v_bottom,min = √(5gr)
```

**概念辨析。**
竖直圆周临界不是背一个固定公式，而是判断“约束能不能提供所需方向的力”。绳只能拉不能推，所以最高点速度不足时它帮不上忙；杆可以推也可以拉，所以低速过顶并不违反受力条件。

这类题要先问一个问题：约束物能不能“推”？绳不能推，所以算出负拉力就说明模型失效；杆和管道能推能拉，所以约束力方向可以反过来。

#### 公式、变量、单位与条件
- $v_min = \sqrt{gr}$：动画中的临界速度读数。
- $mg + T = mv^{2}/r$：最高点向下的红色重力箭头和紫色绳拉力箭头。
- $T = m(v^{2}/r - g)$：读数中的最高点约束力；若为负，绳模型不成立。
- $v_bottom^{2} = v_top^{2} + 4gr$：从最高点到底端重力势能减少，动画给出底端速度。

**模型对照**

| 模型 | 最高点方程 | 约束力方向限制 | 临界条件 |
|---|---|---|---|
| 绳系小球 | $mg+T=mv^{2}/r$ | $T\ge 0$，只能向下拉 | $T=0 \to v_min=\sqrt{gr}$ |
| 外侧轨道/内壁压力向下 | $mg+N=mv^{2}/r$ | $N\ge 0$，不能吸住小球 | $N=0 \to v_min=\sqrt{gr}$ |
| 轻杆/圆管 | $mg+N=mv^{2}/r$ | `N` 可正可负 | 最高点可 $v=0$，但受力方向会变 |

**适用边界。**
- 小球在竖直平面内做圆周运动，半径固定。
- 绳模型：绳轻且不可伸长，只能提供拉力，不能提供压力。
- 杆/管模型：约束可以双向作用，能拉也能推。
- 若存在明显能量损失，底端速度关系需加入非保守力做功。

#### 应用与迁移
- 选择题：判断能否过最高点、比较绳拉力或轨道压力。
- 计算题：已知半径求过顶最小速度或底端最小速度。
- 综合题：机械能守恒 + 圆周临界 + 受力分析。

**解题路径。**
1. 找最高点，明确圆心方向；
2. 画真实受力，区别绳、杆、内外轨道；
3. 沿向心方向列 $ΣF = mv^{2}/r$；
4. 代入临界条件：绳为 $T=0$，杆低速时允许 $N<0$；
5. 若题目给底端或其他位置速度，用机械能守恒把速度换到最高点。

#### 边界检查与易错点
- 把所有竖直圆周都套 $v_min=\sqrt{gr}$。这个公式只对应绳或外轨道刚好不脱离的最高点临界。
- 最高点方向判断错，把向心方向写成向上。最高点圆心在下方，向心方向一定向下。
- 忽略约束力不能为负。绳的 $T = m(v^{2}/r - g)$ 若算出负值，物理意义是绳松弛，不是“负拉力”。
- 把最低点速度直接代入最高点方程。最低点和最高点高度差 `2r`，速度必须先用能量关系转换。

#### 规律、证据与核心概念
开普勒定律给出了行星运动的经验规律，牛顿把“地面附近物体下落”和“天体绕行”统一起来：让月球绕地球转所需的向心力，正好符合随距离平方反比减小的吸引力。于是得到万有引力定律：




> **公式首次使用卡**：适用边界——两物体可看作质点，或为质量分布均匀的球体且 r 取球心距离；r 是两质心距离，不是表面间距；公式给出力的大小；方向沿两质心连线相互吸引。符号含义在紧邻公式的正文、图注或表格中定义；除题目另有单位外，代入前统一为 SI，结果保留单位并做量纲检查。

```text
F = G M m / r²
G = 6.67×10^-11 N·m²/kg²
```

这是一条普遍规律：任意两个有质量的物体之间都存在引力。

**概念辨析。**
万有引力不是只发生在天体之间，而是所有有质量物体之间都有。平时两个书包之间也有引力，只是质量太小，力小到感觉不到；天体质量巨大，所以引力成为主导运动的力。

#### 公式、变量、单位与条件
- `M、m`：图中两个物体的质量，拖动后圆大小和读数改变。
- `r`：图中两质心之间的虚线距离。
- `F`：图中两条红色引力箭头，大小相等、方向相反。
- $1/r^{2}$：右侧曲线；距离越大，曲线下降越快。

**适用边界。**
- 两物体可看作质点，或为质量分布均匀的球体且 `r` 取球心距离。
- `r` 是两质心距离，不是表面间距。
- 公式给出力的大小；方向沿两质心连线相互吸引。

#### 应用与迁移
- 选择题：比较质量或距离改变后引力大小。
- 计算题：地球表面重力、天体之间引力。
- 综合题：万有引力提供向心力，推导卫星速度、周期和轨道半径关系。

**解题路径。**
1. 判断研究对象能否看作质点或均匀球体；
2. 确定 `M、m、r`，特别检查 `r` 是否为质心距离；
3. 只比较大小时优先用比例法：$F \propto Mm/r^{2}$；
4. 天体圆周运动中常列 $GMm/r^{2} = mv^{2}/r$，先约掉卫星质量 `m`。

#### 边界检查与易错点
- 把 `r` 当作两个物体表面之间的距离。球体问题通常要用球心距。
- 认为质量大的物体受到的引力更大。相互作用力大小相等，只是质量不同导致加速度不同。
- 做比例题时忘记平方反比：距离变为 `2r`，引力变为 `1/4`，不是 `1/2`。

**证据任务。**
- 拖动太阳增大 `e`：看太阳始终在焦点，不在椭圆中心；近日点更近，远日点更远。
- 点击播放：看近日点附近运动快，远日点附近运动慢。
- 拖动椭圆右端增大 `a`：看周期 `T` 如何变长，并观察 $T^{2}/a^{3}$ 是否保持同一常量。
- 增大中心天体质量 `M`：看同一 `a` 下周期是否变短。

#### 规律、证据与核心概念
开普勒定律最初来自第谷的大量天文观测数据，后来牛顿用万有引力定律解释了它们。高中阶段要会用三条结论：

```text
第一定律：行星绕太阳的轨道是椭圆，太阳在椭圆的一个焦点上。
第二定律：太阳和行星的连线在相等时间内扫过相等面积。
第三定律：同一中心天体下，T²/a³ = 常量。
```

若中心天体质量为 `M`，由万有引力理论可写成：

```text
T²/a³ = 4π²/(GM)
```

同一中心天体时 `M` 不变，所以 $T^{2}/a^{3}$ 相同；不同中心天体时常量不同。

**概念辨析。**
第二定律不是说行星角速度恒定，而是说“面积速度”恒定。近日点距离小，为了同样时间扫过同样面积，行星必须跑得更快；远日点距离大，行星可以跑得更慢。

第三定律中的 `a` 是椭圆半长轴。圆轨道只是特殊情况，此时 $a=r$。

| 公式/定律 | 图中对应 | 动画读数 |

**适用边界。**
- 开普勒三定律适用于在引力主导下绕同一中心天体运动的行星、卫星等。
- 第三定律比较时必须确认中心天体相同；中心天体不同，常量不同。
- 椭圆轨道用半长轴 `a`，不能把近日点或远日点距离直接当作 `a`。
- 面积定律描述面积速度，不是线速度恒定，也不是角速度恒定。

#### 应用与迁移
- 选择题：判断太阳位置、近日点远日点速度、面积定律。
- 计算题：用 $T^{2}/a^{3}$ 比较两个天体周期或轨道大小。
- 综合题：椭圆轨道和圆轨道衔接到卫星变轨、近地点远地点速度。

**解题路径。**
1. 先判断是否绕同一中心天体。
2. 轨道若是椭圆，先找半长轴 `a`。
3. 比周期用 $T^{2}/a^{3}=\text{常量}$，即 $T\propto a^(3/2)$。
4. 比速度用面积定律：近日点快，远日点慢。
5. 若题目给近日距和远日距，先算 $a=(r_{\text{近}}+r_{\text{远}})/2$。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “太阳在椭圆中心” | 开普勒第一定律说太阳在焦点 | 画出焦点，不画在中心 |
| “行星匀速绕太阳转” | 椭圆轨道上线速度变化 | 近日点快、远日点慢 |
| “相等时间转过相等角度” | 第二定律是相等面积 | 说“面积速度相等” |
| “第三定律用近日点距离” | 第三定律用半长轴 `a` | $a=(r_{\text{近}}+r_{\text{远}})/2$ |
| “所有天体 T²/a³ 都相同” | 常量取决于中心天体质量 | 只有同一中心天体才相同 |

#### 规律、证据与核心概念
同步卫星仍然是圆轨道卫星，万有引力提供向心力：

```text
GMm/r² = m(2π/T)²r
T² = 4π²r³/GM
```

对地球静止卫星，`T` 必须等于地球自转周期；高中计算常取约 `24h`，精确讨论应取一个恒星日。因此轨道半径 `r` 被唯一确定：

```text
r = ³√(GMT²/4π²)
h = r - R
```

代入地球数据，得到 $r\approx 4.2\times 10^4\ \mathrm{km}$，离地高度 $h\approx 3.6\times 10^4\ \mathrm{km}$。

**概念辨析。**
“同步”只强调周期相同；“静止”强调相对地面位置不变。若轨道倾斜，卫星虽然每天回到同一经度附近，但会在地面上方南北移动，不会固定在一点。若不在赤道平面，万有引力指向地心，不能提供一个始终让卫星悬在某个非赤道纬度上空的向心力。

- 围绕地球做近似圆轨道运动。

- 选择题：判断同步卫星是否必须在赤道上空、高度是否唯一。
- 计算题：由 $T=24h$ 求轨道半径或离地高度。
- 比较题：同步卫星与近地卫星的速度、周期、加速度比较。

**解题路径。**
1. 看到“同步卫星”，先写 $T=24h$。
2. 用 $T^{2}=4\pi ^{2}r^{3}/GM$ 求地心距 `r`。
3. 若求高度，最后写 $h=r-R$。
4. 判断概念题时检查三条件：周期相同、赤道平面、同向。
5. 比较速度时回到圆轨道公式 $v=\sqrt{GM/r}$，同步卫星轨道半径大，速度小于近地卫星。

| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “同步卫星可在任意纬度上空静止” | 静止卫星必须在赤道平面 | 写清 $i=0^{\circ}$ |

| “计算时 r=h” | 圆轨道公式用地心距 | $r=R+h$ |
| “同步卫星速度最大” | 轨道半径越大圆轨道速度越小 | 用 $v=\sqrt{GM/r}$ |

**证据任务。**
- 抓右侧第一次点火点：看同一位置从 $v_{1}$ 加速到 $vt_{1}$ 后，轨道为什么从圆变成椭圆。
- 抓左侧第二次点火点：看高轨半径越大，转移椭圆、$vt_{2}$ 和 $\Delta v_{2}$ 怎样变化。
- 拖红色卫星沿转移椭圆走：看近地点快、远地点慢，理解椭圆上不能套圆轨公式。
- 切到高圆阶段，比较 $v_{2}$ 与 $v_{1}$，理解“升轨过程要加速，但最终高圆轨道反而低速”。

#### 规律、证据与核心概念
圆轨道速度来自万有引力提供向心力：

```text
v_c = √(GM/r)
```

转移轨道是椭圆，不能再用圆轨速度。高考核心要求是理解变轨过程中的速度、能量和轨道变化；下面的椭圆轨道速度（vis-viva）公式属于**拓展推导，不要求记忆**，题目使用时应给出或允许由能量关系推导：

```text
v = √[GM(2/r - 1/a)]
a = (r1 + r2)/2
```

升轨时：

```text
第一次点火：Δv1 = vt1 - v1
第二次点火：Δv2 = v2 - vt2
```

两次点火都沿速度方向加速，分别把轨道从低圆变成椭圆、再从椭圆变成高圆。

**概念辨析。**
变轨的关键是“同一点速度不同，轨道就不同”。在近地点同一位置，若速度等于 $v_{1}$，卫星留在低圆；若速度增大到 $vt_{1}$，它进入椭圆。到远地点时，若不再加速，会沿椭圆返回；再加速到 $v_{2}$，才稳定在高圆轨道。

#### 公式、变量、单位与条件
- $v_{1} = \sqrt{GM/r1}$：动画中低圆轨道速度柱，绑定右侧第一次点火点的半径 $r_{1}$。
- $vt_{1} = √[GM(2/r_{1} - 1/a)]$：近地点转移速度，对应第一次切向加速后的速度柱。
- $vt_{2} = √[GM(2/r_{2} - 1/a)]$：远地点转移速度，拖动左侧第二次点火点时同步变化。
- $v_{2} = \sqrt{GM/r2}$：高圆轨道速度，对应第二次加速圆化后的速度柱。
- $\Delta v_{1}、\Delta v_{2}$：右侧读数直接显示两次点火需要补上的速度差。

**适用边界。**
- 中心天体质量远大于卫星质量，轨道近似开普勒轨道。
- 点火时间很短，可近似为瞬时改变速度大小和轨道能量。
- 此处演示的是低圆到高圆的霍曼转移；降轨时点火方向相反。

#### 应用与迁移
- 选择题：判断变轨时速度、机械能、轨道半径如何变化。
- 计算题：比较圆轨速度与转移轨道速度。
- 综合题：同步卫星发射、远地点/近地点受力与速度判断。

**解题路径。**
1. 先判断卫星所在轨道：圆轨道还是椭圆转移轨道；
2. 圆轨道才用 $v=\sqrt{GM/r}$；
3. 同一位置比较速度：速度变大通常进入更高能轨道；
4. 近地点和远地点分别列能量或使用椭圆速度公式；
5. 判断加速/减速时，把速度方向与点火方向画出来。

#### 边界检查与易错点
- 把变轨椭圆上的速度也套 $v=\sqrt{GM/r}$。这个公式只适用于圆轨道。
- 认为升到高轨后速度一定更大。稳定高圆轨道速度比低圆小，但升轨过程需要先加速进入椭圆。
- 把“加速”理解成向外推。实际点火通常沿切线方向改变速度，轨道随能量改变。

- 抓任一颗星向外拖：看 `L` 增大时 $\omega$ 变小、`T` 变长。

- 让 $m_{1}=m_{2}$：看共同质心是否在两星中点，两个轨道半径是否相等。
- 比较 $v_{1}$ 和 $v_{2}$：看两星周期相同，但线速度不一定相同。

#### 规律、证据与核心概念
两星之间的万有引力大小相等，方向相反，这同一对引力分别提供两星的向心力：

```text
Gm1m2/L² = m1ω²r1
Gm1m2/L² = m2ω²r2
```

共同质心关系：

```text
m1r1 = m2r2
r1 + r2 = L
```

联立可得双星角速度：

```text
ω² = G(m1 + m2)/L³
T = 2π√[L³ / G(m1 + m2)]
```

**概念辨析。**
双星不是“一个绕另一个转”的单卫星模型，而是“两个都在转”。它们的向心力大小相等，但因为质量和轨道半径不同，线速度可以不同；共同点是角速度相同、周期相同。

#### 公式、变量、单位与条件
- `L`：图中两星之间的虚线距离。
- $r_{1}、r_{2}$：两星到共同质心的轨道半径。
- $m_{1}r_{1}=m_{2}r_{2}$：动画中质量变大的一侧轨道半径变小，读数同时给出两边乘积。
- $Gm_{1}m_{2}/L^{2}$：两星之间的紫色引力箭头，大小相等。
- $\omega ^{2}=G(m_{1}+m_{2})/L^{3}$：读数中的角速度。
- $v=\omega r$：右侧速度柱显示半径不同会导致线速度不同。

**适用边界。**
- 两星只受彼此万有引力，轨道近似圆形。
- 两星绕共同质心转动，角速度和周期相同。
- `L` 保持不变；若椭圆轨道或第三天体扰动明显，需要更复杂模型。

#### 应用与迁移
- 选择题：比较两星半径、速度、周期、向心力。
- 计算题：已知质量和距离求周期或角速度。
- 综合题：由观测周期、距离反推双星总质量。

**解题路径。**
1. 画共同质心，不要把某一颗星当圆心；
2. 写几何关系 $r_{1}+r_{2}=L$；
3. 写质心关系 $m_{1}r_{1}=m_{2}r_{2}$；
4. 对任一星列万有引力提供向心力；
5. 利用两星 $\omega 、T$ 相同，比较 $v=\omega r$。

#### 边界检查与易错点
- 把两星距离 `L` 当作其中一颗星的轨道半径。实际每颗星的轨道半径分别是 $r_{1}、r_{2}$。
- 认为质量大的星受力更大。两星之间是相互作用力，大小相等。
- 认为两星线速度一定相等。它们角速度相同，线速度 $v=\omega r$，半径不同则线速度不同。

**证据任务。**
- 抓红色力箭头端点绕箱子转：盯住投影 `F∥`，看它怎样从同向变短、到 $90^{\circ}$ 变为 `0`、再反向。
- 保持 $\alpha =37^{\circ}$，把 `F` 从 `10 N` 拖到 `40 N`：比较 `F∥` 和 `W` 是否都按同一倍数变大。
- 保持 $F=20\ \mathrm{N}$，抓蓝色位移终点把 `s` 从 `2 m` 拖到 `8 m`：观察箭头更长时，`W` 是否随位移成正比。
- 把 $\alpha$ 调到 $90^{\circ}$：注意红色力仍然存在，但绿色投影为 `0`，因此功为 `0`。

#### 规律、证据与核心概念
功的定义来自“力沿位移方向的分量 × 位移”。因为位移 `s` 是向量，只有力在位移方向上的分量会改变物体沿这段位移的运动效果。

对恒力 `F` 做分解：

```text
沿位移方向：F∥ = F cosα
垂直位移方向：F⊥ = F sinα
```

垂直分量 `F⊥` 只改变接触压力、约束或轨迹方向，不在这段位移方向上积累效果；沿位移方向的分量 `F∥` 才参与做功。因此：

```text
W = F∥s = Fs cosα
```

$\cos \alpha$ 负责决定符号：$\alpha <90^{\circ}$ 时 $\cos \alpha >0$，`F∥` 与位移同向，做正功；$\alpha =90^{\circ}$ 时 $\cos \alpha =0$，不做功；$\alpha >90^{\circ}$ 时 $\cos \alpha <0$，`F∥` 与位移反向，做负功。

**概念辨析。**
功是过程量，不是状态量。问一个力做了多少功，必须同时说明“哪个力”“哪段位移”“力与位移的夹角”。只说“箱子受到 20 N 的力”，不能判断功；只说“箱子走了 5 m”，也不能判断功。

判断时抓住一句话：**看这个力在位移方向上的投影**。

| 情况 | 图像判断 | 功的符号 | 典型例子 |
|---|---|---|---|
| `F∥` 与 `s` 同向 | 投影箭头顺着位移 | 正功 | 人拉箱子前进、重力使物体下落 |
| $F∥=0$ | 投影消失 | 零功 | 水平滑动时支持力不做功、匀速圆周运动中向心力对瞬时位移不做功 |
| `F∥` 与 `s` 反向 | 投影箭头逆着位移 | 负功 | 摩擦力阻碍滑动、重力阻碍竖直上抛 |

#### 公式、变量、单位与条件
- `F`：图中红色力箭头；拖红色端点远离箱子，红箭头变长，读数 `F∥` 和 `W` 同步变化。
- `s`：图中蓝色 A 到 B 位移箭头；拖蓝色 B 点，箱子从 A 到 B 的位移变长，`W` 随 `s` 成正比变化。
- $\alpha$：红色力箭头与蓝色位移箭头之间的夹角；绕箱子拖红色端点，右侧 $W-\alpha$ 图像上的当前点移动。
- $F∥=Fcos\alpha$：图中沿位移方向的投影箭头；它同向、消失、反向，分别对应正功、零功、负功。
- $W=Fs \cos \alpha$：动画右侧“做功读数”；单位是焦耳，$1\ \mathrm{J} = 1\ \mathrm{N·m}$。

**适用边界。**
- 本公式适用于**恒力**在一段**直线位移**上的做功。若力的大小或方向不断变化，需要分段求和、看 `F-s` 图像面积，或使用动能定理。
- `s` 是位移，不是路程。来回走一圈位移可能为 `0`，但摩擦力对全程仍可能做负功，这时不能把 $W=Fs \cos \alpha$ 简单套在“总位移”上。
- $\alpha$ 必须是该力与该段位移方向的夹角，不是斜面角、速度与水平夹角或题目中随便给出的角。
- 多个力同时作用时，每个力要单独算功，总功是各力做功的代数和：$W_{\text{总}} = W_{1} + W_{2} + ...$。

#### 应用与迁移
- 选择题：判断一个力是否做功、做正功还是负功，重点看力与位移方向。
- 计算题：已知 $F、s、\alpha$，直接求恒力做功，或先把力分解后求功。
- 多力问题：分别算重力、弹力、摩擦力、拉力、支持力做功，再求代数和。
- 能量综合题：把各力做功接到动能定理、机械能守恒或功能关系中。

**解题路径。**
1. 先画过程：标出研究对象从 A 到 B 的位移方向。
2. 再画力：只对题目要求的那个力或所有外力分别判断。
3. 找夹角：找“这个力”与“这段位移”的夹角 $\alpha$。
4. 算投影：先想 $F∥=Fcos\alpha$ 是同向、为零还是反向。
5. 列公式：恒力用 $W=Fs \cos \alpha$；多力问题分别列功再代数相加。
6. 查单位和符号：功的单位是 `J`，正负号必须与物理意义一致。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “有力就一定做功” | 力可能垂直位移，投影为 `0` | 先画 `F∥`，再判断功 |
| “走了多远就把 s 取多远” | 恒力公式中的 `s` 是位移；摩擦力等变方向阻力要按具体路径分段 | 先明确研究的是哪一段过程 |
| “α 是题目里出现的任意角” | $\alpha$ 只能是该力与位移方向夹角 | 在图上把力和位移箭头都画出来 |
| “负功就是力的大小为负” | 力的大小不为负，负号来自反向投影 | 用 `F∥` 的方向解释符号 |
| “支持力一定不做功” | 只在支持力与位移垂直时不做功；电梯中支持力对人可做功 | 判断垂直关系，而不是背力名 |

- 把蓝色速度端点拖回物体：注意有力也可能瞬时功率为 `0`，因为此刻速度为 `0`。

#### 规律、证据与核心概念
平均功率来自做功快慢：

```text
P = W / t
```

它描述一段时间内的平均效果。若 `W-t` 图像是一条直线，斜率就是平均功率；若图像弯曲，某一时刻的切线斜率才是瞬时功率。

瞬时功率可以由很短时间内的做功推出。短时间 $\Delta t$ 内，物体位移约为 $\Delta s=v\Delta t$，恒力在这段小位移上做功：

```text
ΔW = F Δs cosα = F v Δt cosα
```

两边除以 $\Delta t$，得到瞬时功率：

```text
P = ΔW/Δt = Fv cosα = F∥v
```

**概念辨析。**
功率回答“快不快”，功回答“多少”。汽车发动机功率大，不表示某一瞬间一定做了更多功，而表示它单位时间内能够输出更多能量。

功率也可以为负。若阻力方向与速度方向相反，`F∥` 为负，$P=F∥v<0$，表示这个力正在从物体机械能中取走能量，例如空气阻力、摩擦阻力对运动物体的功率通常为负。

#### 公式、变量、单位与条件
| 公式 | 图中对应 | 读数含义 |
|---|---|---|
| $P=W/t$ | 动画左侧 `W-t` 图像斜率 | 一段时间内平均做功快慢 |
| $1\ \mathrm{W} = 1\ \mathrm{J/s}$ | 右侧平均功率读数 | 每秒做 `1 J` 功就是 `1 W` |
| $P=Fvcos\alpha$ | 右侧可拖红色力、可拖蓝色速度、夹角 $\alpha$ | 某一时刻该力做功快慢 |
| $F∥=Fcos\alpha$ | 沿速度方向的投影箭头 | 真正参与瞬时功率的是 `F∥` |
| $P=F∥v$ | 动画右侧瞬时功率读数 | 投影同向为正，反向为负，垂直为零 |

**适用边界。**
- $P=W/t$ 是平均功率，必须对应一段明确时间和这段时间内的功。
- $P=Fvcos\alpha$ 是瞬时功率，`v` 是该时刻物体速度，$\alpha$ 是该力与速度方向的夹角。
- 若力与速度同向，功率为正；若反向，功率为负；若垂直，功率为零。
- 多个力同时作用时，可以分别求各力的功率；合力功率等于各力功率的代数和。
- 机车启动中常用 $P=Fv$，前提是牵引力与速度同向。

#### 应用与迁移
- 选择题：判断平均功率、瞬时功率、正负功率。
- 计算题：由 `W、t` 求平均功率，或由 $F、v、\alpha$ 求瞬时功率。
- 图像题：由 `W-t` 图像斜率判断功率大小。
- 机车启动题：用 $P=Fv$ 分析恒功率启动、最大速度。

**解题路径。**
1. 判断题目问平均功率还是瞬时功率。
2. 平均功率：找清一段时间内总功 `W` 和用时 `t`，列 $P=W/t$。
3. 瞬时功率：找该时刻的力 `F`、速度 `v`、二者夹角 $\alpha$，列 $P=Fvcos\alpha$。
4. 图像题：看 `W-t` 图像斜率；越陡功率越大。
5. 检查符号：阻力功率应为负，垂直速度的力功率为零。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “功越大，功率一定越大” | 还要看时间 | 用 $P=W/t$ 比较 |
| “平均功率等于某一瞬间功率” | 平均描述一段时间，瞬时描述某一时刻 | 先看题目问“一段时间”还是“某时刻” |
| “P=Fv 总能直接用” | 只在力与速度同向时成立 | 一般写 $P=Fvcos\alpha$ |
| “支持力一定功率为零” | 只有支持力与速度垂直时才为零 | 看该时刻力与速度夹角 |
| “功率不能为负” | 阻力取走机械能时功率为负 | 用 `F∥` 方向判断符号 |

图中三根箭头对应三个判断量：红色牵引力 `F`、紫色阻力 `f`、蓝色速度 `v`。机车启动题本质是把功率关系 $P=Fv$ 和牛顿第二定律 $F-f=ma$ 放在同一张图里看。

**动态表征。**
动画显示两种常考启动方式：

- **恒功率启动**：一开始就按额定功率输出，`v` 增大时 $F=P/v$ 减小，`a` 逐渐减小。
- **恒牵引力到额定功率**：先保持牵引力 $F_{0}$ 不变，汽车匀加速；当 $P=F_{0}v$ 达到额定功率后，进入恒功率阶段。

右侧读数同步显示当前阶段、牵引力、输出功率、加速度、最大速度和临界速度。学生可以直接拖 `F-v` 图像上的当前点，或拖车下方蓝色速度端点，观察 `F、P、a` 如何联动。

**证据任务。**
- 选择“恒功率启动”，抓 `F-v` 图像当前点向右拖：看 $F=P/v$ 是否变小，$a=(F-f)/m$ 是否逐渐变小。
- 把当前点拖到接近绿色 `vmax` 线：看 `F` 是否接近 `f`，加速度是否接近 `0`。
- 选择“恒牵引力到额定功率”，从小速度开始拖动：先看 `F` 不变、$P=Fv$ 增大；超过橙色临界线 $v_{1}=P_{\text{额}}/F_{0}$ 后，进入恒功率阶段。
- 增大阻力 `f`：看最大速度 $vmax=P_{\text{额}}/f$ 为什么会变小。

#### 规律、证据与核心概念
机车受到牵引力 `F` 和阻力 `f`。沿运动方向列牛顿第二定律：

```text
F - f = ma
```

功率与牵引力、速度关系为：

```text
P = Fv
```

若恒功率启动，$P=P_{\text{额}}$，所以：

```text
F = P额/v
a = (P额/v - f)/m
```

随着 `v` 增大，`F` 减小，`a` 减小。当 $a=0$ 时：

```text
F = f
P额 = fvmax
vmax = P额/f
```

若先恒牵引力启动，$F=F_{0}$，则 $a=(F_{0}-f)/m$ 为常量，功率 $P=F_{0}v$ 随速度增大。当 $P=F_{0}v$ 达到额定功率 $P_{\text{额}}$，临界速度为：

```text
v1 = P额/F0
```

之后不能继续保持 $F_{0}$，否则功率会超过额定值，只能转入恒功率阶段。

**概念辨析。**
机车启动题不是单纯的功率题，也不是单纯的牛顿第二定律题。它的核心链条是：

```text
功率限制牵引力 → 牵引力与阻力决定合力 → 合力决定加速度 → 加速度改变速度 → 速度又反过来改变牵引力
```

因此恒功率启动不是匀加速运动；速度越大，牵引力越小，加速度越小。最大速度不是题目给的“终点”，而是由 $F=f$ 这个动力学条件自然得到。

#### 公式、变量、单位与条件
| 公式 | 图中对应 | 读数含义 |
|---|---|---|
| $P=Fv$ | 红色牵引力箭头和蓝色速度箭头 | 牵引力与速度同向时的瞬时功率 |
| $F=P_{\text{额}}/v$ | 恒功率阶段的 `F-v` 曲线和可拖当前点 | 速度越大，牵引力越小 |
| $F-f=ma$ | 红色牵引力与紫色阻力的差 | 合力决定加速度 |
| $vmax=P_{\text{额}}/f$ | `F-v` 曲线与 $F=f$ 水平线交点 | 最大速度处加速度为零 |
| $v_{1}=P_{\text{额}}/F_{0}$ | 恒牵引力阶段转恒功率阶段的分界点 | 达到额定功率后不能继续保持 $F_{0}$ |

**适用边界。**
- 默认机车在水平直线上运动，牵引力与速度同向，阻力大小近似恒定。
- $P=Fv$ 中的 `F` 是牵引力，不是合力。
- $vmax=P_{\text{额}}/f$ 只适用于阻力恒定、水平运动、达到额定功率后继续加速的模型。
- 若题目给坡道、变阻力或空气阻力随速度变化，必须重新列受力和功率关系，不能直接套 `P/f`。
- 若题目问“平均功率”，不能直接用某一时刻的 `Fv`。

#### 应用与迁移
- 选择题：判断恒功率启动时 `F、a、v` 的变化趋势。
- 图像题：识别 `v-t`、`a-t`、`F-v`、`P-v` 图像。
- 计算题：求最大速度、临界速度、某时刻加速度。
- 综合题：先恒牵引力后恒功率的分段启动。

**解题路径。**
1. 判断启动方式：恒功率，还是先恒牵引力再恒功率。
2. 写功率关系：牵引力与速度同向时 $P=Fv$。
3. 写动力学关系：$F-f=ma$。
4. 求最大速度：令 $a=0$，得 $F=f$，再用 $P_{\text{额}}=fvmax$。
5. 分段题求临界速度：恒牵引力阶段达到额定功率时 $v_{1}=P_{\text{额}}/F_{0}$。
6. 图像题检查趋势：恒功率阶段 `F` 下降，`a` 下降，`v-t` 斜率变小。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “恒功率启动就是匀加速” | 恒功率时 $F=P/v$ 变小，合力变小 | 用 $a=(P/v-f)/m$ 判断 |
| “P=Fv 里的 F 是合力” | 发动机功率对应牵引力做功，不是合力做功 | `F` 写牵引力，合力另写 `F-f` |
| “最大速度时牵引力为 0” | 最大速度时合力为 0，不是牵引力为 0 | 条件是 $F=f$ |
| “阻力越大最大速度越大” | $vmax=P_{\text{额}}/f$，阻力在分母 | 阻力越大，最大速度越小 |
| “先恒牵引力可一直保持 F0” | 超过 $v_{1}=P_{\text{额}}/F_{0}$ 会使功率超过额定值 | 到临界速度后转恒功率 |

|---|---|---|

| 固定 `h`，增大 $\mu$ | `Wf` 更负，$W_{\text{合}}$ 变小，$v_{2}$ 变小 | 摩擦做负功，减少动能增量 |
| 固定 $h、\mu$，改变 $\theta$ | 斜面越缓，路程越长，摩擦负功越多 | 重力看高度，摩擦看路程 |

#### 规律、证据与核心概念
先从最简单的直线恒力运动看它怎么来。若物体受合力 $F_{\text{合}}$，沿位移 `s` 运动，由牛顿第二定律和运动学公式：

```text
F合 = ma
v2² - v1² = 2as
```

把 $a=(v_{2}^{2}-v_{1}^{2})/(2s)$ 代入 $F_{\text{合}}=ma$，再乘以位移 $s$：

```text
F合 s = 1/2 mv2² - 1/2 mv1²
```

左边 $F_{\text{合}} s$ 是合力做功，右边是动能变化，所以得到：

```text
W合 = ΔEₖ = Ek2 - Ek1
```

更一般的曲线运动、变力运动，也可以把过程分成很多小段，每小段都用“力在位移方向上的功”累加，最后仍得到同一结论：**合力做功等于动能变化**。

**概念辨析。**
动能定理解决的是这类问题：

```text
我不想一步步求加速度和时间，
只想知道从 A 到 B 以后速度变成多少。
```

它把题目拆成两边：

- 左边看过程：这一路上所有外力一共做了多少功。
- 右边看状态：末动能比初动能多了多少。

所以动能定理特别适合“路程弯、受力多、过程复杂，但只问初末速度”的题。

#### 公式、变量、单位与条件
对照上面的斜面图和动画读数：

| 公式 | 图中对应 | 什么时候为正/负 |
|---|---|---|
| $W_{\text{合}} = \Delta E_{k}$ | 动画右侧 $W_{\text{合}}$ 柱和 $\Delta E_{k}$ 柱必须相等 | $W_{\text{合}}>0$ 速度变大，$W_{\text{合}}<0$ 速度变小 |
| $\Delta E_{k} = 1/2 mv_{2}^{2} - 1/2 mv_{1}^{2}$ | A 点速度 $v_{1}$、B 点速度 $v_{2}$ | 动能是标量，只看速率大小 |
| $WG = mgh$ | 图中竖直高度差 `h`，不是斜面长度 | 下滑时重力做正功 |
| $Wf = -\mu_{k}mg\cos\theta \cdot s$ | 仅限本页固定粗糙斜面、持续相对滑动，且法向力为 $mg\cos\theta$ | 动摩擦力对滑块做负功 |
| $WN = 0$ | 支持力垂直斜面位移 | 垂直位移的力不做功 |

这就是为什么不能只写“$W=mgh$”：如果有摩擦，真正进入动能变化的是 $W_{\text{合}} = WG + Wf + WN$。

**适用边界。**
- 研究对象必须明确，例如“滑块”或“小球”。
- 研究过程必须明确，例如“从 A 点到 B 点”。
- $W_{\text{合}}$ 是这个过程中**所有外力做功的代数和**，不是某一个力做功。
- $\Delta E_{k}$ 是末动能减初动能：$Ek_{2} - Ek_{1}$，顺序不能反。
- 可以用于直线、曲线、恒力、变力；但每个力做功必须能算清楚或用图像/面积/题给信息得到。
- 涉及内力、系统能量转化时，要先判断研究对象，不能随便把系统内力当外力。

#### 应用与迁移
- 选择题：判断某段过程中动能增大、减小还是不变。
- 计算题：已知高度、摩擦、拉力或阻力，求末速度。
- 多过程题：上坡、下坡、粗糙面、弹簧段分段求总功。
- 曲线运动题：不方便列运动学公式时，用合力做功直接联系速度。

**解题路径。**
1. **画过程**：标出 A 点、B 点、位移方向和高度差。
2. **选对象**：通常选一个物体；若选系统，要重新判断哪些力是外力。
3. **列做功表**：重力、拉力、摩擦、支持力分别做多少功。
4. **求总功**：$W_{\text{合}} = W_{1} + W_{2} + ...$，注意负功。
5. **写动能变化**：$\Delta E_{k} = 1/2 mv_{2}^{2} - 1/2 mv_{1}^{2}$。
6. **列等式**：$W_{\text{合}} = \Delta E_{k}$。
7. **检查结果**：若总功为正，速度应变大；若算出相反，回头查符号。

#### 边界检查与易错点
- 把某个力做功当合力做功（W合 是所有力做功代数和）。
- 忘记摩擦力做负功。
- 混淆动能和动量（动能是标量 ½mv²，动量是矢量 mv）。
- 把高度 `h` 和斜面路程 `s` 混用：重力做功看高度，摩擦做功看路程。
- 把 $\Delta E_{k}$ 写反：应是末动能减初动能，不是初动能减末动能。

**错误诊断卡**

| 学生写法 | 问题 | 改法 |
|---|---|---|
| $mgh = 1/2 mv^{2}$ | 只适用于无摩擦、由静止下滑 | 有摩擦时写 $mgh - \mu mgcos\theta \cdot s = 1/2 mv^{2}$ |
| $Wf = \mu mgcos\theta \cdot s$ | 忘了摩擦方向与位移相反 | 应为 $Wf = -\mu mgcos\theta \cdot s$ |
| $WG = mg\cdot s$ | 把斜面路程当高度 | 应为 $WG=mgh=mg\cdot s\cdot \sin \theta$ |
| $W_{\text{合}} = Ek_{1} - Ek_{2}$ | 动能变化顺序写反 | 应为 $W_{\text{合}}=Ek_{2}-Ek_{1}$ |

#### 情境与现象
同一个物体放在桌面上方，若把地面选为零势能面，它的重力势能为正；若把更高的桌面选为零势能面，它的重力势能可能为零甚至为负。学生容易因此困惑：势能到底是真实的，还是随便变的？

关键是分清两件事：

```text
重力势能 Ep 的数值依赖参考面；
重力势能的变化 ΔEp 和重力做功 WG 只由初末高度差决定。
```

**证据任务。**
- 固定 $h_{1}=7\ \mathrm{m}、h_{2}=2\ \mathrm{m}$，拖动橙色参考面线 $h_{0}$：看 $Ep_{1}$ 和 $Ep_{2}$ 是否改变，再看 $\Delta E_{p}$ 和 `WG` 是否不变。
- 直接把 B 点拖到比 A 点更高：看重力做功如何从正变负，势能变化如何从负变正。
- 增大质量 `m`：看 $E_{p}、\Delta E_{p}、WG$ 是否都按比例变大。
- 把 A、B 两点拖到等高：看参考面怎么选，重力做功都为 `0`。

#### 规律、证据与核心概念
重力做功与路径无关，只与高度差有关。物体从高度 $h_{1}$ 到 $h_{2}$：

```text
WG = mg(h1 - h2)
```

为了把“重力做功”改写成“某个状态量的变化”，定义重力势能：

```text
Ep = mg(h - h0)
```

其中 $h_{0}$ 是参考面高度。于是：

```text
ΔEp = Ep2 - Ep1
     = mg(h2-h0) - mg(h1-h0)
     = mg(h2-h1)
```

所以：

```text
WG = mg(h1-h2) = -ΔEp
```

这说明重力做正功时，重力势能减少；重力做负功时，重力势能增加。

**概念辨析。**
重力势能属于“物体-地球”系统，不是物体孤立拥有的东西。参考面只是人为规定的零点，就像温度可以用摄氏度或开尔文表示，零点变了，数值会变，但实际温差不乱。

所以做题时不要问“这个物体势能到底是多少”而不选参考面。更重要的是看势能变化：同一过程的 $\Delta E_{p}$ 和 `WG` 不会因为参考面改变而改变。

#### 公式、变量、单位与条件
| 公式 | 图中对应 | 读数含义 |
|---|---|---|
| $E_{p}=mg(h-h_{0})$ | A/B 点到可拖橙色参考面的竖直高度 | 势能数值可正、零、负 |
| $\Delta E_{p}=Ep_{2}-Ep_{1}$ | 右侧能量柱 $Ep_{2}$ 减 $Ep_{1}$ | 末状态势能减初状态势能 |
| $WG=mg(h_{1}-h_{2})$ | 可拖 A、B 两点的竖直高度差 | 下降为正，上升为负 |
| $WG=-\Delta E_{p}$ | 动画中 `WG` 柱与 $\Delta E_{p}$ 柱大小相等、方向相反 | 重力做功等于势能减少量 |

**适用边界。**
- 近地面重力场中，`g` 近似恒定，可用 $E_{p}=mgh$。
- 必须先选参考面，才能谈某一点的势能数值。
- 势能可以为正、零或负；这不代表能量“没有意义”，只代表相对所选参考面的位置。
- 重力做功只与初末高度差有关，与路径无关。
- 若高度变化很大、`g` 不再近似恒定，需要用更一般的万有引力势能表达，高中近地面模型不再直接适用。

#### 应用与迁移
- 选择题：判断参考面改变后 $E_{p}、\Delta E_{p}、WG$ 哪些变、哪些不变。
- 计算题：由高度差求重力做功或势能变化。
- 机械能题：把 $E_{p}=mgh$ 与动能 $E_{k}=1/2mv^{2}$ 组合。
- 功能关系题：用 $WG=-\Delta E_{p}$ 判断能量转化。

**解题路径。**
1. 先选参考面，写清 `h` 是相对参考面的高度。
2. 求某点势能，用 $E_{p}=mg(h-h_{0})$。
3. 求势能变化，用 $\Delta E_{p}=Ep_{2}-Ep_{1}$。
4. 求重力做功，优先用 $WG=mg(h_{1}-h_{2})$。
5. 检查符号：下降时 $WG>0、\Delta E_{p}<0$；上升时 $WG<0、\Delta E_{p}>0$。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “不选参考面也能说 Ep” | 势能数值依赖零势能面 | 先说明参考面 |
| “势能不能为负” | 参考面以上为正，以下可为负 | 负势能只表示低于参考面 |
| “参考面改变，重力做功也改变” | 重力做功只看初末高度差 | 用 $WG=mg(h_{1}-h_{2})$ 检查 |
| “WG=ΔEp” | 符号反了 | 正确是 $WG=-\Delta E_{p}$ |
| “重力做功看路径长度” | 重力是保守力 | 只看竖直高度差 |

- 后续应用：B2-21 机械能守恒、B2-22 功能关系、M-08 弹簧模型

#### 情境与现象
把弹簧压短或拉长后松手，弹簧能把小球弹出去。说明被压缩或拉伸的弹簧储存了能量。离原长越远，弹簧恢复原状的能力越强，储存的弹性势能越大。

这里最容易错的是符号 `x`：它不是弹簧现在的总长度，而是**相对原长的形变量**。




> **公式首次使用卡**：适用边界——弹簧必须在弹性限度内，满足胡克定律 F=kx；x 是相对原长的形变量，拉伸和压缩都可代入，但势能取决于 x^{2}；弹性势能通常取原长位置为零势能位置。符号含义在紧邻公式的正文、图注或表格中定义；除题目另有单位外，代入前统一为 SI，结果保留单位并做量纲检查。

```text
x = 现长 - 原长
```

拉伸可取 $x>0$，压缩可取 $x<0$，但弹性势能与 $x^{2}$ 成正比，所以拉伸和压缩同样大小时，弹性势能相同。

**证据任务。**
- 直接拖动画面里的滑块远离原长：看蓝色 `x` 尺、红色弹力箭头和 `Ep` 抛物线高度是否一起变大。

- 设置或拖成 $x_{1}=0.4\ \mathrm{m}、x_{2}=0.1\ \mathrm{m}$：看弹簧靠近原长时，弹力做正功，弹性势能减少。
- 设置或拖成 $x_{1}=0.1\ \mathrm{m}、x_{2}=0.4\ \mathrm{m}$：看外力把弹簧拉得更远时，弹力做负功，弹性势能增加。
- 比较 $x=0.4\ \mathrm{m}$ 和 $x=-0.4\ \mathrm{m}$：看拉伸和压缩同样大小时，弹性势能相同，但弹力方向相反。

#### 规律、证据与核心概念
弹簧弹力满足胡克定律：

```text
F弹 = -kx
```

负号表示弹力方向总是指向原长。弹性势能定义为“弹力做功的相反数”。当弹簧从 $x_{1}$ 变化到 $x_{2}$ 时，弹力做功等于弹性势能减少：

```text
W弹 = Ep1 - Ep2 = -ΔEp
```

由 `F-x` 图像看，弹力大小随形变量线性变化，储存的能量对应三角形面积：

```text
Ep = 1/2 × kx × x = 1/2 kx²
```

**概念辨析。**
弹性势能属于“发生弹性形变的物体或弹簧系统”。弹簧不变形时，$x=0$，弹性势能为 `0`。离原长越远，无论是拉长还是压短，弹簧恢复原状的能力越强，弹性势能越大。

弹力做功的符号看“是否靠近原长”：靠近原长，弹力方向与运动趋势一致，弹力做正功；远离原长，外力把弹簧越拉越远或越压越短，弹力做负功。

#### 公式、变量、单位与条件
| 公式 | 图中对应 | 读数含义 |
|---|---|---|
| $x=\text{现长}-\text{原长}$ | 可拖动滑块与蓝色形变量尺 | 不是弹簧总长度 |
| $F_{\text{弹}}=-kx$ | 红色弹力箭头与当前力读数 | 弹力指向原长，大小随 `|x|` 增大 |
| $E_{p}=1/2kx^{2}$ | 可拖动 $x_{1}/x_{2}$ 点、`Ep-x` 抛物线和能量柱 | 拉伸、压缩同样大小时势能相同 |
| $W_{\text{弹}}=-\Delta E_{p}$ | 底部能量账本的 $\Delta E_{p}$ 与 $W_{\text{弹}}$ 两列 | 弹力做功等于弹性势能减少 |

**适用边界。**
- 弹簧必须在弹性限度内，满足胡克定律 $F=kx$。
- `x` 是相对原长的形变量，拉伸和压缩都可代入，但势能取决于 $x^{2}$。
- 弹性势能通常取原长位置为零势能位置。
- 非理想弹簧、超过弹性限度或发生塑性形变时，$E_{p}=1/2kx^{2}$ 不再直接适用。
- 多弹簧或弹簧与物体组成系统时，要先明确研究对象和能量归属。

#### 应用与迁移
- 选择题：判断弹簧拉伸、压缩时弹性势能大小和弹力做功正负。
- 计算题：由 `k、x` 求弹性势能，或由势能变化求速度。
- 图像题：由 `F-x` 图像面积求弹性势能。
- 综合题：弹簧与动能、重力势能、机械能守恒结合。

**解题路径。**
1. 找原长位置，确定形变量 `x`。
2. 判断是否在弹性限度内，可否用胡克定律。
3. 求弹性势能：$E_{p}=1/2kx^{2}$。
4. 求弹力做功：$W_{\text{弹}}=Ep_{1}-Ep_{2}$。
5. 检查物理意义：靠近原长时弹性势能减少，弹力做正功。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “x 是弹簧长度” | 公式中的 `x` 是形变量 | 先标原长，再量现长与原长差 |
| “压缩时弹性势能为负” | $E_{p}=1/2kx^{2}$ 恒不小于 0 | 压缩和拉伸同样大小势能相同 |
| “弹力做功等于 ΔEp” | 符号反了 | 正确是 $W_{\text{弹}}=-\Delta E_{p}$ |
| “弹簧越短势能一定越大” | 要看离原长多远 | 比较 `|x|` |
| “任何弹簧都能用 1/2kx²” | 超过弹性限度不满足胡克定律 | 先判断模型条件 |

| 操作 | 画面读数 | 应得结论 |
|---|---|---|
| $\mu =0$，增大 `h` | `Ep` 增大，底端 `Eₖ` 等量增大，$E_{\text{机}}$ 不变 | 只有重力做功时机械能守恒 |

#### 规律、证据与核心概念
机械能是动能、重力势能和弹性势能的总和：

```text
E机 = Eₖ + Ep重 + Ep弹
```

若运动过程中只有重力或弹力做功，重力做功等于重力势能减少，弹力做功等于弹性势能减少；这些“减少”会转化为动能增加，机械能总量不变：

```text
Ek1 + Ep1 = Ek2 + Ep2
```

如果有摩擦力、空气阻力等非保守力做功，机械能会转化为内能、声能等，机械能不守恒。此时不能写 $E_{1}=E_{2}$，应改用功能关系或总能量守恒。

**概念辨析。**
机械能守恒的条件是“只有重力或弹力做功”，不是“只受重力或弹力”。可以有支持力、绳的拉力、轨道压力等其他力存在，只要这些力不做功，机械能仍可能守恒。

判断时不要背力名，要看做功：支持力若始终垂直位移，不做功；绳拉力若方向始终垂直速度，也不做功；摩擦力通常与位移反向，做负功，会使机械能减少。

#### 公式、变量、单位与条件
| 公式 | 图中对应 | 读数含义 |
|---|---|---|
| $E_{p}=mgh$ | 橙色高度尺和紫色势能柱 | 高度越低，重力势能越小 |
| $E_{k}=1/2mv^{2}$ | 小球速度箭头和蓝色动能柱 | 下滑越快，动能越大 |
| $E_{\text{机}}=E_{k}+E_{p}$ | 绿色机械能柱和过程图像 | 光滑时绿色柱不变，表示机械能守恒 |
| $\Delta E_{\text{机}}=W_{\text{其他}}$ | 粗糙段、红色内能柱 | 摩擦做负功，机械能减少 |
| $E_{\text{总}}=E_{\text{机}}+E_{\text{内}}$ | 橙色总能量柱 | 即使机械能不守恒，总能量仍守恒 |

**适用边界。**
- 研究对象或系统必须明确。常见选择是“物体 + 地球”或“物体 + 弹簧”。
- 只有重力或弹力做功，或其他力做功代数和为零时，机械能守恒。
- 摩擦力、空气阻力、碰撞损失等非保守力做功时，机械能不守恒。
- 需统一参考面计算重力势能，初末状态不能换参考面。
- 轨道支持力、细线拉力不一定破坏守恒，关键看它是否做功。

#### 应用与迁移
- 选择题：判断机械能是否守恒。
- 实验题：验证机械能守恒定律。
- 计算题：用机械能守恒求速度、高度。
- 综合题：圆周+机械能、弹簧+机械能。

**解题路径。**
1. 选系统：物体、地球、弹簧是否纳入系统。
2. 查做功：除重力、弹力外，其他力是否做功。
3. 选参考面：通常选最低点为零势能面。
4. 写初态机械能：$E_{1}=Ek_{1}+Ep_{1}$。
5. 写末态机械能：$E_{2}=Ek_{2}+Ep_{2}$。
6. 若满足守恒，列 $E_{1}=E_{2}$；若不满足，改写 $\Delta E_{\text{机}}=W_{\text{其他}}$。
7. 检查结果：高度降低时速度应增大；若有摩擦，速度应小于光滑情形。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “有支持力就不守恒” | 支持力若垂直位移，做功为 0 | 看做功，不看力名 |
| “有摩擦也能直接 E1=E2” | 摩擦做负功，机械能转化为内能 | 改用 $\Delta E_{\text{机}}=Wf$ 或能量守恒 |
| “参考面随便换” | 同一道题初末势能必须用同一参考面 | 先选定参考面再列式 |
| “机械能守恒就是能量守恒” | 机械能只是总能量的一部分 | 区分机械能和总能量 |
| “圆周最高点只用机械能就够” | 还要满足向心力临界条件 | 能量求速度，再判临界 |

| 改变质量 `m` 或拖动过程位置 | $W_{\text{合}}$ 与 $\Delta E_{k}$ | 合力做功对应动能变化 |
| 对比所有读数 | $W_{\text{合}}、WG、Wf$ | 不同力做功对应不同能量，不可混写 |

#### 规律、证据与核心概念
常用功能关系必须和研究对象、系统边界一起使用：
- 合力做功 = 动能变化（动能定理）
- 重力做功 = 重力势能减少（势能定义）
- 弹力做功 = 弹性势能减少
- 除重力、弹力以外的外力对所选系统做功 = 系统机械能变化

把它写成一张“做功-能量”账本：

```text
W合 = ΔEₖ
WG = -ΔEp重
W弹 = -ΔEp弹
W其他 = ΔE机（先选系统）
本页固定粗糙斜面模型：Wf = ΔE机
两接触物相对滑动生热：Q = fk·s相对
```

功能关系不是新公式堆叠，而是先问清“研究谁、系统边界在哪里、哪个力做功”，再确定改变的是哪一种能量。摩擦力可以做正功、负功或零功；摩擦生热取决于接触面的相对滑动路程，二者不能无条件画等号。

**概念辨析。**
功能关系是"什么力做功改变什么能量"的对应表。做题时先判断"什么力做功"，再找"对应什么能量变化"。这比直接用动能定理更有针对性。

#### 公式、变量、单位与条件
| 公式 | 图中对应 | 读数含义 |
|---|---|---|
| $W_{\text{合}}=\Delta E_{k}$ | 右侧第一行与底部蓝色两列 | 合力总功对应动能变化 |
| $WG=-\Delta E_{p}=mg(h_{1}-h_{2})$ | 橙色高度尺、重力箭头、紫色两列 | 重力做功对应重力势能减少 |
| $Wf=\Delta E_{\text{机}}$ | 本页固定粗糙斜面、滑块为研究对象、动摩擦为唯一其他力 | 仅为当前演示模型的特例 |
| $W_{\text{其他}}=\Delta E_{\text{机}}$ | 机械能账本 | 除重力、弹力外的力做功改变机械能 |
| $Q=f_{k}s_{\text{相对}}$ | 两接触物组成的系统 | 相对滑动产生的内能增加 |

**适用边界。**
- $W_{\text{合}}=\Delta E_{k}$ 永远针对合力总功，不是某一个力。
- $WG=-\Delta Ep_{\text{重}}$ 只针对重力做功，且重力做功与路径无关。
- $W_{\text{弹}}=-\Delta Ep_{\text{弹}}$ 只针对弹力做功，前提是弹簧在弹性限度内。
- $W_{\text{其他}}=\Delta E_{\text{机}}$ 中“其他力”指除重力、弹力以外的力对系统做功。
- 摩擦力对某个物体可做正功、负功或零功；不能用“摩擦力一定做负功”判断。
- $Q=f_{k}s_{\text{相对}}$ 描述两接触物相对滑动产生的内能，不能用其中某一个物体的位移替代相对路程。

#### 应用与迁移
- 选择题：判断力做功与能量变化对应关系。
- 计算题：用功能关系求速度、高度、摩擦力。
- 综合题：多过程能量转化分析。

**解题路径。**
1. 画过程，确定初末状态和研究对象。
2. 列做功清单：重力、弹力、摩擦力、拉力、支持力分别是否做功。
3. 给每个关系先配研究边界：合力看单个物体动能，重力/弹力看相应势能，其他外力看所选系统机械能。
4. 根据题目目标选关系：求速度常用 $W_{\text{合}}=\Delta E_{k}$，求机械能损失常用 $W_{\text{其他}}=\Delta E_{\text{机}}$。
5. 检查符号和边界：势能减少对应保守力做正功；只有在本页限定模型中，摩擦负功才直接等于机械能减少。

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “某个力做功等于 ΔEₖ” | 动能定理要求合力总功 | 写清 $W_{\text{合}}=ΣWi$ |
| “摩擦力做功等于动能变化” | 摩擦只是众多力之一 | 若求动能，先求合力总功 |
| “W其他=ΔEₖ” | 其他力做功对应机械能变化 | 应为 $W_{\text{其他}}=\Delta E_{\text{机}}$ |
| “摩擦做功就等于生热” | 功针对某个物体的位移，生热取决于两接触面的相对滑动 | 用 $Q=f_{k}s_{\text{相对}}$，并先选系统 |
| “重力做功看斜面长” | 重力做功只看高度差 | 用 $WG=mg\Delta h$ |
| “功能关系随便套一个” | 每个关系都有对应力和能量 | 先问哪个力，再问哪种能量 |

#### 规律、证据与核心概念
能量守恒定律说：能量既不会凭空产生，也不会凭空消失，只会从一种形式转化为另一种形式，或从一个物体转移到另一个物体。

对本页“滑块 + 斜面 + 地球”近似封闭系统，斜面固定、滑块持续相对滑动且忽略声等其他能量时：

```text
初始能量：E0 = mgh
相对滑动生热：E内 = Q = fk·s相对
本固定斜面特例：Q = -Wf(滑块)
末动能：Eₖ = E0 - E内
总能量：Eₖ + E内 = E0
```

若没有摩擦，$E_{\text{内}}=0$，机械能守恒。在本页固定粗糙斜面模型中，机械能减少量等于滑块—斜面系统增加的内能；更一般情境还可能有外界做功、声能或接触面双方的机械能变化，必须重新列能量账本。

**概念辨析。**
机械能守恒是能量守恒的特殊情况。它要求机械能内部的动能、重力势能、弹性势能相互转化，而没有被摩擦、碰撞、空气阻力等过程转化成内能、声能等。

能量守恒更普遍。即使机械能不守恒，只要把内能、声能、电能等都算进系统，总能量仍守恒。

| 公式 | 图中对应 | 读数含义 |

| $E_{\text{内}}=Q=f_{k}s_{\text{相对}}$ | 内能柱 | 两接触物相对滑动产生的内能 |
| $Q=-Wf$ | 仅限本页固定斜面特例 | 滑块位移等于相对滑动路程，且斜面不动 |
| $E_{k}+E_{\text{内}}=E_{0}$ | 总能量柱保持不变 | 能量形式改变，总量不变 |
| $E_{0}=mgh$ | 斜面高度 `h` | 初始重力势能 |

**适用边界。**
- 能量守恒适用于所有物理过程，但必须把相关能量形式和系统边界选清楚。
- 本页总能量账本选择“滑块 + 斜面 + 地球”为近似封闭系统；若系统与外界交换能量，必须把传热和外界做功计入。
- 机械能守恒只在没有非保守力做功，或非保守力做功为零时成立。
- 有摩擦、碰撞、空气阻力时，机械能通常不守恒，但总能量守恒。
- 若题目只要求机械能，不能把内能混进去；若题目问总能量，不能漏掉热、声、电等形式。

#### 应用与迁移
- 选择题：判断“机械能守恒”和“能量守恒”的区别。
- 计算题：由机械能损失求内能增加或摩擦做功。
- 实验题：解释实验中机械能减少去了哪里。
- 综合题：多过程能量转化路径分析。

**解题路径。**
1. 先选系统：物体、地球、弹簧、斜面是否包含在内。
2. 列能量形式：动能、重力势能、弹性势能、内能等。
3. 判断机械能是否守恒：看是否有非保守力做功。
4. 若机械能不守恒，把减少的机械能转化到内能、声能等。
5. 写总能量守恒：初态总能量 = 末态总能量。

| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “机械能不守恒就是能量不守恒” | 机械能只是总能量的一部分 | 把内能等形式算进去 |
| “摩擦力做功使能量消失” | 能量转移或转化，不能只看滑块机械能 | 先选系统；相对滑动生热写 $Q=f_{k}s_{\text{相对}}$ |

#### 核心物理机制与证据
狭义相对论的两个基本假设是：

```text
所有惯性参考系中物理规律相同；
真空中光速 c 对所有惯性观察者都相同。
```

由这两个假设可推出相对论因子：

```text
γ = 1 / √(1 - v²/c²) = 1 / √(1-β²)
```

当 $\beta$ 很小时，$\gamma \approx 1$，相对论公式退化为经典近似；当 $\beta$ 接近 `1` 时，$\gamma$ 急剧增大：

```text
时间膨胀：Δt = γΔt0
长度收缩：L = L0/γ
```

**概念辨析。**
“运动钟变慢”不是说钟坏了，而是不同惯性参考系对同一过程的时间测量不同。$\Delta t_{0}$ 是事件发生在同一地点的参考系测得的固有时间；外部观察者测得的时间间隔更长。

“长度收缩”只发生在运动方向上，垂直运动方向的长度不收缩。它也不是视觉透视，而是高速运动下长度测量的物理规律。

#### 定量关系、变量、单位与边界
| 公式 | 图中对应 | 读数含义 |
|---|---|---|
| $\beta =v/c$ | 速度滑块和可拖动红点 | 表示速度占光速的比例 |
| $\gamma =1/\sqrt{1-\beta ^{2}}$ | $\gamma -\beta$ 曲线上的可拖动当前点 | 接近光速时急剧增大 |
| $\Delta t=\gamma \Delta t_{0}$ | 光钟读数 | 外部观察运动钟时间间隔更长 |
| $L=L_{0}/\gamma$ | 飞船长度条 | 运动方向长度变短 |

**适用边界。**
- 这些公式属于狭义相对论，适用于惯性参考系和匀速相对运动。
- 只有速度接近光速时效应显著；日常低速 $\beta$ 很小，经典力学足够准确。
- 长度收缩只沿运动方向发生。
- 不能把相对论效应理解为视觉错觉或仪器误差。
- 高中阶段只要求基础理解和简单代入，不展开广义相对论。

#### 系统影响与权衡
- 选择题：判断时间膨胀、长度收缩是否显著。
- 计算题：给定 $\beta$，求 $\gamma 、\Delta t、L$。
- 概念题：区分固有时间、外部观察时间和运动方向长度。
- 判断题：低速极限下相对论公式是否回到经典近似。

**解题路径。**
1. 先算 $\beta =v/c$。
2. 求 $\gamma =1/\sqrt{1-\beta ^{2}}$。
3. 时间题用 $\Delta t=\gamma \Delta t_{0}$。
4. 长度题用 $L=L_{0}/\gamma$，并确认方向是运动方向。
5. 若 $\beta$ 很小，可判断相对论效应极小。

| 易错说法 | 为什么错 | 改法 |
|---|---|---|

#### 情境与现象

临界速度：vmax = √(μgr)

- 调到更高速度：区分地球逃逸和太阳系逃逸。
- 改变发射高度：看圆轨道速度和逃逸速度为什么会变化。

#### 核心物理机制与证据
近地圆轨道由万有引力提供向心力：

```text
GMm/R² = mv1²/R
v1 = √(GM/R) ≈ 7.9 km/s
```

逃逸速度来自机械能不再为负：

```text
1/2 mv2² - GMm/R = 0
v2 = √(2GM/R) ≈ 11.2 km/s
```

教材常用的第三宇宙速度约 `16.7 km/s`，指在忽略大气、地球自转和其他天体摄动等理想条件下，从地球附近**沿地球公转方向**发射，借助地球已有的日心公转速度，以最省能方式逃离太阳系所需的最小地面附近发射速度。改变发射方向或位置，阈值必须重新计算。

#### 证据限制与常见误区
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “第一宇宙速度越高轨道越高” | 对圆轨道，半径越大速度越小 | 用 $v=\sqrt{GM/r}$ |
| “逃逸后不受引力” | 引力仍有，只是不再束缚 | 能量非负 |
| “速度大于 7.9 就一定逃逸” | 还要达到约 11.2 | 分清三档速度 |
| “任意方向达到 16.7 就一定是第三宇宙速度” | 第三宇宙速度的最小值利用地球公转速度，依赖发射方向和理想条件 | 写清沿地球公转方向和参考系 |

#### 核心物理机制与证据
牛顿力学的边界可用三个问题判断：

```text
速度是否接近光速？若是，需相对论。
引力是否极强或尺度是否宇宙级？若是，需广义相对论/宇宙学观点。
对象是否微观粒子？若是，需量子观点。
```

狭义相对论常用：

```text
γ = 1/√(1-β²)
Δt = γΔt0
L = L0/γ
```

当 $\beta$ 很小时，$\gamma \approx 1$，经典力学是相对论在低速下的近似。

#### 科学方法与证据
| 范围 | 典型对象 | 主要理论 |
|---|---|---|
| 低速弱引力宏观 | 汽车、抛体、普通机械 | 牛顿力学 |
| 高速接近光速 | 粒子加速器、高速飞船 | 狭义相对论 |
| 强引力宇宙尺度 | 黑洞、宇宙膨胀 | 广义相对论/宇宙学 |
| 微观尺度 | 电子、光子、原子 | 量子理论 |

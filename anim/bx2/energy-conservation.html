侧视轨道 + 能量图像 + 内能剖面 + 数据记录；可直接拖动滑块改变位置，拖动橙色高度手柄改变初始能量
观察任务：本实验选择“滑块 + 固定斜面 + 地球”为近似封闭系统。拖动滑块时盯住橙色总能量柱：机械能可以变少，但 `Ep+Ek+E内` 必须始终等于初始 `mgh`。
实验控制
Ep=35.0 J　Ek=41.0 J
E内=24.0 J　E机=76.0 J
E总=100.0 J　E0=100.0 J
公式对照
E0 = mgh，对应橙色高度尺。
E机 = Ep + Ek，对应紫/蓝/绿能量柱。
E内 = Q = fk·s相对；本固定斜面特例才有 Q=-Wf(滑块)。
E总 = Ep + Ek + E内 = E0。
有摩擦：机械能减少，但减少量进入内能，总能量守恒。

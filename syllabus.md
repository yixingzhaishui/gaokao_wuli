# 高考依据与使用说明

> 本站依据《四川高考物理网站开发规格 V1.2.2》开发，目标是让高三学生**看见现象、感受到变化、理解规律由来、掌握适用条件、形成解题思路**。

> **非四川省教育考试院官方项目。当前为内容审校测试版；课标映射、物理内容、答案和来源正在复核。**

<h2 id="四川新高考物理范围">四川新高考物理范围</h2>

四川省普通高考实行“3+1+2”模式，物理为首选科目之一，以原始成绩计入高考文化成绩。物理考试内容为普通高中课程方案和课程标准确定的**必修和选择性必修**内容。

本站按以下范围建设（当前不作“全部审定”承诺）：

- 必修1、必修2、必修3；
- 选择性必修1、选择性必修2、选择性必修3；
- 课程标准要求的学生必做实验；
- 高频模型、易错点、真题训练作为强化层。

<h2 id="本站学习方法">本站学习方法</h2>

每个知识点按下面链路学习，不要跳步：

```text
先看现象 → 示意图 → 示意动画 → 交互动画 → 观察任务 → 规律由来 → 概念理解 → 公式绑定 → 适用条件 → 应用题型 → 解题思路 → 易错点 → 例题训练
```

- **先看见**：用图和动画建立画面，不要先背定义。
- **再感受**：拖动参数，看物理量怎么变。
- **再抽象**：在看到现象后再记公式，公式里每个符号都要能在图里找到。
- **再应用**：用例题和方法提炼把知识变成解题能力。

<h2 id="完成度与验收说明">完成度与验收说明</h2>

每个知识点按 A/B/C 三档验收：

| 等级 | 类型 | 要求 |
|---|---|---|
| A | 核心交互知识点 | 完整 14 节 + 至少 1 个交互动画 + 2 道例题 |
| B | 基础概念知识点 | 精简 9 项：示意图 + 示意动画 + 例题 |
| C | 背景拓展知识点 | 精简 7 项：图示 + 概念 + 易错点 + 判断题 |

`data/progress.json` 现已拆分为多轴审核状态：

| 轴 | 发布要求 |
|---|---|
| `curriculum_map` | 课标条款、模块归属和主节点通过审核 |
| `physics_content` | 物理事实、条件、公式和边界通过教师审核 |
| `examples_recalculated` | 数值、符号、单位和结论完成独立复算 |
| `source_review` | 来源等级、页码、哈希和双签达到政策 2.0 |
| `pedagogy_review` | 教学梯度、表述和难度标注通过审核 |
| `interaction_qa` / `mobile_qa` | 交互和移动端测试通过 |
| `publish` | 以上各轴全部通过后才能解除 `blocked` |

旧字段 `done / partial / review` 只为兼容页面结构和交互建设统计，不代表内容审定。学生页统一显示“interaction passed · content pending”，当前所有节点的 `publish` 均为 `blocked`。

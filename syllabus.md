# 高考依据与使用说明

> 本站依据《四川高考物理网站开发规格 V1.2.2》开发，目标是让高三学生**看见现象、感受到变化、理解规律由来、掌握适用条件、形成解题思路**。

<h2 id="四川新高考物理范围">四川新高考物理范围</h2>

四川省普通高考实行“3+1+2”模式，物理为首选科目之一，以原始成绩计入高考文化成绩。物理考试内容为普通高中课程方案和课程标准确定的**必修和选择性必修**内容。

本站覆盖范围：

- 必修1、必修2、必修3；
- 选择性必修1、选择性必修2、选择性必修3；
- 课程标准要求的学生必做实验；
- 高频模型、易错点、真题训练作为强化层。

<h2 id="本站学习方法">本站学习方法</h2>

每个知识点按下面链路学习，不要跳步：

```text
先看现象 → 示意图 → 示意动画 → 交互动画 → 观察任务 → 规律由来 → 概念理解 → 公式绑定 → 适用条件 → 应用题型 → 解题思路 → 易错点 → 例题训练
```

- **先看见**：用图和动画建立画面，不要先背定义。
- **再感受**：拖动参数，看物理量怎么变。
- **再抽象**：在看到现象后再记公式，公式里每个符号都要能在图里找到。
- **再应用**：用例题和方法提炼把知识变成解题能力。

<h2 id="完成度与验收说明">完成度与验收说明</h2>

每个知识点按 A/B/C 三档验收：

| 等级 | 类型 | 要求 |
|---|---|---|
| A | 核心交互知识点 | 完整 14 节 + 至少 1 个交互动画 + 2 道例题 |
| B | 基础概念知识点 | 精简 9 项：示意图 + 示意动画 + 例题 |
| C | 背景拓展知识点 | 精简 7 项：图示 + 概念 + 易错点 + 判断题 |

页面状态标签（来自 `data/progress.json`）：

| 状态 | 含义 |
|---|---|
| pending | 未开始 |
| draft | 有文字草稿 |
| partial | 有部分图/动画/例题，但未达标 |
| review | 等待审核 |
| done | 满足对应等级验收标准 |

> 标注为 `pending / draft / partial` 的页面会显示“正在开发中”提示，不冒充完成。

#### 情境与现象

同一辆汽车，研究“从成都到重庆用了多久”时，车身长短、车轮转动、车头朝向都不影响行程计算，可以把整车看作一个点；但研究车轮转动、汽车掉头、车身是否越过停止线时，大小、形状或姿态就不能忽略。

质点不是一种物体，而是一种模型：在这个问题中，把物体的大小和形状暂时忽略。

- 保持研究目标为“行程”，拖动终点手柄增大研究范围 `s`：看判据尺上的 `l/s` 怎样变小，为什么更容易看作质点。

- 切换到“转动”或“姿态”：即使物体很小，也看它为什么不能被一个点替代。
- 增大物体长度 `l` 或减小距离 `s`：看尺寸相对运动范围不可忽略时，模型如何失效。

#### 规律、证据与核心概念

物理建模要抓主要因素、舍次要因素。若物体的大小、形状、转动、姿态对研究问题没有明显影响，就可把物体简化为有质量的点：




> **公式首次使用卡**：适用边界——物体大小和形状相对研究范围可忽略；不研究物体自身转动、姿态、形变或各部分运动差异；只关心整体位置、位移、速度、加速度等整体运动量。符号含义在紧邻公式的正文、图注或表格中定义；除题目另有单位外，代入前统一为 SI，结果保留单位并做量纲检查。

```text
质点 = 有质量、无大小形状的理想化模型
```

它不是说物体真的没有大小，而是说在当前问题中，大小和形状不影响主要结论。

**概念辨析。**

能否看作质点取决于“研究什么”，而不是取决于“物体大不大”。地球研究公转时可看作质点，乒乓球研究旋转时不能看作质点。

#### 公式、变量、单位与条件

| 判断问题          | 交互量或情境对应       | 结论                     |
| ----------------- | ---------------------- | ------------------------ |
| `l/s` 很小      | 物体长度与运动距离读数 | 研究行程位置时可看作质点 |
| 研究过线/过桥     | 前端、后端、停止线     | 不能看作质点             |
| 研究转动          | 车轮旋转箭头           | 不能看作质点             |
| 研究姿态          | 车身方向变化           | 不能看作质点             |
| 研究形状/受力分布 | 物体各部分不同         | 不能看作质点             |

**适用边界。**

- 物体大小和形状相对研究范围可忽略。
- 不研究物体自身转动、姿态、形变或各部分运动差异。
- 只关心整体位置、位移、速度、加速度等整体运动量。

#### 应用与迁移

- 判断题：某物体在某研究问题中能否看作质点。
- 建模题：说明忽略大小和形状的理由。
- 运动学题：把运动员、汽车、卫星等简化为质点后列运动规律。

**解题路径。**

1. 先看题目研究目标：整体运动，还是转动/姿态/形状。
2. 若研究整体位置，再比较物体尺寸与运动距离是否很小。
3. 若研究转动、姿态、通过某线的先后、受力分布，通常不能看作质点。
4. 最后用一句话说明“忽略什么，为什么能忽略”。

#### 边界检查与易错点

| 易错说法                         | 为什么错                         | 改法             |
| -------------------------------- | -------------------------------- | ---------------- |
| “小物体一定是质点”             | 乒乓球旋转时不能忽略形状         | 看研究目标       |
| “大物体一定不能看作质点”       | 地球公转时大小相对轨道很小       | 看尺寸与范围比   |
| “质点就是没有质量的点”         | 质点保留质量，只忽略大小形状     | 说“有质量的点” |
| “同一物体能否看作质点固定不变” | 同一汽车研究行程可，研究车轮不可 | 一题一判         |

#### 规律、证据与核心概念

参考系是描述物体运动时选作“静止”的物体或坐标系。速度、位移、轨迹都要相对某个参考系来描述。

**这条公式不是背下来的，是从"位移"推出来的**（动画顶部就是在演这一步）：

在同一段时间 $\Delta t$ 内——

- 物体相对地面走的位移：$x_{\text{物}} = v_{\text{物/地}} \cdot \Delta t$
- 参考系（如车 A）相对地面走的位移：$x_{\text{参}} = v_{\text{参/地}} \cdot \Delta t$
- 那么"物体相对参考系"走了多远，就是两者之差：$x_{\text{物/参}} = x_{\text{物}} − x_{\text{参}} = (v_{\text{物/地}} − v_{\text{参/地}})\cdot \Delta t$

两边同除以 $\Delta t$，就得到相对速度公式：

```text
v物/参 = v物/地 − v参/地
```

它的含义就是动画里那步减法：**先看物体对地怎么动，再减掉"参考系自己对地也在动"，剩下的才是站在参考系上看到的速度**。若以车 A 为参考系，代入 $v_{\text{参/地}} = vA$，则车 A 自己的速度算得 `0`；地面的速度算得 `−vA`（看起来向后退）。

**概念辨析。**

“静止”不是绝对的。一个物体可以相对某个参考系静止，同时相对另一个参考系运动。高中运动学默认常选地面为参考系，但题目若指定火车、船、水流、传送带为参考系，就要按题意改写速度。

#### 公式、变量、单位与条件

| 关系              | 图中对应         | 动画读数          |
| ----------------- | ---------------- | ----------------- |
| $v_{\text{乘客/地}}=vA$   | 乘客随车 A 运动  | 选地面时乘客速度  |
| $v_{\text{乘客}}/A=0$     | 乘客在车厢中不动 | 选车 A 时乘客速度 |
| $v_{\text{乘客}}/B=vA-vB$ | 两车相对运动     | 选车 B 时乘客速度 |
| $v_{\text{地}}/A=-vA$     | 地面相对车向后   | 选车 A 时地面速度 |

**适用边界。**

- 描述运动前必须选参考系。
- 高中常默认地面为参考系；但相对运动题要按题目指定参考系。
- 不同参考系下速度、位移、轨迹可不同，但同一物理过程不能混用多个参考系的数据。

#### 应用与迁移

- 概念题：判断“运动/静止”是否完整。
- 相对运动题：两车、船与水、雨滴与人等速度转换。
- 图像题：明确图像描述的是相对哪个参考系的位置或速度。

**解题路径。**

1. 先圈出题目指定的参考系。
2. 若未指定，通常默认地面。
3. 写清每个速度的下标：物体相对谁。
4. 用 $v_{\text{物/参}}=v_{\text{物/地}}-v_{\text{参/地}}$ 做转换。
5. 不在同一参考系下的速度、位移不能直接相加减。

#### 边界检查与易错点

| 易错说法                         | 为什么错               | 改法               |
| -------------------------------- | ---------------------- | ------------------ |
| “乘客是静止的”                 | 没说相对车还是地面     | 写“相对车厢静止” |
| “运动是绝对的”                 | 运动描述依赖参考系     | 先选参考系         |
| “相对速度就是两个速度直接相加” | 要先规定正方向和参考系 | 用带下标公式       |
| “一道题里随便换参考系”         | 混用数据会出错         | 全程统一或明确转换 |

**交互探究。**

可直接操作：拖动画面中的红色 $t_{1}/t_{2}$ 时刻游标，改变时间轴上的两个点；拖动小车旁的橙色速度端点，改变匀速 `v`。点**播放**：紫色播放头（当前时刻，一个移动的点）扫过时间轴，下面小车匀速前进，进入区间后**绿色位移条一段段增长**，长度 = $v\cdot (t−t_{1})$，走到 $t_{2}$ 时正好 = $x=v\Delta t$——**位移是"走"出来的，用的是时间间隔不是时刻读数**。红色虚标演示"错用 $t_{2}$ 当时长"会多走多少，把易错点直接暴露在画面上。

- 保持 $t_{1}=2\ \mathrm{s}$、$t_{2}=5\ \mathrm{s}$：确认 $\Delta t=3\ \mathrm{s}$，不是 `5s`。
- 拖动小车旁的速度端点改变 `v`：看正确末位移 $v\Delta t$ 和错用 $vt_{2}$ 的红色虚线怎样同时变化。

#### 规律、证据与核心概念

时间轴像数轴。时刻是时间轴上的一个点；时间间隔是两个点之间的长度：

```text
Δt = t2 - t1
```

运动公式里的 `t` 通常表示“从开始计时到当前的时间间隔”。若题目给出两个时刻，必须先相减。

**概念辨析。**

“第 3 秒末”是 $t=3\ \mathrm{s}$ 这个时刻；“第 3 秒内”是从 $t=2\ \mathrm{s}$ 到 $t=3\ \mathrm{s}$ 的 `1s` 时间间隔；“前 3 秒”是从 $t=0$ 到 $t=3\ \mathrm{s}$ 的 `3s` 时间间隔。

#### 公式、变量、单位与条件

| 表述          | 时间轴对应             | 用法         |
| ------------- | ---------------------- | ------------ |
| $t=2\ \mathrm{s}$      | 一个红点               | 时刻         |
| $t=5\ \mathrm{s}$      | 一个红点               | 时刻         |
| $\Delta t=t_{2}-t_{1}$ | 两红点之间的橙色线段   | 时间间隔     |
| $x=v\Delta t$    | 过程线段对应的运动时间 | 计算过程位移 |

**适用边界。**

- 任何过程计算都要使用时间间隔。
- 题目给“某秒末”通常是时刻；给“某秒内、前几秒、从几秒到几秒”通常是时间间隔。
- 若计时起点不是 `0`，要先求 $\Delta t$。

#### 应用与迁移

- 概念题：判断时刻和时间间隔。
- 计算题：由两个时刻求过程时间。
- 图像题：读某一时刻的纵坐标，或读两时刻之间的变化量。

**解题路径。**

1. 看到“末、初、时”先判断是否为时刻。
2. 看到“内、经历、用时、从……到……”先判断为时间间隔。
3. 若要代入运动公式，先写 $\Delta t=t_{2}-t_{1}$。
4. 再把 $\Delta t$ 代入位移、速度或加速度公式。

#### 边界检查与易错点

| 易错说法                    | 为什么错                 | 改法            |
| --------------------------- | ------------------------ | --------------- |
| “第 3 秒末是 3 秒时间”    | 第 3 秒末是时刻 $t=3\ \mathrm{s}$ | 说“3s 这个点” |
| “第 3 秒内是 3s”          | 第 3 秒内是 `2s~3s`    | 时间间隔 `1s` |
| “从 2s 到 5s 用 5s”       | 过程时间是差值           | $\Delta t=5-2=3\ \mathrm{s}$  |
| “公式里的 t 总是秒表读数” | 公式用过程时间           | 先找计时起点    |

#### 情境与现象

从教室门口走到讲台，可以直走，也可以绕过课桌再到讲台。两种走法的起点和终点相同，但走过的路线长短不同。物理上把“实际走过轨迹的长度”叫路程，把“从起点指向终点的位置变化”叫位移。

> **公式首次使用卡**：适用边界——只要研究物体位置变化，就可以用位移；直线运动中，若沿正方向走，位移为正；沿反方向走，位移为负；曲线运动中，路程沿轨迹算，位移用起点到终点的直线箭头表示。符号含义在紧邻公式的正文、图注或表格中定义；除题目另有单位外，代入前统一为 SI，结果保留单位并做量纲检查。

**证据任务。**

- 只拖蓝色“绕路点”：看蓝色轨迹变长，红色位移箭头是否改变。
- 拖终点 B：看红色箭头的长度、方向以及 $\Delta x/\Delta y$ 怎样改变。
- 点击“拉直路线”：比较路程 `s` 和位移大小 $\vert \Delta r\vert$ 是否接近相等。
- 点击“回到起点”：看路程可以不为 0，而位移为什么变为 0。

#### 规律、证据与核心概念

描述运动不能只问“走了多远”，还要问“位置怎样变了”。路程来自实际轨迹的长度累加，是标量；位移来自位置矢量的变化，是矢量：

```text
Δr = r2 - r1
```

若在平面直角坐标系中，从 A 到 B 的坐标变化为 $\Delta x$、$\Delta y$，位移大小为：

```text
|Δr| = √(Δx² + Δy²)
```

**概念辨析。**

- 路程回答“实际走过多少路”，没有方向，永远不为负。
- 位移回答“从哪里到哪里”，有大小和方向。
- 同一起点和终点可以有很多条路线，路程可以不同，位移相同。
- 绕操场跑一圈回到起点：路程是一圈长度，位移为零。

#### 公式、变量、单位与条件

| 量      | 图中对应       | 含义               |
| ------- | -------------- | ------------------ |
| `s`   | 蓝色曲线长度   | 实际轨迹累加，标量 |
| $\Delta r$ | 红色 A→B 箭头 | 位置变化，矢量     |
| $\Delta x$ | 水平方向虚线段 | 位移的水平分量     |
| $\Delta y$ | 竖直方向虚线段 | 位移的竖直分量     |
| $\vert \Delta r \vert =\sqrt{\Delta x^{2}+\Delta y^{2}}$  |

**适用边界。**

- 只要研究物体位置变化，就可以用位移。
- 直线运动中，若沿正方向走，位移为正；沿反方向走，位移为负。
- 曲线运动中，路程沿轨迹算，位移用起点到终点的直线箭头表示。
- 只有单向直线运动时，路程才等于位移大小；往返或绕路时通常不相等。

#### 应用与迁移

- 概念题：判断路程、位移、位移大小。
- 图像题：由坐标变化求位移，或判断回到起点时位移。
- 平均速度题：平均速度用位移除以时间，平均速率用路程除以时间。

**解题路径。**

1. 先找起点 A 和终点 B。
2. 问“路程”就沿实际轨迹累加。
3. 问“位移”就从 A 画到 B，并说明方向。
4. 若给坐标，先求 $\Delta x$、$\Delta y$，再求 $\vert \Delta r\vert$。
5. 遇到往返运动，特别检查位移是否可能为零。

#### 边界检查与易错点

| 易错说法                 | 为什么错                               | 改法                         |
| ------------------------ | -------------------------------------- | ---------------------------- |
| “路程就是位移”         | 路程是轨迹长度，位移是起点到终点的箭头 | 先画 A、B，再画轨迹          |
| “位移没有方向”         | 位移是矢量                             | 必须说明从 A 指向 B 或正负号 |
| “绕一圈位移等于一圈长” | 回到起点时位置没有变化                 | 位移为 0，路程为一圈长       |
| “曲线运动不能求位移”   | 位移只看初末位置                       | 曲线运动仍可画 A→B 箭头     |

- 编号：B1-05
- 模块：必修1

#### 情境与现象

同一辆小车在前半程缓慢、后半程加速。全程平均速度只能回答“这段路整体怎样”，却不能回答“第 3 秒末速度多大”。仪表盘上的测速仪只显示此刻的读数：为什么两个答案都叫速度，却回答不同的问题？

#### 默认引导演示（约 60 秒）

1. **完整播放**：小车先慢后快，运动条同步标出各段位移和用时；先只显示全程平均速度。

> **公式首次使用卡**：适用边界——平均速度适用于任何运动，定义 v̄ = Δx/Δt；瞬时速度要求 Δt→0 的极限，实验中用"足够短"的时间近似；矢量性：必须带方向（正负号），直线运动中沿正方向为正、反方向为负。符号含义在紧邻公式的正文、图注或表格中定义；除题目另有单位外，代入前统一为 SI，结果保留单位并做量纲检查。

**证据任务。**

- 拖动橙色 `Q` 点靠近紫色 `P` 点，看橙色割线和蓝色切线是不是越来越重合？差值是不是趋近 0？
- 拖动紫色 `P` 点到不同位置，看切线斜率（=瞬时速度）怎么变？加速运动中斜率随 t 增大还是减小？
- 把 a 拖到 0（匀速），看割线和切线是否始终重合？此时平均速度=瞬时速度吗？

#### 规律、证据与核心概念

平均速度 $\bar v=\Delta x/\Delta t$ 把一段过程的总位移均摊到总时间，因此会抹平“前慢后快”的细节，不能回答某一时刻的速度。要描述第 3 秒末，就把测量窗口限制在这一时刻附近；窗口越短，期间速度变化越小，短时平均速度越能代表该时刻。理想地令 $\Delta t\to0$，得到瞬时速度。在 x-t 图上，这对应割线趋近切线；微积分表达作为拓展。

**概念辨析。**

- **平均速度**是矢量：v̄ = Δx/Δt，方向与位移 Δx 相同，大小反映这段时间整体快慢。
- **瞬时速度**也是矢量：v = lim(Δt→0) Δx/Δt，方向沿运动轨迹切线方向，大小就是速率计读数。
- **平均速率** = 路程/时间，是标量，不一定等于平均速度的大小（如绕操场跑一圈，位移为零→平均速度为零，但平均速率不为零）。
- 实验中用"极短时间"近似测瞬时速度：光电门用挡光时间 Δt 极短，v ≈ Δx/Δt。

#### 公式、变量、单位与条件

- v̄ = Δx/Δt = (x₂ - x₁)/(t₂ - t₁) → 图中割线斜率
- v = lim(Δt→0) Δx/Δt → 图中切线斜率
- x-t 图：直线运动 x-t 图斜率 = 瞬时速度；匀速为直线（斜率恒定），变速为曲线（斜率变化）
- 匀变速：v = v0 + at（瞬时速度随时间线性变化）

**适用边界。**

- 平均速度适用于任何运动，定义 v̄ = Δx/Δt。
- 瞬时速度要求 Δt→0 的极限，实验中用"足够短"的时间近似。
- 矢量性：必须带方向（正负号），直线运动中沿正方向为正、反方向为负。
- 注意区分"平均速度"（矢量，用位移）和"平均速率"（标量，用路程）。

#### 高考迁移（少量验证）

- **快速预测**：前慢后快的运动中，全程平均速度能否等于第 3 秒末速度？不能仅凭“都叫速度”判断。
- **错误辨析**：“挡光片越窄，测瞬时速度一定越准。”近似区间会变短，但挡光时间也变短，计时相对误差可能增大，必须权衡。
- **迁移任务**：给出同一运动的分段数据、测速仪读数和 x-t 图，分别指出平均速度、平均速率和瞬时速度。高考常把三个量放在同一情境中考查研究对象和时间范围。

**解题路径。**

1. 区分题目问"平均"还是"瞬时"：平均用 v̄=Δx/Δt，瞬时用某时刻速度公式或 x-t 图切线斜率。
2. 实验测瞬时速度：用短时间内的平均速度近似。挡光片变窄可减小平均速度近似误差，但也会缩短挡光时间、放大计时相对误差，应在两者之间权衡。
3. x-t 图问题：斜率 = 瞬时速度，直线斜率恒定=匀速，曲线斜率变化=变速。
4. 注意矢量性：平均速度方向与位移同向，不是"运动方向"。

#### 边界检查与易错点

- **把平均速度当瞬时速度**：用整段时间的 v̄ 当作某时刻的 v（只有匀速才相等）。
- **混淆平均速度与平均速率**：平均速度用位移（矢量），平均速率用路程（标量），绕一圈平均速度为零但速率不为零。
- **忽略方向**：平均速度是矢量，回程会使位移减小，平均速度可能为零。
- **认为瞬时速度必须用公式算**：x-t 图切线斜率就是瞬时速度，不一定要套 v=v0+at。

#### 情境与现象

两辆车在相同的 `2 s` 内速度变化不同：一辆从 `2 m/s` 变到 `6 m/s`，另一辆从 `10 m/s` 变到 `12 m/s`。哪辆“速度改变得更快”？只比较末速度无法回答，还必须比较速度的变化量和完成这次变化所用的时间。

> **公式首次使用卡**：$a=\Delta v/\Delta t$ 表示一段时间内的平均加速度；加速度方向与 $\Delta v$ 相同。直线运动中的正负号只表示方向，加速或减速要比较 $v$ 与 $a$ 的方向。瞬时加速度是时间间隔趋近于零的极限；匀变速运动中平均加速度与瞬时加速度相同。

#### 默认引导演示（约 70 秒）

1. **先看变化**：固定 `2 s`，并排播放两辆车，显示 $\vec v_1$、$\vec v_2$ 和 $\Delta\vec v$；学生先预测谁的速度改变更快。
2. **四幕连播**：依次展示 $v>0,a>0$、$v>0,a<0$、$v<0,a<0$、$v<0,a>0$，始终保留正方向、速度箭头与 $\Delta v$ 箭头。
3. **关键暂停**：在 $v<0,a<0$ 时只问“加速度为负，物体是在加速还是减速？”揭示判断依据是 $v$ 与 $a$ 同号，而不是 `a` 的正负。
4. **变量对比**：保持 $\Delta v$ 不变，把 $\Delta t$ 缩短一半，观察加速度大小加倍；最后才写 $a=\Delta v/\Delta t$。

> **一句话抓手**：加速度表示“速度每秒改变多少”；正负号表示方向，不直接表示加速或减速。

- 点击 $v=0,a\ne 0$：看速度为零时，为什么 $\Delta v/\Delta t$ 仍然不为零。
- 点击“刹车减速”：看 `a` 与当前速度反向时，物体为什么是减速。

#### 规律、证据与核心概念

加速度定义为速度变化率 a = Δv/Δt。在 v-t 图上，Δv/Δt 就是直线的斜率——所以**v-t 图斜率 = 加速度**。匀变速运动中 a 恒定，v-t 图是直线。加速度是矢量，方向与 Δv 相同，不一定与 v 相同：a 与 v 同向则速度增大（加速），反向则速度减小（减速）。

**概念辨析。**

- **加速度不是速度**：a 大不代表 v 大（刚启动时 v=0 但 a 很大），v 大不代表 a 大（高速匀速时 a=0）。
- **加速度方向**：与 Δv 方向相同，与 v 方向可以相同（加速）也可以相反（减速）。
- **v=0 时 a 可不为零**：速度为零只是"这一刻没动"，加速度描述"速度正在怎么变"，下一刻就会有速度。典型例子：竖直上抛最高点 v=0、a=g。
- **a 的正负**：a 为正只代表加速度沿正方向，不代表"加速"；是否加速要看 a 与 v 是否同向。

#### 公式、变量、单位与条件

- a = Δv/Δt = (v - v0)/t → 图中 v-t 图斜率
- v = v0 + at → 图中 v-t 直线
- a-t 图：匀变速为水平线（a 恒定）
- 判断加速/减速：a·v > 0 加速，a·v < 0 减速（不是看 a 的正负）

**适用边界。**

- a = Δv/Δt 定义平均加速度；瞬时加速度 a = lim(Δt→0) Δv/Δt。
- 匀变速运动 a 恒定，v-t 图为直线。
- 加速度是矢量，直线运动中用正负号表示方向。
- 高考中"加速度"通常指瞬时加速度，匀变速时等于平均加速度。

#### 高考迁移（少量验证）

- **快速预测**：$v=-4\ \mathrm{m/s}$、$a=-2\ \mathrm{m/s^2}$ 时，速度大小在增大，物体沿负方向加速。
- **错误辨析**：“加速度为负，所以物体一定减速。”错；负号只说明加速度沿所选负方向。
- **迁移任务**：从 v-t 图某点的纵坐标和斜率同时判断运动方向与快慢变化。高考常用“图线在横轴下方但斜率为负”制造把方向和加速减速混为一谈的陷阱。

**解题路径。**

1. 已知 v0、v、t，直接用 a = (v - v0)/t；
2. 已知 v-t 图，求斜率即得 a（注意单位）；
3. 判断加速/减速：看 a 与 v 同向还是反向，不是看 a 正负；
4. 与力结合：F合 = ma，先受力分析求合力，再除以质量得 a。

#### 边界检查与易错点

- **认为速度大加速度就大**：高速匀速 a=0；刚启动 v=0 但 a 很大。
- **认为速度为零加速度就为零**：上抛最高点 v=0、a=g；刹车刚停瞬间 v=0、a 仍为刹车加速度。
- **用 a 的正负判断加速减速**：a>0 只代表沿正方向，是否加速要看 a 与 v 是否同向。
- **混淆加速度与速度方向**：减速时 a 与 v 反向，不是 a 变小。

#### 规律、证据与核心概念

x-t 图的每个点都表示一个时刻的位置。若取两点 $(t_{1},x_{1})$、$(t_{2},x_{2})$，连线斜率为：

```text
k = Δx/Δt = (x2-x1)/(t2-t1)
```

位移除以时间就是平均速度。若两点无限接近，割线变成切线，切线斜率就是该时刻的瞬时速度。

**概念辨析。**

- 图上一个点：表示“某时刻物体在什么位置”。
- 图线升高：位置坐标增大，不等于物体往上爬。
- 图线水平：位置不变，物体静止。
- 斜率为正：沿正方向运动；斜率为负：沿负方向运动；斜率为零：速度为零。
- 曲线越来越陡：速度大小在增大；曲线越来越平：速度大小在减小。

#### 公式、变量、单位与条件

| 图像元素             | 物理含义 | 公式或判断        |
| -------------------- | -------- | ----------------- |
| 横坐标 `t`         | 时刻     | 读某一时刻        |
| 纵坐标 `x`         | 位置坐标 | 读该时刻位置      |
| 两点纵坐标差 $\Delta x$ | 位移     | $\Delta x=x_{2}-x_{1}$     |
| 两点横坐标差 $\Delta t$ | 时间间隔 | $\Delta t=t_{2}-t_{1}$     |
| 割线斜率             | 平均速度 | $v_{\text{平均}}=\Delta x/\Delta t$ |
| 切线斜率             | 瞬时速度 | $v=dx/dt$       |

**适用边界。**

- x-t 图用于描述直线运动或选定坐标轴上的位置变化。
- 斜率表示速度，不能用图线长短表示路程。
- 图像在时间轴上回头通常表示“位置开始反向变化”，不是物体沿图线折返。
- 坐标轴单位必须看清；不同题的比例尺可能不同。

#### 应用与迁移

- 读数题：由某时刻读位置，或由两个时刻求位移。
- 斜率题：比较速度大小和方向。
- 运动状态题：判断静止、匀速、加速、减速、反向。
- 追及题：两条 x-t 图线交点表示同一时刻同一位置。

**解题路径。**

1. 先看坐标轴：横轴是不是 `t`，纵轴是不是 `x`。
2. 读点：确定某时刻的位置。
3. 看斜率：判断速度大小和方向。
4. 看交点：判断是否相遇。
5. 遇到曲线：比较切线斜率，不用图线弯曲方向当运动轨迹。

#### 边界检查与易错点

| 易错说法                     | 为什么错                 | 改法                   |
| ---------------------------- | ------------------------ | ---------------------- |
| “x-t 图线就是运动轨迹”     | 横轴是时间，不是空间坐标 | 图线表示位置随时间变化 |
| “图线向上就是物体向上运动” | 纵轴是位置坐标 x         | 应说沿正方向运动       |
| “斜率越大加速度越大”       | x-t 斜率是速度           | 加速度看斜率变化快慢   |
| “交点表示速度相同”         | 交点表示同一时刻同一位置 | 速度要比较斜率         |

#### 情境与现象

> **公式首次使用卡**：v-t 图纵坐标是选定正方向下的速度；斜率 $\Delta v/\Delta t$ 表示加速度；很短时间内的位移近似为 $v\Delta t$，累加得到有符号面积。横轴上方面积为正位移、下方为负位移；路程需把各段面积取绝对值后累加。

可直接操作：拖 v 轴上的 $v_{0}$ 点改变纵截距；拖 `6 s` 端点改变直线斜率；拖橙色 `t` 游标改变面积边界。实时变化：小车速度箭头、v-t 直线斜率、正负面积填充、位移读数、路程读数。临界提示：$a=0$ 时为匀速（面积=矩形）；图线穿过时间轴时，位移和路程不再相等。

**证据任务。**

- 拖动 $v_{0}$ 点，看图线在纵轴上的截距怎样决定初速度。
- 拖动右端点，看直线斜率怎么变；斜率越陡，加速度大小越大。
- 拖动橙色 `t` 游标，看橙色/蓝色面积如何变成位移；让图线穿过时间轴，比较“位移”和“路程”的读数差别。

#### 规律、证据与核心概念

相邻时刻的速度变化率是 $\Delta v/\Delta t$，它既是图线斜率，也是这段时间的平均加速度。很短时间 $\Delta t$ 内，速度近似不变，位移约为 $v\Delta t$；把所有小段相加，就得到 v-t 图与时间轴之间的有符号面积。横轴下方 $v<0$，对应沿负方向的位移，因此求路程要累加各段面积的绝对值。

**概念辨析。**

v-t 图像不是物体的运动路径。纵轴是速度大小，横轴是时间。看 v-t 图要养成"三看"习惯：一看纵截距（初速度），二看斜率（加速度），三看面积（位移）。

#### 公式、变量、单位与条件

- 纵截距 = v₀（初速度）→ 图中直线与 v 轴交点
- 斜率 = a = Δv/Δt → 图中直线的倾斜程度
- 面积 = x = v₀t + ½at² → 图中直线下方三角形/梯形面积
- 某时刻速度 v = v₀ + at → 图中 t 处的纵坐标

**适用边界。**

- v-t 图像适用于任何直线运动（匀速、匀变速、非匀变速均可）。
- 匀变速直线运动的 v-t 图像是直线；非匀变速是曲线。
- 面积在横轴上方为正位移，下方为负位移。

#### 高考迁移（少量验证）

- **快速预测**：图线在横轴下方且斜率为正时，物体沿负方向运动但速度大小可能减小。
- **错误辨析**：“图线与时间轴围成的几何面积就是路程。”错；横轴下方给负位移，路程要分段取绝对值。
- **迁移任务**：读取本页演示中越过时间轴的 v-t 图线，分别求位移和路程，并解释两者为何不同。高考常把“方向改变”和“加速度改变”包装在同一幅图像中。

**解题路径。**

1. 识别图像类型（确认横纵轴物理量）；
2. 看截距 → 初速度；
3. 看斜率 → 加速度（注意正负）；
4. 看面积 → 位移（注意正负，路程取绝对值面积）；
5. 看交点 → 两物体速度相等的时刻。

#### 边界检查与易错点

- 把 v-t 图线长度当成位移（位移是面积，不是线长）。
- 把 v-t 图线当成运动轨迹。
- 忽略横轴下方面积为负位移，求路程时未取绝对值。
- 混淆斜率正负与运动方向（斜率正只代表加速度沿正方向，不代表物体一定沿正方向运动）。

- 拖动蓝色小车：看频闪点间距、x-t 图斜率、v-t 图高度是否同时改变。

- 拖动绿色起点旗：区分位置 $x=x_{0}+vt$ 和位移 $\Delta x=vt$；打开“对比匀加速”，比较蓝点等间距和橙点间距递增的区别。

#### 规律、证据与核心概念

匀速直线运动的核心是速度恒定：

```text
v = Δx/Δt = 常量
```

所以任意时间 `t` 内的位移与时间成正比。若从原点出发：

```text
x = vt
```

若初位置不是原点：

```text
x = x0 + vt
```

**概念辨析。**

- “匀速”不是“速度大小差不多”，而是速度大小和方向都不变。
- “直线”说明运动方向不改变，轨迹是一条直线。
- 速度为正表示沿正方向运动；速度为负表示沿负方向运动；速度为零是静止。
- 匀速运动的加速度为零，因为速度没有变化。

#### 公式、变量、单位与条件

| 画面或图像       | 对应量  | 公式含义                     |
| ---------------- | ------- | ---------------------------- |
| 频闪相邻点距离   | $\Delta x$ | 相同 $\Delta t$ 内 $\Delta x$ 相等 |
| 小车位置读数     | `x`   | 从原点出发时 $x=vt$        |
| x-t 图直线斜率   | `v`   | $v=\Delta x/\Delta t$                |
| v-t 图水平线高度 | `v`   | 速度恒定                     |
| v-t 图斜率       | `a`   | 匀速时 $a=0$               |

**适用边界。**

- 物体做直线运动。
- 速度保持恒定，包括大小和方向都不变。
- $x=vt$ 默认初位置 $x_{0}=0$；若不从原点出发，应写 $x=x_{0}+vt$。
- 代入时要先选正方向，`v` 和 `x` 可以带正负号。

#### 应用与迁移

- 计算题：已知 `v、t` 求位移，或已知位移和时间求速度。
- 图像题：判断 x-t 图直线斜率、v-t 图水平线。
- 追及题：两物体匀速运动时，用 $x=x_{0}+vt$ 列方程。
- 概念题：区分匀速、静止、匀加速。

**解题路径。**

1. 判断是否速度恒定，方向是否不变。
2. 选定正方向和计时起点。
3. 若从原点出发，用 $x=vt$；若有初位置，用 $x=x_{0}+vt$。
4. 图像题中，x-t 看斜率，v-t 看水平线高度。
5. 检查单位和正负号，不把路程、位移、位置混用。

#### 边界检查与易错点

| 易错说法                     | 为什么错                   | 改法                       |
| ---------------------------- | -------------------------- | -------------------------- |
| “匀速就是速率不变”         | 方向变了就不是匀速直线运动 | 速度大小和方向都要不变     |
| “x=vt 任何时候都能用”      | 初位置不为零时少了 $x_{0}$  | 写 $x=x_{0}+vt$             |
| “v-t 图水平线表示位置不变” | v-t 纵轴是速度             | 水平线表示速度恒定         |
| “x-t 图水平线表示匀速”     | x 不变，斜率为零           | 这是静止，不是有速度的匀速 |

- 拖动 `6 s` 端点让 `a` 从正变负，看频闪点间距从变大变成变小；注意相邻相等时间位移差 $\Delta s=aT^{2}$ 是否恒定。

#### 规律、证据与核心概念

从"速度均匀变化"出发：v = v0 + at（速度随时间线性变化）。位移是 v-t 图像面积：梯形面积 x = ½(v0+v)t = v0t + ½at²。消去 t 得 v² - v0² = 2ax。

**概念辨析。**

匀变速 = 加速度恒定。三个公式各有用途：知道时间用 v=v0+at 和 x=v0t+½at²；不知道时间用 v²-v0²=2ax。选公式要看已知什么、求什么。

#### 公式、变量、单位与条件

- v = v0 + at → v-t 图纵坐标
- x = v0t + ½at² → v-t 图梯形面积
- v² - v0² = 2ax → 不含时间的导出公式
- a = (v-v0)/t → v-t 图斜率

**适用边界。**

- 加速度 a 恒定（大小和方向都不变）。
- 仅适用于直线运动。
- 刹车问题中 a 恒定但运动有终止条件（v=0 后停止）。
- 正方向选定后 a、v、x 带正负号代入。

#### 应用与迁移

- 选择题：判断运动性质、比较物理量。
- 实验题：打点计时器验证匀变速。
- 计算题：刹车距离、追及相遇、自由落体、平抛分运动。

**解题路径。**

1. 选正方向，确定 v0、a 的正负；
2. 判断是否匀变速（a 是否恒定）；
3. 已知什么、求什么 → 选公式（含 t 用前两式，不含 t 用第三式）；
4. 代入求解，检查单位 and 正负号；
5. 刹车问题先求停止时间，判断是否超时。

#### 边界检查与易错点

- 把 ½at² 写成 at²（漏掉 ½）。
- 初速度不为零时漏掉 v0t 项。
- 刹车问题把停下后的时间继续代入位移公式。
- 不选正方向导致 a、x 正负号混乱。

#### 规律、证据与核心概念

刹车是匀减速直线运动的特例：初速度 v0 > 0，加速度 a < 0（与 v 反向）。关键在于**物理约束**：物体速度减到 0 后不会反向运动（车不会倒退），所以 v = v0 - |a|·t 只在 t ≤ t 停 时成立，之后 v = 0、x = x 停。数学上 v0·t - ½|a|·t² 在 t > t 停 后会减小甚至变负，但物理上位移单调不减。

**概念辨析。**

"刹车问题"的本质是**有界匀变速**：公式有适用区间。判断步骤：①先算 t 停 = v0/|a|；②比较题目时间 t 与 t 停；③若 t ≤ t 停，正常代公式；④若 t > t 停，直接用 x = v0²/(2|a|)（因为 t 停时已停）。这一步判断是高考失分重灾区。

#### 公式、变量、单位与条件

- v = v0 - |a|·t → 图中 v-t 直线，仅 t ≤ t 停 有效
- x = v0·t - ½|a|·t² → 图中 v-t 图面积（三角形），仅 t ≤ t 停
- t 停 = v0/|a| → 图中 v-t 直线与 t 轴交点
- x 停 = v0²/(2|a|) → 图中 v-t 图三角形面积，t > t 停 后用此式
- 注意：t > t 停 后 v ≡ 0、x ≡ x 停，不能再代含 t 的公式

**适用边界。**

- 初速度 v0 > 0，加速度与速度反向（减速）。
- 物体速度减到 0 后停止，不反向（区别于"竖直上抛"，上抛 v=0 后会反向下落）。
- 忽略空气阻力，加速度恒定。
- 题目给的时间可能超过 t 停，必须先判断。

#### 应用与迁移

- 选择题：求刹车距离、刹车时间、某时刻速度（注意 t 与 t 停 比较）。
- 计算题：作为综合题第一问（求反应时间+刹车距离=总停车距离）。
- 实验题：测刹车加速度（用频闪或传感器）。

**解题路径。**

1. 先算 t 停 = v0/|a|；
2. 比较题目时间 t 与 t 停：
   - t ≤ t 停：用 v = v0 - |a|t、x = v0t - ½|a|t²；
   - t > t 停：v = 0、x = v0²/(2|a|)；
3. 若求"刹车距离"直接用 x = v0²/(2|a|)，不用求时间；
4. 反应距离 = v0·t 反应，总停车距离 = 反应距离 + 刹车距离。

#### 边界检查与易错点

- **不判断 t 停，直接代公式**（最常见错误）：t > t 停 后得到位移减小甚至变负的荒唐结果。
- 把加速度写成正值代公式（应取负，或用 |a| 配减号）。
- 混淆"刹车距离"x 停 和"某时间内位移"x（前者是最大值，后者依赖 t）。
- 忽略反应时间（实际停车距离 = 反应距离 + 刹车距离）。

#### 规律、证据与核心概念

追及相遇的本质是**位移关系**：两物体位置相等即相遇。设后车 x1、前车 x2 = d + v2·t（前车在前 d 米处匀速），相遇条件 x1 = x2。距离 Δx = x2 - x1 是时间的二次函数（后车匀加速时），其极值在 dΔx/dt = v2 - v1 = 0 处，即**两车速度相等时距离取极值**。但要先判断这是最远距离还是最近距离：后车先慢后快追前车时，速度相等处通常是**最远距离**，之后距离会减小，能否追上要看 $\Delta x=0$ 是否有正时刻解；避免追尾类问题里，速度相等常对应**最近距离**，最近距离大于 0 才说明安全。

**概念辨析。**

"速度相等时距离变化率为零"是追及问题的钥匙。直觉上：后车比前车慢时距离在拉大，后车比前车快时距离在缩小，那么"快慢转换点"（速度相等）就是距离的内部极值点。本页恒正加速度追匀速车的距离函数开口向下，共速处是最大距离；其他模型中它是最大值还是最小值，必须结合相对速度在该时刻前后的符号判断。

#### 公式、变量、单位与条件

- 后车：x1 = v1₀·t + ½a·t² → 图中蓝车位置
- 前车：x2 = d + v2·t → 图中红车位置
- 相遇条件：x1 = x2，即 v1₀·t + ½a·t² = d + v2·t → 解二次方程
- 距离：Δx = x2 - x1 = d + (v2 - v1₀)·t - ½a·t² → 图中 Δx-t 抛物线
- 极值条件：dΔx/dt = v2 - v1 = 0 → t等 = (v2 - v1₀)/a → 图中 v-t 两线交点

**适用边界。**

- 两物体在同一直线上运动（一维）。
- 本页主模型是后车以正的恒加速度持续追赶、前车匀速，时间和道路不设上限；在此条件下后车最终一定追上。
- 可推广到后车匀减速追前车、两车都匀加速等情形，方法相同。
- 相遇条件始终是位置相等；内部距离极值的必要条件是速度相等，极值类型要由相对速度变号方向判断。

#### 应用与迁移

- 计算题：求追上时间、追上前最远距离、避免追尾的最小加速度。
- 选择题：判断能否追上、距离极值时刻、共速时刻。
- 综合题：与刹车（前车急刹）、反应时间结合。

**解题路径。**

1. 写出两车位移表达式 x1(t)、x2(t)（注意参考点和正方向一致）；
2. 相遇问题：令 x1 = x2，解关于 t 的方程并筛选符合运动阶段的正根；
3. 距离极值问题：令 v1 = v2 求 t等，代入 Δx = x2 - x1 得极值；
4. 判断能否追上：看 $\Delta x=0$ 是否有正时刻解；若是避免追尾，再看速度相等时的最近距离是否大于 0；
5. 避免追尾：求恰不相撞的临界条件（Δx 极值 = 0，即速度相等且距离为零）。

#### 边界检查与易错点

- **忽略"速度相等时距离极值"**，只会列 x1=x2 解方程，遇到"最远距离"无从下手。
- 位移表达式参考点不统一（后车从原点、前车从 d 处，忘记加 d）。
- 把不同模型混用：本页恒正加速度、无限时空模型必能追上；有限加速、限速或有限道路模型必须分段判断。
- 混淆"相遇"（位置相同）和"共速"（速度相同），二者不是同一时刻（除非恰好追上）。

#### 规律、证据与核心概念

自由落体是匀变速直线运动的特例：初速度 v0=0，加速度 a=g≈9.8 m/s²（地球表面）。伽利略用斜面"冲淡重力"测出 s∝t²，再外推到竖直方向，得到落体规律。牛顿第二定律 F=mg 与 ma=mg 联立得 a=g，与质量 m 无关——这就是"质量不影响自由落体"的力学根源。

**概念辨析。**

"自由"二字指两个理想化条件：①初速度为零（只是释放，不是抛出）；②只受重力，无空气阻力。现实里空气阻力几乎总存在，所以严格意义的自由落体只在真空管里成立。高考题里"小球下落""物体从高处掉下"默认按自由落体处理，除非题给空气阻力。

#### 公式、变量、单位与条件

- v = gt → 图中 v-t 直线的纵坐标，斜率 = g

- v² = 2gh → 落地速度与高度的关系，不显含时间
- t = √(2h/g) → 落地总时间，读数面板实时显示
- 注意：公式里没有 m，质量不参与

**适用边界。**

- 初速度 v0 = 0（只是释放）。
- 只受重力，忽略空气阻力。
- 加速度 g 在地球表面附近近似恒定 9.8 m/s²；不同星球 g 不同。
- 高度变化远小于地球半径时 g 才恒定，否则要用万有引力变加速。

- 选择题：比较不同质量物体下落时间、落地速度。
- 实验题：频闪照片测 g（用 Δx = gT² 求加速度）。
- 计算题：作为平抛运动竖直分运动；与圆周运动"最高点脱手后"结合。

**解题路径。**

1. 确认是否满足自由落体条件（v0=0、无空气阻力）；
2. 识别已知量：h、t、v 中已知两个即可求第三个；
3. 选公式：含 v 用 v=gt 或 v²=2gh；不含 v 用 h=½gt²；求 t 用 t=√(2h/g)；
4. 多段运动时，自由落体常作为某一段（如平抛竖直分运动、圆周脱手后）。

#### 边界检查与易错点

- 认为重物落得快（错：a=g 与 m 无关）。
- 把"自由落体"和"竖直上抛"混淆（前者 v0=0，后者 v0≠0）。
- 忽略空气阻力，把雨滴、跳伞等真实下落当自由落体（这些是变加速或匀速）。
- 用 v²=2gh 时忘记开方，或把 h 当成 v。

- 看 v-t 图：t 轴上方正面积和下方负面积如何对应上升高度与回落位移？

#### 规律、证据与核心概念

竖直上抛是匀变速直线运动，初速度 v0 向上、加速度 a=-g 恒向下。两种处理法：①分段法，上升段按匀减速到 v=0、下落段按自由落体；②全程法，取向上为正，统一用 v=v0-gt、h=v0t-½gt²。第二种更简洁，且自动体现对称性。

**概念辨析。**

最高点是"速度方向转换点"，不是"加速度转换点"。速度在此刻为零，但加速度始终是 g 向下——否则小球会停在空中不动。对称性来自加速度恒定：上升和下落经过同一高度时速度大小相等、方向相反，t上 = t下 = v0/g。

#### 公式、变量、单位与条件

- v = v0 - gt → 图中 v-t 直线，纵截距 = v0，斜率 = -g
- h = v0t - ½gt² → 图中 v-t 图面积（t 轴上方为正位移）
- 最高点 v=0 → t上 = v0/g，图中直线与 t 轴交点
- Hmax = v0²/(2g) → 图中最高点虚线高度
- 落回抛点 h=0 → t总 = 2v0/g = 2t上，落地速度 = -v0

**适用边界。**

- 只受重力，忽略空气阻力。
- 加速度 g 恒定且方向始终向下（无论上升还是下落）。
- 全程法要求先规定正方向（通常取向上为正），公式中 v、h 带正负号。
- 抛出点为参考点；若从高处抛出落回地面，需分段或重新选参考点。

#### 应用与迁移

- 选择题：求最高点加速度、上升下落时间比、同一高度的速度。
- 计算题：与圆周运动"过最高点后做上抛"、与碰撞"反弹后做上抛"结合。
- 实验题：验证 v² 与 h 的关系（上抛频闪）。

**解题路径。**

1. 规定正方向（推荐向上为正）；
2. 识别已知量：v0、g、t、h、v 中已知两个即可求；
3. 全程法优先：v=v0-gt、h=v0t-½gt²；
4. 求最高点用 v=0 代入；求落回抛点用 h=0 代入；
5. 多段问题时，注意衔接点的速度方向和正负号。

#### 边界检查与易错点

- 认为最高点加速度为零（错：a=g 始终向下，只是 v=0）。
- 上升段写 a=+g、下落段写 a=-g（错：a 始终向下，正负由坐标方向决定，不由运动方向决定）。
- 忽略对称性，把上升和下落当成两个不同加速度的过程。
- 落回抛点时忘记速度方向相反，写成 +v0（应为 -v0）。

**证据任务。**

- 拖动质量旋钮 `m`：看 $G=mg$ 的读数和红色箭头长度是否随质量变大而变大。

- 拖动斜面端点改变 $\theta$：看重力箭头是否仍竖直向下，再观察 `G∥` 和 `G⊥` 怎么变。

#### 规律、证据与核心概念

重力是由于地球吸引而使物体受到的力。地面附近，同一地点的 `g` 近似恒定，因此重力大小与质量成正比：

```text
G = mg
```

其中 `m` 是物体质量，单位 `kg`；`g` 是重力加速度，常取 `9.8 N/kg` 或 `10 N/kg`；`G` 是重力，单位 `N`。

**概念辨析。**

- 质量是物体本身含有物质多少的量，不随地点改变。
- 重力是地球对物体的吸引力，随 `g` 改变而改变。
- 重心是重力的等效作用点；形状规则、密度均匀的物体，重心常在几何中心。
- 重力方向在地面附近近似竖直向下，受力分析时不要画成“垂直斜面向下”。

#### 公式、变量、单位与条件

| 量           | 图中对应             | 含义               |
| ------------ | -------------------- | ------------------ |
| `m`        | 参数面板的质量       | 决定物体重力大小   |
| `g`        | 参数面板的重力加速度 | 所在地点的引力强弱 |
| $G=mg$     | 红色重力箭头与读数   | 重力大小           |
| 重心         | 物体中心红点         | 重力作用点         |
| 竖直向下     | 红色箭头方向         | 重力方向           |
| `G∥、G⊥` | 斜面上的橙/蓝分力    | 后续力的分解使用   |

**适用边界。**

- $G=mg$ 适用于地面附近常见高中力学问题。
- 同一物体在不同星球或不同高度，`m` 不变，但 `g` 和 `G` 可能变。
- 方向“竖直向下”是地面附近的近似；大范围天体问题中万有引力方向指向吸引中心。
- 重心可以在物体外部，例如均匀圆环的重心在圆心。

#### 应用与迁移

- 受力分析题：先画重力，从重心竖直向下。
- 计算题：由质量求重力，或由重力求质量。
- 斜面题：先画重力，再按斜面方向分解。
- 概念题：区分质量、重力、重心和重力方向。

**解题路径。**

1. 找研究对象，标出重心。
2. 从重心画出竖直向下的重力箭头。
3. 用 $G=mg$ 求大小，注意 `g` 的取值。
4. 若在斜面上，再把重力分解为沿斜面和垂直斜面的分力。
5. 最后检查单位：质量用 `kg`，重力用 `N`。

#### 边界检查与易错点

| 易错说法                   | 为什么错                   | 改法                 |
| -------------------------- | -------------------------- | -------------------- |
| “质量就是重力”           | 质量单位 kg，重力单位 N    | 用 $G=mg$ 联系二者 |
| “斜面上重力垂直斜面向下” | 重力方向与斜面无关         | 始终画竖直向下       |
| “重心一定在物体内部”     | 圆环、弯曲物体重心可在外部 | 重心是等效作用点     |
| “月球上物体质量变小”     | 质量不随地点变             | 月球上是重力变小     |

#### 规律、证据与核心概念

弹力不是凭空出现的力。两个物体接触或连接时，如果发生弹性形变，形变的物体要恢复原状，就会对接触或连接的另一物体施加力，这个力叫弹力。

```text
弹力产生条件：接触或连接 + 弹性形变
弹力方向：指向恢复原状的方向
```

本节先掌握定性判断；弹簧弹力与形变量的定量关系 $F=kx$ 在 B1-17 胡克定律中专门学习。

**概念辨析。**

- 支持力是弹力：桌面被压弯，想恢复原状，于是向上推物体。
- 压力也是弹力：物体压桌面，桌面也受到物体的弹力。
- 绳的拉力是弹力：绳被拉紧，想缩短，拉力沿绳指向绳收缩方向。
- 弹簧弹力是弹力：伸长时拉回，压缩时推开。

#### 公式、变量、单位与条件

| 场景             | 画面对应                           | 方向判断                 |
| ---------------- | ---------------------------------- | ------------------------ |
| 桌面支持力 `N` | 桌面微小下陷，红箭头向上           | 垂直接触面，指向物体     |
| 压力             | 物体压桌面，蓝箭头向下             | 垂直接触面，指向被压物体 |
| 绳拉力 `T`     | 绳被拉紧                           | 沿绳，指向绳收缩方向     |
| 弹簧弹力         | 弹簧伸长或压缩                     | 沿弹簧，指向恢复原长     |
| 形变量 `x`     | 被拖动物体/绳端/弹簧端点和画面形变 | $x=0$ 时弹力为 0       |

**适用边界。**

- 必须有接触或连接；没有相互作用通道就没有接触弹力。
- 必须有弹性形变；有些形变很小但仍存在，例如桌面支持力。
- 支持力方向通常垂直接触面；绳的拉力沿绳；轻杆弹力可沿杆推或拉，需结合约束判断。
- 超过弹性限度后物体可能不能恢复原状，此时不能简单按弹性模型处理。

#### 应用与迁移

- 受力分析题：判断支持力、压力、绳拉力是否存在及方向。
- 平衡题：由受力平衡求支持力或拉力大小。
- 斜面题：支持力垂直斜面，不是竖直向上。
- 弹簧题：过渡到胡克定律 $F=kx$。

**解题路径。**

1. 先找物体与谁接触或连接。
2. 判断接触处或连接处是否发生弹性形变。
3. 想象形变若恢复原状，会把研究对象往哪个方向推或拉。
4. 画出弹力方向：面接触垂直接触面，绳沿绳，弹簧沿轴线。
5. 再结合平衡或牛顿第二定律求大小。

#### 边界检查与易错点

| 易错说法                   | 为什么错                   | 改法                 |
| -------------------------- | -------------------------- | -------------------- |
| “接触就一定有弹力”       | 接触但无挤压形变时可无弹力 | 看是否相互挤压或拉伸 |
| “支持力总是竖直向上”     | 支持力垂直接触面           | 斜面支持力垂直斜面   |
| “绳可以推物体”           | 轻绳只能拉不能推           | 绳力沿绳且指向绳收缩 |
| “桌面没变形所以没支持力” | 微小形变肉眼不易见         | 用恢复原状理解支持力 |

- 把红点拖到绿色限度线外：看提示从“可用”变为“不能直接用”。

#### 规律、证据与核心概念

在弹性限度内，多次测量弹簧的伸长量 `x` 与弹力大小 `F`，作 `F-x` 图像会得到一条过原点的直线。直线说明 `F` 与 `x` 成正比，斜率就是劲度系数 `k`：

```text
F = kx
```

若只讨论大小，写 $F=k\vert x\vert$；若讨论方向，弹力总指向恢复原长的方向。

**概念辨析。**

- `x` 是形变量：伸长量或压缩量，不是弹簧总长度。
- `k` 是劲度系数：表示弹簧“硬不硬”，单位常用 `N/m`。
- `F-x` 图像斜率越大，弹簧越硬。
- 胡克定律只在弹性限度内成立，超过后弹簧可能不能恢复原长。

#### 公式、变量、单位与条件

| 量       | 动画/图像对应                                     | 含义                 |
| -------- | ------------------------------------------------- | -------------------- |
| `x`    | 原长位置到当前端点的距离，或 `F-x` 图红点横坐标 | 形变量，可正可负     |
| `F`    | 红色弹力箭头和读数                                | 弹力大小             |
| `k`    | `F-x` 图像斜率，可拖紫色点改变                  | 劲度系数             |
| $F=k \vert x \vert$                    |
| 弹性限度 | 绿色限度线和状态提示                              | 超过后不能直接套公式 |

**适用边界。**

- 只适用于弹性限度内的弹簧或近似弹性体。
- `x` 必须是相对原长的伸长量或压缩量。
- 单位要统一：若 `k` 用 `N/m`，`x` 必须用 `m`。
- 弹力方向与形变量方向相反，指向恢复原长。

#### 应用与迁移

- 计算题：由 `k、x` 求弹力，或由 `F、x` 求 `k`。
- 图像题：由 `F-x` 斜率求劲度系数。
- 平衡题：弹簧拉力与重力、支持力、摩擦力平衡。
- 综合题：弹簧瞬时问题、连接体与弹簧系统。

**解题路径。**

1. 找弹簧原长和当前长度，先求形变量 `x`。
2. 判断是否在弹性限度内。
3. 单位统一后代入 $F=kx$ 求大小。
4. 画方向：伸长时弹簧拉回，压缩时弹簧推开。
5. 图像题中直接用斜率 $k=\Delta F/\Delta x$。

#### 边界检查与易错点

| 易错说法               | 为什么错                 | 改法                    |
| ---------------------- | ------------------------ | ----------------------- |
| “x 是弹簧总长度”     | 胡克定律中的 x 是形变量  | $x=\text{当前长度}-\text{原长}$     |
| “k 越大，x 一定越大” | k 是弹簧属性，不是形变量 | 同一 F 下 k 越大 x 越小 |
| “超过限度仍用 F=kx”  | 超过弹性限度可能永久形变 | 先判断是否在限度内      |
| “弹力方向总向右”     | 方向取决于伸长或压缩     | 总指向恢复原长          |

#### 情境与现象

人向后蹬地却向前走，传送带也能靠摩擦把物块向前带动，所以“摩擦力总与物体运动方向相反”并不成立。真正需要比较的是**两个接触面之间**有没有相对运动，以及若没有外界约束，它们将向哪边相对滑动。

> **公式首次使用卡**：静摩擦满足 $0\le f_s\le f_{s,\max}$，只有临界状态才取 $f_{s,\max}=\mu_sN$；接触面已经相对滑动时才用 $f_k=\mu_kN$。$N$ 是正压力，不必等于 $mg$；$\mu_s$ 与 $\mu_k$ 不得无条件混为同一参数。

> **模型边界卡**：$0\le f_s\le f_{s,\max}$ 描述静摩擦的可调范围，只有“恰要滑动”时才取上限；$f_k=\mu_kN$ 只用于已经相对滑动的接触面。$N$ 是正压力，不必等于 $mg$。中学模型通常把 $f_{s,\max}$ 写成 $\mu_sN$，并在题目明确时才近似取 $\mu_s=\mu_k$。

可直接操作：拖蓝色外力 `F` 端点或 `f-F` 图上的橙色红点改变推力；拖 $\mu$ 旋钮改变接触面粗糙程度；拖 `m` 旋钮改变正压力 $N=mg$。实时变化：摩擦力大小、箱子状态（静止/滑动）、加速度、`f-F` 图像和临界线。临界提示：$F=fₛ,max$ 时“恰好要动”。

**证据任务。**

- 拖动蓝色 `F` 端点从 0 缓慢增大，看摩擦力怎么跟着变大，箱子是否始终静止。
- 当 `F` 超过最大静摩擦力 $f_{s,\max}=\mu_sN$ 时，看箱子状态变成什么；滑动摩擦改由 $f_k=\mu_kN$ 决定，不再随 `F` 同步增大。
- 拖动 $\mu$ 或 `m`，看 `fₛ,max` 临界线怎么移动，箱子更容易动还是更难动。
- 切换“人走路”和“传送带”情境，先说出接触面间相对运动趋势，再揭示摩擦方向，验证摩擦可以充当动力。

#### 规律、证据与核心概念

摩擦来自相互接触并挤压的表面。若接触面已经相对滑动，摩擦阻碍相对滑动；若尚未滑动，摩擦阻碍可能发生的相对滑动。静摩擦是被动响应：它取“维持当前约束所需要的值”，但不能超过上限。超过上限后约束失效，接触面相对滑动，摩擦转为滑动摩擦。

**概念辨析。**

摩擦力不是“阻碍物体对地运动的力”，而是阻碍**接触面间**相对运动或其趋势的力。方向判断必须写清研究对象和另一接触面；因此地面对行人的静摩擦、传送带对较慢物块的滑动摩擦都可以与物体对地速度同向。

#### 公式、变量、单位与条件

- 静摩擦力：$0\le f_s\le f_{s,\max}$；具体值由平衡条件或动力学条件求，不存在普遍的 $f_s=F_{\text{外}}$。
- 最大静摩擦力：$f_{s,\max}=\mu_sN$，只在临界状态取到。
- 滑动摩擦力：$f_k=\mu_kN$，适用于接触面已经相对滑动的中学近似模型。
- 本页水平单推力演示中才有 $N=mg$；斜面、竖直外力或其他竖直加速度情形必须重新求 $N$。

**适用边界。**

- 没有接触和挤压，就没有接触摩擦；有接触也不保证摩擦力非零。
- $f_k=\mu_kN$ 仅用于滑动状态；静摩擦只能先求“所需值”，再检查是否超过 $f_{s,\max}$。
- $\mu_s$ 与 $\mu_k$ 是接触面的模型参数，二者不能无条件写成同一个 $\mu$。
- “最大静摩擦略大于滑动摩擦”是常见材料的经验现象，不是本页需要无条件套用的普遍定律；以题设模型为准。

#### 高考迁移（少量验证）

- **快速预测**：人向后蹬地时，地面对人的静摩擦向前，是使人加速的动力。
- **错误辨析**：“物体没动，所以静摩擦等于 $\mu N$。”错；未动时先由运动状态求所需静摩擦，只有临界才等于最大值。
- **迁移任务**：物块轻放在向右运动的传送带上，先用 $v_{\text{物}}-v_{\text{带}}$ 判断相对滑动，再判断摩擦方向；改变初速度使物块比带快，结论随之反转。高考在这里考的是参照对象与阶段转换，而不是背“摩擦向后”。

**解题路径。**

1. 写清研究对象和与它接触的另一表面，确认接触并挤压。
2. 比较两个接触面的相对运动；若尚未滑动，设想去掉摩擦后的相对运动趋势。
3. 摩擦方向与相对运动或趋势相反。
4. 静摩擦先由平衡或牛顿第二定律求“所需值”，再检验 $f_s\le f_{s,\max}$。
5. 已经滑动才用 $f_k=\mu_kN$，并重新求正压力 $N$。

#### 边界检查与易错点

- 把静摩擦力直接写成 $\mu N$（静摩擦由运动约束决定，不是固定值）。
- 认为摩擦力总是阻碍运动（摩擦力可以是动力，如传送带加速物块）。
- 忽略 N 的变化（斜面上 N=mgcosθ，不是 mg）。
- 只看物体对地速度判方向（必须比较两个接触面的相对运动或趋势）。

#### 规律、证据与核心概念

力是矢量，两个共点力的共同作用可以用一个等效的力代替，这个力叫合力。实验和矢量几何都表明：合力遵循平行四边形法则。若 $F_{1}$ 与 $F_{2}$ 的夹角为 $\theta$：

```text
F合² = F1² + F2² + 2F1F2cosθ
```

合力方向沿平行四边形对角线。

**概念辨析。**

- 合力不是“新增加的第三个力”，而是几个力的等效替代。
- 分析物体时，要么画分力，要么画合力，不能把分力和合力同时当作真实外力重复计算。
- 夹角越大，合力通常越小。
- 当两个等大反向力作用在同一点时，合力为零。

1. 判断是否为共点力。

当斜面与水平面夹角为 $\theta$ 时：

```text
G∥ = mg sinθ
G⊥ = mg cosθ
```

- 分力不是额外增加的新力，而是原力的等效替代。
- 分解方向不是唯一的，应按解题需要选择。
- 斜面上常沿斜面和垂直斜面分解，因为一个方向对应运动趋势，一个方向对应支持力。
- $mg \sin \theta$ 让物块沿斜面下滑；$mg \cos \theta$ 压紧斜面，影响支持力和摩擦力。

| 蓝色垂直斜面箭头 | $G⊥=mg \cos \theta$ | 压紧斜面的分力         |
| 斜面角           | $\theta$           | 斜面与水平面的夹角     |
| 虚线平行四边形   | 等效关系         | 两分力合成回原来的重力 |

- 分解前后作用效果等效，不能把原力和分力重复计入方程。

- 若题目给的是重力与某方向的夹角，sin/cos 关系要重新从图中判断。
- 分解方向可自选，但通常选互相垂直且便于列方程的方向。

#### 应用与迁移

- 斜面题：重力分解为 $mg \sin \theta$ 和 $mg \cos \theta$。
- 平衡题：沿两个方向列 $ΣF=0$。
- 牛顿第二定律题：沿加速度方向和垂直方向分解。
- 图像/作图题：由分力反推原力或判断分力方向。

**解题路径。**

1. 明确要把哪个力分解。
2. 选择分解方向：通常沿运动方向和垂直约束方向。
3. 画出原力和两个分力，检查分力合成能回到原力。
4. 用三角函数写分力大小，先确认角 $\theta$ 在图中的位置。

- 拖到左右角相等：看 $T_{1}$ 是否等于 $T_{2}$，水平分量是否相互抵消。
- 把左右两侧同时拖得更平：看两根绳的拉力是否明显变大。
- 只把一侧绳端拖得更平：看 $T_{1}/T_{2}$ 如何重新分配，体会非对称时不能只凭“更平”单独判断大小。

- 看右侧力三角形：三条力首尾相接回到起点时，才表示合力为零。

#### 规律、证据与核心概念

物体处于静止或匀速直线运动状态时，加速度为零。由牛顿第二定律：

```text
ΣF = ma = 0
```

所以共点力平衡的条件是合力为零。分解到坐标轴上：

```text
ΣFx = 0
ΣFy = 0
```

若只有三个力平衡，这三个力首尾相接可以组成闭合三角形。

**概念辨析。**

- 平衡不是没有受力，而是所有力的合力为零。
- 静止是平衡；匀速直线运动也是平衡。
- 三力平衡时，任意两个力的合力与第三个力等大反向。
- 选坐标轴的目的，是把矢量平衡转化为两个方向上的代数方程。

| 画面元素      | 物理量             | 含义                 |
| ------------- | ------------------ | -------------------- |
| 左蓝箭头      | $T_{1}$             | 左绳对结点的拉力     |
| 右蓝箭头      | $T_{2}$             | 右绳对结点的拉力     |
| 红箭头        | `G`              | 重物对结点的向下拉力 |
| 力三角形闭合  | $ΣF=0$          | 三个力合力为零       |
| 水平/竖直读数 | $ΣFx=0、ΣFy=0$ | 两方向分别平衡       |

**适用边界。**

- 研究对象可视为质点或共点力作用对象。
- 物体静止或做匀速直线运动。
- 若物体有加速度，则不能用 $ΣF=0$，应使用 $ΣF=ma$。
- 画受力图时只画外力，不画研究对象对外界的力。

#### 应用与迁移

- 绳结平衡：由角度和重力求绳拉力。
- 斜面平衡：沿斜面和垂直斜面列方程。
- 支架/杆平衡：判断力的方向与大小。
- 图解法：三力平衡闭合三角形。

**解题路径。**

1. 选研究对象，画完整受力图。
2. 判断是否平衡：静止或匀速直线运动。
3. 选方便坐标轴，通常沿绳、沿斜面或水平/竖直。
4. 分解所有不在坐标轴上的力。
5. 分别列 $ΣFx=0、ΣFy=0$，求未知力。

#### 边界检查与易错点

| 易错说法                   | 为什么错                       | 改法                      |
| -------------------------- | ------------------------------ | ------------------------- |
| “静止就是不受力”         | 静止可能受多个力               | 判断合力是否为零          |
| “平衡只要竖直方向抵消”   | 水平方向也要平衡               | 同时列 $ΣFx=0、ΣFy=0$ |
| “把物体对绳的拉力也画上” | 这是作用在绳上的力             | 只画研究对象受到的外力    |
| “绳越水平拉力越小”       | 水平分量需要抵消，竖直分量变小 | 绳越平通常拉力越大        |

- 看竖直分量校验条：确认两根绳的 $T \sin \theta$ 加起来仍等于 `G`。

#### 规律、证据与核心概念

动态平衡仍然满足平衡条件：

```text
ΣF = 0
```

对称双绳吊重物时，水平分量相互抵消，竖直方向满足：

```text
2T sinθ = G
T = G/(2sinθ)
```

所以当 $\theta$ 变小时，$\sin \theta$ 变小，`T` 必须变大。

**概念辨析。**

- “动态”不是物体在加速运动，而是受力结构或几何位置在缓慢变化。
- 每一个瞬间都可以看作一个新的平衡状态。
- 动态平衡常考“某力变大、变小、先变后变”等趋势判断。
- 图解法的核心是看力三角形如何变形。

#### 公式、变量、单位与条件

| 红色向下箭头 | `G`       | 重物重力，常保持不变   |
| 两个蓝色箭头 | `T`       | 两根绳的拉力           |
| 绳与水平夹角 | $\theta$      | 决定竖直分量大小       |
| 竖直分量     | $T \sin \theta$ | 每根绳向上托重物的部分 |
| 曲线红点     | 当前 `T`  | 随 $\theta$ 改变的拉力   |

- 变化过程足够慢，每个位置都可看作平衡。
- 研究对象受共点力作用。
- 若物体出现明显加速度，就不能用平衡条件，应改用牛顿第二定律。
- 公式 $T=G/(2sin\theta )$ 只适用于对称双绳模型。

#### 应用与迁移

- 趋势题：判断某个拉力随角度增大或减小。
- 极值题：绳接近水平时拉力趋于很大。
- 图解题：用力三角形看边长变化。
- 计算题：由角度和重力求绳拉力。

**解题路径。**

1. 确认物体仍处于平衡。
2. 找出哪个力大小不变、哪个力方向不变、哪个角度在变。
3. 选择解析法或图解法。
4. 解析法列 $ΣFx=0、ΣFy=0$。
5. 图解法画闭合力三角形，观察边长随角度变化。

| 易错说法                 | 为什么错                     | 改法                     |
| ------------------------ | ---------------------------- | ------------------------ |
| “动态平衡就是有加速度” | 动态指参数变化，不是瞬时加速 | 每一瞬间仍满足 $ΣF=0$ |
| “绳越平拉力越小”       | 竖直分量变小，必须增大 T     | 用 $2\ \mathrm{T} \sin \theta =G$ 判断   |
| “只看力大小不看方向”   | 方向变化会改变分量           | 画力三角形               |
| “套对称公式到非对称题” | 非对称时 T1 不等于 T2        | 重新列两个方向平衡方程   |

- 编号：B1-23

#### 情境与现象

日常经验似乎告诉我们“不推就会停”，但同一辆小车以相同初速度滑上毛毡、木板和近似无摩擦轨道时，表面越光滑，速度保持得越久。真正让车慢下来的是什么？

#### 默认引导演示（约 65 秒）

1. **三面连续对比**：相同小车、相同初速度，依次进入粗糙面、较光滑面、近似无摩擦面；同步显示阻力箭头和速度箭头。
2. **暂停预测**：第二幕结束时只问“若阻力继续减小，停车距离怎样变化？”
3. **理想外推**：把阻力逐级减小到零，频闪点由逐渐变密变为等间距，v-t 图由下降线趋向水平线。
4. **揭示因果**：车减速是阻力造成的；没有合力时，原有速度不需要力维持。

> **模型边界**：无摩擦是理想情形，用来分离“物体自身的惯性”和“环境阻力”的作用，不声称现实表面绝对无摩擦。

**证据任务。**

- 拖动粗糙度尺把 $\mu$ 从大调小：看停止距离是否变大，频闪点是否更疏、更远。
- 把 $\mu$ 拖到 `0`：看摩擦箭头是否消失，速度是否保持不变，`v-t` 图是否变成水平线。
- 拖动蓝色速度发射器改变 $v_{0}$：看没有摩擦时，物体是否仍能自己保持运动。
- 拖动时间尺红点：看同一辆车在不同时刻的速度读数如何变化。

#### 规律、证据与核心概念

伽利略通过斜面实验和理想化推理指出：物体停下来不是因为“没有力推它”，而是因为摩擦等阻力改变了它的运动。若没有外力或合力为零，运动状态不会改变。

牛顿第一定律：

```text
一切物体总保持静止状态或匀速直线运动状态，直到有外力迫使它改变这种状态为止。
```

**概念辨析。**

- 力不是维持运动的原因，力是改变运动状态的原因。
- 运动状态包括速度大小和方向。
- 惯性是物体保持原有运动状态的性质。
- 质量是惯性大小的量度，质量越大，运动状态越难改变。

#### 公式、变量、单位与条件

| 画面元素         | 物理量           | 含义                     |
| ---------------- | ---------------- | ------------------------ |
| 摩擦力红箭头     | 阻力/合力        | 改变速度，使车停下       |
| 速度蓝箭头       | `v`            | 合力为 0 时保持不变      |
| 粗糙度尺         | $\mu$           | 越小阻力越小             |
| 频闪点           | 位置记录         | 越光滑，间距越不容易缩小 |
| `v-t` 图水平线 | 匀速直线运动     | 合力为 0                 |
| 停止距离         | 运动状态改变程度 | 阻力越小越远             |

**适用边界。**

- 牛顿第一定律适用于惯性参考系。
- 合力为零时，物体可以静止，也可以匀速直线运动。
- 若合力不为零，运动状态会改变，应使用牛顿第二定律。
- 日常物体会停下，是因为存在摩擦、空气阻力等外力。

#### 高考迁移（少量验证）

- **快速预测**：汽车熄火后仍向前运动并逐渐减速；继续运动来自惯性，减速来自阻力。
- **错误辨析**：“牛顿第一定律就是 $F=ma$ 在 $F=0$ 时的特例。”错；第一定律确立惯性运动并给出惯性参考系基础，不应只当代数代入。
- **迁移任务**：比较同一小车在三种表面的频闪图，外推无阻力情形。高考常通过理想实验考查“证据支持到哪里、理想化外推了什么”。

**解题路径。**

1. 先判断物体是否受合力。
2. 合力为零：保持原状态，静止仍静止，运动则匀速直线。
3. 合力不为零：速度大小或方向会改变。
4. 解释惯性现象时，抓住“物体想保持原来的速度”。
5. 不要把“没有力”误解成“不能运动”。

#### 边界检查与易错点

| 易错说法                   | 为什么错             | 改法             |
| -------------------------- | -------------------- | ---------------- |
| “运动需要力维持”         | 无合力时可以匀速运动 | 力改变运动状态   |
| “合力为零，速度一定为零” | 也可能匀速直线运动   | 看原来的运动状态 |
| “速度大惯性大”           | 惯性由质量决定       | 质量越大惯性越大 |
| “物体停下是因为力用完了” | 停下是阻力作用结果   | 找摩擦或空气阻力 |

#### 情境与现象

用同样的合力拉轻车和重车，轻车的速度改变得更快；拉同一辆车时，合力越大，速度改变也越快。怎样用实验把“力、质量、加速度”三者的关系分开，而不是直接把结论写成公式？

> **公式首次使用卡**：$\vec F_{\text{合}}=m\vec a$ 连接同一时刻、同一研究对象的合外力与加速度，二者方向一致。中学范围默认宏观低速物体和惯性参考系；`F` 不是随手挑出的某一个力。列式前先选对象、画受力图并求矢量和。

#### 默认引导演示（约 90 秒）

1. **实验一：控制质量**。保持小车质量不变，合力依次取三档，自动记录加速度；每次测量后在 `a-F合` 图描点。
2. **暂停归纳**：隐藏拟合线，只问“合力加倍，加速度怎样变？”学生预测后显示过原点直线，得到 $a\propto F_{\text{合}}$。
3. **实验二：控制合力**。保持合力不变，逐步增加质量，记录 `a-m` 数据，并同步改画 `a-1/m` 图。
4. **暂停归纳**：只问“为什么改画 $a-1/m$？”揭示过原点直线，得到 $a\propto1/m$。
5. **合并规律**：学生先口述“同一物体的加速度与合外力成正比，与质量成反比”，最后才显示 $\vec F_{\text{合}}=m\vec a$。

> **实验焦点**：比较关系时一次只改变一个量；这里的 `F` 始终指同一时刻作用在同一研究对象上的合外力。

> **公式边界卡**：$\vec F_{\text{合}}=m\vec a$ 连接同一时刻、同一研究对象的合外力与加速度，二者方向一致。中学范围默认宏观低速物体和惯性参考系；`F` 不是随手挑出的某一个力。代入前统一单位，并先把各力按选定方向求矢量和。

**证据任务。**

- 播放第一轮实验：质量保持不变，检查每组数据是否落在 `a-F合` 直线上。
- 播放第二轮实验：合力保持不变，增加质量；比较 `a-m` 曲线与 `a-1/m` 直线哪一个更直接显示比例关系。
- 打开“加入阻力”拓展：对比外加拉力与合力，确认公式中必须使用 $F_{\text{合}}$。

#### 规律、证据与核心概念

实验采用控制变量法：保持 $m$ 不变，改变并测量合外力，数据支持 $a\propto F_{\text{合}}$；保持 $F_{\text{合}}$ 不变，改变质量，数据支持 $a\propto1/m$。两组证据合并，在选定单位制下写成 $\vec F_{\text{合}}=m\vec a$。这是从实验关系归纳出的定律，不是从购物车例子直接“证明”的结论。

**概念辨析。**

F=ma 中的 F 是**合力**，不是某个力。加速度方向与合力方向相同。质量和加速度是"原因→结果"关系：合力产生加速度，不是力产生速度。

#### 公式、变量、单位与条件

- F合 = ma → 图中红箭头（合力）= 紫箭头（加速度）× 质量
- a = F合/m → a-F 图像斜率 = 1/m
- F合 = F外 - f（有摩擦时）→ 图中合力 = 外力减去摩擦力

**适用边界。**

- 宏观低速物体（v 远小于光速）。
- 惯性参考系（地面近似为惯性系）。
- F 是合力，不是某个力。
- a 与 F 合力同时存在、同时消失、方向相同。

#### 高考迁移（少量验证）

- **快速预测**：恒定合力作用于质量不变的物体，产生恒定加速度，不是恒定速度。
- **错误辨析**：“拉力是 `6 N`，所以直接写 `6=ma`。”只有当 `6 N` 就是合外力时才成立；还要计入摩擦、重力分量等其他力。
- **迁移任务**：实验所得 `a-F外` 图在横轴有正截距，解释为未补偿阻力，并改用合力重画。高考常用图像截距考查实验误差和“外力不等于合力”。

**解题路径。**

1. 选研究对象（单个物体或整体）；
2. 画受力图，标明各力方向；
3. 选定正方向或坐标轴；
4. 求合力（正交分解后分别写 $\sum F_x$、$\sum F_y$）；
5. 列分量式 $\sum F_x=ma_x$、$\sum F_y=ma_y$；
6. 检查单位、方向、接触与绳拉力等模型边界，再连接运动学关系。

#### 边界检查与易错点

- 把某个力当合力（F=ma 中的 F 是合力，不是外力中某一个）。
- 忽略力的方向导致 a 正负号错。
- 认为 a 和 v 方向一定相同（a 与 F合 同向，v 与 a 可反向即减速）。

#### 情境与现象

把两个力传感器对拉，无论缓慢拉、突然拉还是改变其中一端的质量，两只传感器是否总同时出现读数？如果读数相等，为什么两个物体的运动效果还可能不同？

#### 默认引导演示（约 60 秒）

1. 两车接触前，两只传感器都为零；接触推压时，两读数同时出现并始终等大；分开时同时归零。
2. 在推力最大处暂停，只问“这两个力能否在同一张受力图中抵消？”随后把两箭头分别移入两车各自的受力图。
3. 保持相互作用力不变，只改变一车质量，观察两力仍相等而加速度不同。
4. 用人推墙、人走路、马拉车依次迁移“谁对谁”的命名方法。

**证据任务。**

- 拖动红色推力手柄改变 `F`：看两个力箭头是否始终等长反向。

- 看右侧对照图：确认相互作用力作用在不同物体上，不能像平衡力那样在同一受力图里抵消。

#### 规律、证据与核心概念

力是物体间的相互作用。只要 A 对 B 施力，B 必同时对 A 施力。这两个力叫作用力和反作用力：

```text
F_AB = -F_BA
```

它们大小相等、方向相反、作用在同一直线上，且分别作用在两个不同物体上。

**概念辨析。**

- 作用力和反作用力同时产生、同时消失。
- 它们性质相同，例如都是弹力、都是摩擦力或都是万有引力。
- 它们作用在不同物体上，所以不能相互抵消。
- 加速度是否相同，要看各自质量和各自所受合力。

| 紫色加速度箭头 | $a=F/m$    | 运动效果可因质量不同而不同 |

**适用边界。**

- 适用于任意两个物体间的相互作用。
- 作用力和反作用力一定作用在不同物体上。
- 它们不能直接相互抵消；是否平衡要看同一物体上所有力的合力。
- 第三定律只说明力的关系，不直接说明运动状态。

#### 高考迁移（少量验证）

- **快速预测**：人推墙时，墙对人的力与人对墙的力等大，即使墙没有明显运动。
- **错误辨析**：“一对等大反向的力一定平衡。”错；平衡力必须作用在同一物体上，作用反作用力作用在两个物体上。
- **迁移任务**：分别画人和地面的受力图解释走路。高考常把“相互作用力等大”与“运动效果相同”混在一起，后者还取决于各物体的质量和其他受力。

**解题路径。**

1. 找“谁对谁”的力，必须写清施力物体和受力物体。
2. 反作用力就是“受力物体对施力物体”的力。
3. 判断是否同性质、等大、反向、共线。
4. 若两个力作用在同一物体上，它们不是作用反作用力。
5. 受力分析只画研究对象受到的力。

#### 边界检查与易错点

| 易错说法                       | 为什么错               | 改法                   |
| ------------------------------ | ---------------------- | ---------------------- |
| “作用反作用力会抵消”         | 它们作用在不同物体上   | 不能在同一受力图中抵消 |
| “大物体对小物体的力更大”     | 第三定律说明两力等大   | 加速度不同是质量不同   |
| “先有作用力后有反作用力”     | 两力同时产生           | 写成一对相互作用       |
| “重力和支持力是一对反作用力” | 它们都作用在同一物体上 | 它们可能是平衡力       |

- 前置知识：牛顿第二定律、受力分析
- 后续应用：竖直方向动力学、圆周运动临界、航天失重

#### 情境与现象

人站在电梯体重计上。电梯向上运动时，读数可能变大、等于正常值，也可能变小；只看速度方向为什么不够？体重计直接测的是人对秤的压力，其大小等于秤对人的支持力，而不是把重力本身“称大或称小”。

> **公式首次使用卡**：人与电梯地板保持接触并具有相同加速度，取向上为正时 $N-mg=ma$。超重或失重描述支持力（视重）相对 $mg$ 的变化，不表示真实重力改变；当 $a=-g$ 时 $N=0$，但重力仍存在。

#### 默认引导演示（约 80 秒）

2. **两次暂停预测**：在“上升减速”和“下降减速”处只让学生根据加速度方向预测读数，不用速度方向作答。
3. **变量对比**：保持速度方向向上，只把加速度由向上改为向下，观察读数由超重变为失重。
4. **完全失重**：令 $a=-g$，支持力与读数降为零，但重力箭头保持存在。

> **观察焦点**：速度说明“正在向哪走”，加速度说明“合力偏向哪边”；超重或失重由后者决定。

可直接拖动：电梯旁的紫色手柄改变加速度 `a`，底部质量尺改变 `m`，`g` 尺改变重力场强。实时变化：真实重力 `mg`、支持力/视重 `N`、体重秤读数、`N/mg` 对比条和状态提示（超重、失重、完全失重）。

**证据任务。**

- 把紫色加速度手柄拖到上方：看 `N` 是否大于 `mg`，秤读数是否变大。
- 把手柄拖到下方：看 `N` 是否小于 `mg`，秤读数是否变小。



> **适用条件回看**：人与电梯地板相对静止时可用 $N-mg=ma$；若人离开地板，支持力为零，不能继续假设人与地板保持接触。代入前统一为 SI 单位。

- 点击或拖到自由下落 $a=-g$：看 `N` 是否为 0，并确认红色 `mg` 箭头仍然存在。
- 拖动质量尺或 `g` 尺：看 `mg` 和 `N` 如何同比变化，但“超重/失重”仍由加速度方向决定。

#### 规律、证据与核心概念

对电梯中的人受力分析，取向上为正：

```text
N - mg = ma
N = m(g+a)
```

所以视重由支持力 `N` 决定，不是由真实重力是否变化决定。

**概念辨析。**

- 超重：视重大于真实重力，即 $N>mg$，加速度向上。
- 失重：视重小于真实重力，即 $N<mg$，加速度向下。
- 完全失重：$N=0$，但重力仍存在，例如自由下落电梯中的人。
- 判断超重/失重看加速度方向，不看速度方向。

#### 公式、变量、单位与条件

| 画面元素          | 物理量    | 含义                     |
| ----------------- | --------- | ------------------------ |
| 向上红箭头        | `N`     | 支持力，弹簧秤读数，视重 |
| 向下红箭头        | `mg`    | 真实重力                 |
| 紫色箭头          | `a`     | 电梯和人的加速度         |
| 读数 $N=m(g+a)$ | 视重计算  | 向上为正                 |
| 状态提示          | 超重/失重 | 由 `N` 与 `mg` 比较  |

**适用边界。**

- 人与电梯地板相对静止，具有相同加速度。
- 竖直方向取向上为正时用 $N-mg=ma$。
- 超重/失重只描述视重变化，不描述真实重力变化。
- 若人离开地板，支持力 $N=0$，不能再假设人与地板接触。

#### 高考迁移（少量验证）

- **快速预测**：电梯上升但正在减速时，加速度向下，读数小于 $mg$，处于失重状态。
- **错误辨析**：“下降就是失重，完全失重就是没有重力。”两句都错；前者要看加速度，后者是支持力为零而重力仍存在。
- **迁移任务**：给出电梯 v-t 图，先由斜率判加速度，再判断秤读数。高考常用运动方向遮蔽真正决定视重的加速度方向。

**解题路径。**

1. 选研究对象：人或物体。
2. 画受力：向上支持力 `N`，向下重力 `mg`。
3. 判断加速度方向，取向上为正。
4. 列式 $N-mg=ma$。
5. 比较 `N` 与 `mg`：大于为超重，小于为失重，等于为正常。

#### 边界检查与易错点

| 易错说法             | 为什么错                    | 改法             |
| -------------------- | --------------------------- | ---------------- |
| “超重是重力变大”   | 真实重力 `mg` 基本不变    | 变的是支持力 N   |
| “上升一定超重”     | 匀速上升 N=mg，减速上升失重 | 看加速度方向     |
| “失重就是没有重力” | 完全失重时仍受重力          | N=0，不是 mg=0   |
| “速度向下就是失重” | 失重看加速度向下            | 判断 a，不判断 v |

#### 规律、证据与核心概念

连接体共同运动时，整体法把内力抵消：$F_{\text{外}}=(m_{1}+m_{2})a$。本页统一讨论“外力拉右侧 $m_{2}$”的水平连接体：先整体求 `a`，再隔离左侧 $m_{1}$ 求绳张力。光滑时 $T=m_{1}a$；有摩擦时 $T-\mu m_{1}\ \mathrm{g}=m_{1}a$。

**概念辨析。**

整体法把多个物体当一个物体看，内力（如绳张力）在整体法中抵消，只看外力。隔离法单独看一个物体，内力变成外力。先整体后隔离是连接体的核心思路。

#### 公式、变量、单位与条件

- 整体法：F = (m1+m2)·a → 图中 F 箭头 = 总质量 × a

- 隔离 m2：F - T = m2·a → F = T + m2·a（验证一致）
- 有摩擦：F - μ(m1+m2)g = (m1+m2)a

**适用边界。**

- 两物体加速度相同（绳不可伸长、共同运动）。
- 轻绳质量不计，绳力处处相等。
- 整体法和隔离法适用于同一加速度的系统。

#### 应用与迁移

- 计算题：求共同加速度和绳张力。
- 综合题：连接体 + 斜面、连接体 + 摩擦、跨滑轮连接体。

**解题路径。**

1. 判断两物体加速度是否相同；
2. 整体法：F外=(m1+m2)a 求 a；
3. 隔离法：对受力较少的物体列方程求内力；
4. 检查：对另一个物体列方程验证。

#### 边界检查与易错点

- 整体法时把内力 T 算进合力（整体法只看外力）。
- 隔离时选错研究对象（选受力简单的物体隔离更容易）。
- 两物体加速度不同时仍用整体法（加速度不同不能整体法）。

- 拖动 μ，看 `fₛ,max` 和临界外力 `Fcrit` 怎么变，μ 越大越容易还是越难相对滑动。
- 拖动 M：看同样的摩擦力要带动更重木板时，临界外力如何改变。
- 拖动时间游标：看滑痕 $\Delta x=xm-xM$ 怎样增长，什么时候达到板长 `L`。

#### 规律、证据与核心概念

板块核心是摩擦力分析。相对静止时，摩擦力是静摩擦（由牛顿第二定律决定，$\le fₛ,max$）；相对滑动时，摩擦力是滑动摩擦（$f=\mu N$）。外力拉上方物块时，假设共同运动可得 $a=F/(m+M)$，木板需要的摩擦力为 $f_{\text{需}}=Ma=MF/(m+M)$。临界条件是 $f_{\text{需}}=fₛ,max=\mu mg$，所以：

```text
Fcrit = μmg(m+M)/M
```

**概念辨析。**

板块模型的关键是判断"相对静止还是相对滑动"。判断方法：假设相对静止，求共同加速度 a，再看提供这个 a 需要的摩擦力是否超过 fₛ,max。超过则相对滑动，分别列方程。

#### 公式、变量、单位与条件

- 相对静止：F=(m+M)a，木板所需摩擦 $f_{\text{需}}=Ma$（$f_{\text{需}} \le fₛ,max$）
- 临界条件：$MF/(m+M)=\mu mg \to Fcrit=\mu mg(m+M)/M$
- 相对滑动（木板所在水平地面光滑）：am=(F-μmg)/m，aM=μmg/M
- 相对位移：Δx = ½(am-aM)t²
- 摩擦生热：Q=f滑·Δx，其中 Δx 是两接触面的相对滑动距离，不是物块对地位移

**适用边界。**

- 物块在木板上，两者间有摩擦。
- 木板与地面间可有摩擦（需额外考虑）。
- 判断相对滑动是核心步骤。

#### 应用与迁移

- 计算题：求临界外力、相对滑动后加速度、脱离时间。
- 综合题（高综合）：板块 + 能量守恒、板块 + 传送带组合。

**解题路径。**

1. 判断相对静止还是滑动（假设共动求 f，比较 fₛ,max）；
2. 相对静止：整体法求 a；
3. 相对滑动：分别列 am、aM；
4. 求相对位移和脱离条件；
5. 若题目问能量，使用 $Q=f_{\text{滑}}\cdot \Delta x$，注意这里的 $\Delta x$ 是滑痕长度；
6. 检查是否脱离木板，脱离后原来的板块接触方程失效。

#### 规律、证据与核心概念

传送带核心是摩擦力方向由相对运动方向决定。物块速度 < v带 时，物块相对传送带向后，摩擦力向前（动力），物块加速；物块速度 > v带 时，摩擦力向后（阻力），物块减速；共速后无相对运动，摩擦力消失。

**概念辨析。**

传送带问题要分阶段：加速到共速是一个阶段，共速后是另一个阶段。每阶段摩擦力方向可能不同。关键是判断"物块相对传送带"的运动方向，不是物块对地的方向。

#### 公式、变量、单位与条件

- 相对传送带慢：$a=+\mu g$；相对传送带快：$a=-\mu g$ → 对应图中的红色摩擦箭头方向
- 共速时间：$t_{\text{共}}=\vert v_{\text{带}}-v_{0}\vert /(\mu g)$ → 对应时间手柄和阶段读数
- 共速位移：$s_{\text{物}}=v_{0}t_{\text{共}}+1/2\cdot a\cdot t_{\text{共}}^{2}$ → 对应右侧 $s_{\text{物}}$ 与 `L` 的比较
- 划痕长度：$\Delta x=\vert x_{\text{带}}-x_{\text{物}}\vert$ → 对应下方两条位移条的差
- 共速后：水平带上 $v_{\text{物}}=v_{\text{带}}$，无相对运动，摩擦力为 0

**适用边界。**

- 水平传送带（倾斜传送带需考虑重力分量）。
- 物块与传送带间动摩擦因数 μ 恒定。
- 判断共速前后摩擦力是否变化是核心。

#### 应用与迁移

- 计算题：求共速时间、加速位移、划痕长度。
- 综合题：倾斜传送带、多段传送带、传送带 + 能量。

**解题路径。**

1. 判断物块初速度与传送带速度的关系；
2. 确定摩擦力方向（由相对运动方向决定）；
3. 分阶段列方程（加速到共速 / 共速后匀速）；
4. 检查物块是否在共速前离开传送带；
5. 划痕长度 = 相对位移 = |x物 - x带|。

#### 边界检查与易错点

- 共速后仍认为有摩擦力（共速后无相对运动，摩擦力消失）。
- 把物块位移当相对传送带位移（划痕是相对位移）。
- 忽略摩擦力方向可能改变（v0 > v带 时摩擦力反向）。
- 漏判短传送带条件（可能还没共速就从端点离开）。

**证据任务。**

- 切换“剪断 A-B 绳”：看 A 的弹簧力保持，B 的绳力立刻为 0。
- 切换“剪断上弹簧”：看被剪断的弹簧力为什么立刻消失。

- 拖动 `x` 或点击“令 x=平衡值”：看 $t=0-$ 的 $ΣF=0$ 如何决定原弹力。

#### 规律、证据与核心概念

剪断前，物块静止，竖直方向平衡。取向上为正：

```text
F弹 - mg - T0 = 0
F弹 = kx = mg + T0
```

剪断轻绳的一瞬间，轻绳不能继续拉物块，所以 $T'=0$；但弹簧需要形变量改变后弹力才改变，瞬时形变量仍为 `x`，所以：

```text
F弹' = F弹 = kx = mg + T0
F合 = F弹' - mg = T0
a = F合/m = T0/m
```

真正改变的是约束条件，不是弹簧的长度立刻改变。

**概念辨析。**

- 弹簧力：由形变量决定，形变量不能在一个瞬间突变，所以弹簧力瞬时不突变。
- 轻绳拉力：轻绳只能拉不能压，剪断或松弛后拉力可以立刻变成 0。
- 轻杆支持力或压力：理想轻杆可拉可压，力的大小和方向可随约束瞬时调整。
- 瞬时问题要问的是“刚剪断后的加速度”，不是随后一段时间的运动全过程。

#### 公式、变量、单位与条件

| 画面元素     | 物理量          | 公式对应                    | 读图方法               |
| ------------ | --------------- | --------------------------- | ---------------------- |

**适用边界。**

- 弹簧为轻弹簧，瞬时形变量可认为不突变。
- 轻绳被剪断、松弛或失去接触时，张力可突变为 0。
- 只讨论刚剪断后的瞬时加速度，不直接代表随后速度和位移。
- 若题目含多个物体，要分别判断每个物体连接的是弹簧、轻绳、轻杆还是接触面。
- 若弹簧本身被剪断或脱离物体，被剪掉的那段弹力不再作用在物体上。

#### 应用与迁移

- 单物体剪绳：先用剪断前平衡求弹簧力，再用剪断后受力求加速度。
- 两物体弹簧连接：剪断外绳后，弹簧连接的两物体瞬时弹力相同，分别列牛顿第二定律。
- 绳与弹簧混合：判断哪些力突变，哪些力不突变。
- 判断题：比较剪断前后某物体的加速度方向和大小。

**解题路径。**

1. 标出时刻：$t=0-$ 表示剪断前，$t=0+$ 表示剪断后瞬时。
2. 在 $t=0-$ 画受力图，利用平衡或已知运动求弹簧形变量和弹簧力。
3. 判断突变：剪断的绳力为 0，脱离接触的支持力为 0，未改变形变量的弹簧力保持原值。
4. 在 $t=0+$ 重新画受力图。
5. 对每个物体列 $F_{\text{合}}=ma$，求瞬时加速度。

#### 边界检查与易错点

| 易错说法                   | 为什么错                               | 改法                        |
| -------------------------- | -------------------------------------- | --------------------------- |
| “剪断后所有力都马上变”   | 弹簧力由形变量决定，形变量瞬时不突变   | 先保留未断弹簧的原弹力      |
| “绳力也不突变”           | 轻绳断开或松弛后不能继续拉             | 剪断处相关张力直接取 0      |
| “物块瞬间速度也突变”     | 力改变导致加速度突变，速度不能瞬时突变 | 问加速度时重新列 $F_{\text{合}}=ma$ |

| “看弹簧方向就定加速度”   | 还要与重力、其他力合成                 | 用剪断后的合力判断          |

伽利略理想实验说明：物体的运动不需要力来维持，力是改变运动状态的原因。牛顿第一定律在此基础上表述为：一切物体在不受外力或合外力为零时，总保持静止或匀速直线运动状态。

#### 科学方法与证据
| 方法 | 在实验中的表现 | 高考问法 |
|---|---|---|
| 控制变量 | 只改变摩擦或右斜面角度 | 判断实验设计 |
| 理想化 | 忽略摩擦、空气阻力 | 判断模型条件 |
| 极限思想 | 角度趋于 0、摩擦趋于 0 | 史料推理题 |
| 反驳旧观点 | 不需要力维持运动 | 惯性概念题 |

#### 边界检查与易错点
| 易错说法 | 为什么错 | 改法 |
|---|---|---|
| “理想实验不需要真实实验” | 它以真实实验趋势为基础 | 实验事实 + 合理外推 |
| “力维持运动” | 力改变运动状态 | 无力时可保持匀速 |
| “水平面上最后停下说明必须有力” | 停下是阻力造成的 | 去掉阻力看趋势 |

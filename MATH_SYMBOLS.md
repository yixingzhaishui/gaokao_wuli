# 数学符号与单位统一表

> 版本：1.0（2026-07-15）

本表是学生页面、题库、动画读数和解析的统一书写规范。公式首次出现时仍须就地说明变量、单位和适用条件。

| 物理量 | 统一符号 | 书写说明 |
|---|---|---|
| 电动势 | $\varepsilon$ | 不与电场强度 $E$ 混用 |
| 电场强度 | $\boldsymbol E$ | 矢量；需要时标方向 |
| 磁感应强度 | $\boldsymbol B$ | 矢量，单位 T |
| 磁通量 | $\Phi$ | 单位 Wb |
| 截止频率 | $\nu_0$ | 禁止写成 `ν 0` 或 `\nu 0` |
| 最大初动能 | $E_{k,\max}$ | 光电效应中强调“最大” |
| 逸出功 | $W_0$ | 与截止频率满足 $W_0=h\nu_0$ |
| 气体体积 | $V_1,V_2$ | 禁止写成 `\mathrm{V}1` |
| 最大静摩擦力 | $f_{s,\max}$ | 达到临界前静摩擦不直接取 $\mu_sN$ |
| 滑动摩擦力 | $f_k$ | 常用 $f_k=\mu_kN$，须说明滑动状态 |
| 最大功率 | $P_{\max}$ | 下标置于数学环境中 |
| 初速度 | $v_0$ | 与字母 `ν`（nu）区分 |
| 摄氏温度 | $t$ | 单位 $^\circ\mathrm C$ |
| 热力学温度 | $T$ | 单位 K，气体状态方程使用热力学温度 |

## 公式与单位排版

- 变量使用数学斜体；单位使用正体，例如 $3.0\ \mathrm{V}$、$0.50\ \mathrm{m/s}$。
- 数字与单位之间保留适当空隙；组合单位统一写法。
- 矢量首次出现时说明方向或坐标正方向。
- 近似等号只在近似计算或模型假设下使用，不能用 `=` 代替。
- 同一页面不得无说明地为同一物理量更换符号。
